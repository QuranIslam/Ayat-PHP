۞ وَيَطُوفُ عَلَيْهِمْ وِلْدَانٌ مُّخَلَّدُونَ إِذَا رَأَيْتَهُمْ حَسِبْتَهُمْ لُؤْلُؤًا مَّنثُورًا|||

{ وَيَطُوفُ } على أهل الجنة، في طعامهم وشرابهم وخدمتهم.

{ وِلْدَانٌ مُخَلَّدُونَ } أي: خلقوا من الجنة للبقاء، لا يتغيرون ولا يكبرون، وهم في غاية الحسن، { إِذَا رَأَيْتَهُمْ } منتشرين في خدمتهم { حَسِبْتَهُمْ } من حسنهم { لُؤْلُؤًا مَنْثُورًا } وهذا من تمام لذة أهل الجنة، أن يكون خدامهم الولدان المخلدون، الذين تسر رؤيتهم، ويدخلون على مساكنهم، آمنين من تبعتهم، ويأتونهم بما يدعون وتطلبه نفوسهم،
