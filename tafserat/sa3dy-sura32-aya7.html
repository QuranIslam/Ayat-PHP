الَّذِي أَحْسَنَ كُلَّ شَيْءٍ خَلَقَهُ ۖ وَبَدَأَ خَلْقَ الْإِنسَانِ مِن طِينٍ|||

{ الَّذِي أَحْسَنَ كُلَّ شَيْءٍ خَلَقَهُ } أي: كل مخلوق خلقه اللّه، فإن اللّه أحسن خلقه، وخلقه خلقًا يليق به، ويوافقه، فهذا عام.

ثم خص الآدمي لشرفه وفضله فقال: { وَبَدَأَ خَلْقَ الْإِنْسَانِ مِنْ طِينٍ } وذلك بخلق آدم عليه السلام، أبي البشر.
