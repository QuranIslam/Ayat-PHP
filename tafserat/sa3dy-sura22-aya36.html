وَالْبُدْنَ جَعَلْنَاهَا لَكُم مِّن شَعَائِرِ اللَّهِ لَكُمْ فِيهَا خَيْرٌ ۖ فَاذْكُرُوا اسْمَ اللَّهِ عَلَيْهَا صَوَافَّ ۖ فَإِذَا وَجَبَتْ جُنُوبُهَا فَكُلُوا مِنْهَا وَأَطْعِمُوا الْقَانِعَ وَالْمُعْتَرَّ ۚ كَذَٰلِكَ سَخَّرْنَاهَا لَكُمْ لَعَلَّكُمْ تَشْكُرُونَ|||

هذا دليل على أن الشعائر عام في جميع أعلام الدين الظاهرة. وتقدم أن الله أخبر أن من عظم شعائره، فإن ذلك من تقوى القلوب، وهنا أخبر أن من جملة شعائره، البدن، أي: الإبل، والبقر، على أحد القولين، فتعظم وتستسمن، وتستحسن، { لَكُمْ فِيهَا خَيْرٌ } أي: المهدي وغيره، من الأكل، والصدقة، والانتفاع، والثواب، والأجر، { فَاذْكُرُوا اسْمَ اللَّهِ عَلَيْهَا } أي: عند ذبحها قولوا " بسم الله " واذبحوها، { صَوَافَّ } أي: قائمات، بأن تقام على قوائمها الأربع، ثم تعقل يدها اليسرى، ثم تنحر.

{ فَإِذَا وَجَبَتْ جُنُوبُهَا } أي: سقطت في الأرض جنوبها، حين تسلخ، ثم يسقط الجزار جنوبها على الأرض، فحينئذ قد استعدت لأن يؤكل منها، { فَكُلُوا مِنْهَا } وهذا خطاب للمهدي، فيجوز له الأكل من هديه، { وَأَطْعِمُوا الْقَانِعَ وَالْمُعْتَرَّ } أي: الفقير الذي لا يسأل، تقنعا، وتعففا، والفقير الذي يسأل، فكل منهما له حق فيهما.

{ كَذَلِكَ سَخَّرْنَاهَا لَكُمْ } أي: البدن { لَعَلَّكُمْ تَشْكُرُونَ } الله على تسخيرها، فإنه لولا تسخيره لها، لم يكن لكم بها طاقة، ولكنه ذللها لكم وسخرها، رحمة بكم وإحسانا إليكم، فاحمدوه.
