انظُرْ كَيْفَ ضَرَبُوا لَكَ الْأَمْثَالَ فَضَلُّوا فَلَا يَسْتَطِيعُونَ سَبِيلًا|||

قال تعالى: { انْظُرْ } متعجبا { كَيْفَ ضَرَبُوا لَكَ الْأَمْثَالَ } التي هي أضل الأمثال وأبعدها عن الصواب { فَضَلُّوا } في ذلك أو فصارت سببا لضلالهم لأنهم بنوا عليها أمرهم والمبني على فاسد أفسد منه.

{ فَلَا يَسْتَطِيعُونَ سَبِيلًا } أي: لا يهتدون أي اهتداء فنصيبهم الضلال المحض والظلم الصرف.
