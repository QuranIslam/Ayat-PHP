اللَّهُ الَّذِي خَلَقَ السَّمَاوَاتِ وَالْأَرْضَ وَمَا بَيْنَهُمَا فِي سِتَّةِ أَيَّامٍ ثُمَّ اسْتَوَىٰ عَلَى الْعَرْشِ ۖ مَا لَكُم مِّن دُونِهِ مِن وَلِيٍّ وَلَا شَفِيعٍ ۚ أَفَلَا تَتَذَكَّرُونَ|||

يخبر تعالى عن كمال قدرته بخلق { السَّمَاوَاتِ وَالْأَرْضَ وَمَا بَيْنَهُمَا فِي سِتَّةِ أَيَّامٍ } أولها، يوم الأحد، وآخرها الجمعة، مع قدرته على خلقها بلحظة، ولكنه تعالى رفيق حكيم.

{ ثُمَّ اسْتَوَى عَلَى الْعَرْشِ } الذي هو سقف المخلوقات، استواء يليق بجلاله.

{ مَا لَكُمْ مِنْ دُونِهِ مِنْ وَلِيٍّ } يتولاكم، في أموركم، فينفعكم { وَلَا شَفِيع } يشفع لكم، إن توجه عليكم العقاب.

{ أَفَلَا تَتَذَكَّرُونَ } فتعلمون أن خالق الأرض والسماوات، المستوي على العرش العظيم، الذي انفرد بتدبيركم، وتوليكم، وله الشفاعة كلها، هو المستحق لجميع أنواع العبادة.
