لَّن تُغْنِيَ عَنْهُمْ أَمْوَالُهُمْ وَلَا أَوْلَادُهُم مِّنَ اللَّهِ شَيْئًا ۚ أُولَٰئِكَ أَصْحَابُ النَّارِ ۖ هُمْ فِيهَا خَالِدُونَ|||

{ لَنْ تُغْنِيَ عَنْهُمْ أَمْوَالُهُمْ وَلَا أَوْلَادُهُمْ مِنَ اللَّهِ شَيْئًا } فلا تدفع عنهم شيئا من العذاب، ولا تحصل لهم قسطا من الثواب، { أُولَئِكَ أَصْحَابُ النَّارِ } الملازمون لها، الذين لا يخرجون عنها، و { هُمْ فِيهَا خَالِدُونَ } ومن عاش على شيء مات عليه.
