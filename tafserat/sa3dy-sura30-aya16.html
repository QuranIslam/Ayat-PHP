وَأَمَّا الَّذِينَ كَفَرُوا وَكَذَّبُوا بِآيَاتِنَا وَلِقَاءِ الْآخِرَةِ فَأُولَٰئِكَ فِي الْعَذَابِ مُحْضَرُونَ|||

{ وَأَمَّا الَّذِينَ كَفَرُوا } وجحدوا نعمه وقابلوها بالكفر { وَكَذَّبُوا بِآيَاتِنَا } التي جاءتهم بها رسلنا { فَأُولَئِكَ فِي الْعَذَابِ مُحْضَرُونَ } فيه، قد أحاطت بهم جهنم من جميع جهاتهم واطَّلع العذاب الأليم على أفئدتهم وشوى الحميم وجوههم وقطَّع أمعاءهم، فأين الفرق بين الفريقين وأين التساوي بين المنعمين والمعذبين؟"
