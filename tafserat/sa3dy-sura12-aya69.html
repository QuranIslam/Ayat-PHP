وَلَمَّا دَخَلُوا عَلَىٰ يُوسُفَ آوَىٰ إِلَيْهِ أَخَاهُ ۖ قَالَ إِنِّي أَنَا أَخُوكَ فَلَا تَبْتَئِسْ بِمَا كَانُوا يَعْمَلُونَ|||

أي: لما دخل إخوة يوسف على يوسف { آوَى إِلَيْهِ أَخَاهُ } أي: شقيقه وهو "بنيامين" الذي أمرهم بالإتيان به، [و] ضمه إليه، واختصه من بين إخوته، وأخبره بحقيقة الحال، و { قَالَ إِنِّي أَنَا أَخُوكَ فَلَا تَبْتَئِسْ } أي: لا تحزن { بِمَا كَانُوا يَعْمَلُونَ } فإن العاقبة خير لنا، ثم خبره بما يريد أن يصنع ويتحيل لبقائه عنده إلى أن ينتهي الأمر.
