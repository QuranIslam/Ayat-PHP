وَهَٰذَا كِتَابٌ أَنزَلْنَاهُ مُبَارَكٌ فَاتَّبِعُوهُ وَاتَّقُوا لَعَلَّكُمْ تُرْحَمُونَ|||

{ وَهَذَا } القرآن العظيم، والذكر الحكيم. { كِتَابٌ أَنْزَلْنَاهُ مُبَارَكٌ } أي: فيه الخير الكثير والعلم الغزير، وهو الذي تستمد منه سائر العلوم، وتستخرج منه البركات، فما من خير إلا وقد دعا إليه ورغب فيه، وذكر الحكم والمصالح التي تحث عليه، وما من شر إلا وقد نهى عنه وحذر منه، وذكر الأسباب المنفرة عن فعله وعواقبها الوخيمة { فَاتَّبِعُوهُ } فيما يأمر به وينهى، وابنوا أصول دينكم وفروعه عليه { وَاتَّقُوا } الله تعالى أن تخالفوا له أمرا { لَعَلَّكُمْ } إن اتبعتموه { تُرْحَمُونَ } فأكبر سبب لنيل رحمة الله اتباع هذا الكتاب، علما وعملا.
