قَالَ إِنَّمَا الْعِلْمُ عِندَ اللَّهِ وَأُبَلِّغُكُم مَّا أُرْسِلْتُ بِهِ وَلَٰكِنِّي أَرَاكُمْ قَوْمًا تَجْهَلُونَ|||

{ قَالَ إِنَّمَا الْعِلْمُ عِنْدَ اللَّهِ } فهو الذي بيده أزمة الأمور ومقاليدها وهو الذي يأتيكم بالعذاب إن شاء. { وَأُبَلِّغُكُمْ مَا أُرْسِلْتُ بِهِ } أي: ليس علي إلا البلاغ المبين، { وَلَكِنِّي أَرَاكُمْ قَوْمًا تَجْهَلُونَ } فلذلك صدر منكم ما صدر من هذه الجرأة الشديدة، فأرسل الله عليهم العذاب العظيم وهو الريح التي دمرتهم وأهلكتهم.
