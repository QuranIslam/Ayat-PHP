۞ يَوْمَ يَجْمَعُ اللَّهُ الرُّسُلَ فَيَقُولُ مَاذَا أُجِبْتُمْ ۖ قَالُوا لَا عِلْمَ لَنَا ۖ إِنَّكَ أَنتَ عَلَّامُ الْغُيُوبِ|||

يخبر تعالى عن يوم القيامة وما فيه من الأهوال العظام، وأن الله يجمع به جميع الرسل فيسألهم: { مَاذَا أُجِبْتُمْ } أي: ماذا أجابتكم به أممكم. فـ { قَالُوا لَا عِلْمَ لَنَا } وإنما العلم لك يا ربنا، فأنت أعلم منا. { إِنَّكَ أَنْتَ عَلَّامُ الْغُيُوبِ } أي: تعلم الأمور الغائبة والحاضرة.
