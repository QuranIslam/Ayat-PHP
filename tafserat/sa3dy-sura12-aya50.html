وَقَالَ الْمَلِكُ ائْتُونِي بِهِ ۖ فَلَمَّا جَاءَهُ الرَّسُولُ قَالَ ارْجِعْ إِلَىٰ رَبِّكَ فَاسْأَلْهُ مَا بَالُ النِّسْوَةِ اللَّاتِي قَطَّعْنَ أَيْدِيَهُنَّ ۚ إِنَّ رَبِّي بِكَيْدِهِنَّ عَلِيمٌ|||

يقول تعالى: { وَقَالَ الْمَلِكُ } لمن عنده { ائْتُونِي بِهِ } أي: بيوسف عليه السلام، بأن يخرجوه من السجن ويحضروه إليه، فلما جاء يوسف الرسول وأمره بالحضور عند الملك، امتنع عن المبادرة إلى الخروج، حتى تتبين براءته التامة، وهذا من صبره وعقله ورأيه التام.

فـ { قَالَ } للرسول: { ارْجِعْ إِلَى رَبِّكَ } يعني به الملك. { فَاسْأَلْهُ مَا بَالُ النِّسْوَةِ اللَّاتِي قَطَّعْنَ أَيْدِيَهُنَّ } أي: اسأله ما شأنهن وقصتهن، فإن أمرهن ظاهر متضح { إِنَّ رَبِّي بِكَيْدِهِنَّ عَلِيمٌ }.
