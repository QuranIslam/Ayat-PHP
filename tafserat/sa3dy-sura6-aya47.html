قُلْ أَرَأَيْتَكُمْ إِنْ أَتَاكُمْ عَذَابُ اللَّهِ بَغْتَةً أَوْ جَهْرَةً هَلْ يُهْلَكُ إِلَّا الْقَوْمُ الظَّالِمُونَ|||

{ قُلْ أَرَأَيْتَكُمْ } أي: أخبروني { إِنْ أَتَاكُمْ عَذَابُ اللَّهِ بَغْتَةً أَوْ جَهْرَةً } أي: مفاجأة أو قد تقدم أمامه مقدمات، تعلمون بها وقوعه. { هَلْ يُهْلَكُ إِلَّا الْقَوْمُ الظَّالِمُونَ } الذين صاروا سببا لوقوع العذاب بهم، بظلمهم وعنادهم. فاحذروا أن تقيموا على الظلم، فإنه الهلاك الأبدي، والشقاء السرمدي.
