لَوْ نَشَاءُ لَجَعَلْنَاهُ حُطَامًا فَظَلْتُمْ تَفَكَّهُونَ|||

{ لَوْ نَشَاءُ لَجَعَلْنَاهُ } أي: الزرع المحروث وما فيه من الثمار { حُطَامًا } أي: فتاتا متحطما، لا نفع فيه ولا رزق، { فَظَلْتُمْ } أي: فصرتم بسبب جعله حطاما، بعد أن تعبتم فيه وأنفقتم النفقات الكثيرة { تَفَكَّهُونَ } أي: تندمون وتحسرون على ما أصابكم، ويزول بذلك فرحكم وسروركم وتفكهكم
