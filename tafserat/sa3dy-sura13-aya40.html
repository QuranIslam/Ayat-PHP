وَإِن مَّا نُرِيَنَّكَ بَعْضَ الَّذِي نَعِدُهُمْ أَوْ نَتَوَفَّيَنَّكَ فَإِنَّمَا عَلَيْكَ الْبَلَاغُ وَعَلَيْنَا الْحِسَابُ|||

يقول تعالى لنبيه محمد صلى الله عليه وسلم: لا تعجل عليهم بإصابة ما يوعدون به من العذاب، فهم إن استمروا على طغيانهم وكفرهم فلا بد أن يصيبهم ما وعدوا به، { إمَا نُرِيَنَّكَ } إياه في الدنيا فتقر بذلك عينك، { أو نتوفينك } قبل إصابتهم فليس ذلك شغلا لك { فَإِنَّمَا عَلَيْكَ الْبَلَاغُ } والتبيين للخلق.

{ وَعَلَيْنَا الْحِسَابُ } فنحاسب الخلق على ما قاموا به، مما عليهم، وضيعوه، ونثيبهم أو نعاقبهم.
