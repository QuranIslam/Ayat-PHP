حَتَّىٰ إِذَا جَاءُوا قَالَ أَكَذَّبْتُم بِآيَاتِي وَلَمْ تُحِيطُوا بِهَا عِلْمًا أَمَّاذَا كُنتُمْ تَعْمَلُونَ|||

{ حَتَّى إِذَا جَاءُوا } وحضروا قال لهم موبخا ومقرعا: { أَكَذَّبْتُمْ بِآيَاتِي وَلَمْ تُحِيطُوا بِهَا } العلم أي: الواجب عليكم التوقف حتى ينكشف لكم الحق وأن لا تتكلموا إلا بعلم، فكيف كذبتم بأمر لم تحيطوا به علما؟ { أَمْ مَاذَا كُنْتُمْ تَعْمَلُونَ } أي: يسألهم عن علمهم وعن عملهم فيجد عليهم تكذيبا بالحق، وعملهم لغير الله أو على غير سنة رسولهم.
