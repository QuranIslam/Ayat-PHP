وَمَا أُمِرُوا إِلَّا لِيَعْبُدُوا اللَّهَ مُخْلِصِينَ لَهُ الدِّينَ حُنَفَاءَ وَيُقِيمُوا الصَّلَاةَ وَيُؤْتُوا الزَّكَاةَ ۚ وَذَٰلِكَ دِينُ الْقَيِّمَةِ|||

فما أمروا في سائر الشرائع إلا أن يعبدوا { اللَّهَ مُخْلِصِينَ لَهُ الدِّينَ } أي: قاصدين بجميع عباداتهم الظاهرة والباطنة وجه الله، وطلب الزلفى لديه، { حُنَفَاءَ } أي: معرضين [مائلين] عن سائر الأديان المخالفة لدين التوحيد. وخص الصلاة والزكاة [بالذكر] مع أنهما داخلان في قوله { لِيَعْبُدُوا اللَّهَ مُخْلِصِينَ } لفضلهما وشرفهما، وكونهما العبادتين اللتين من قام بهما قام بجميع شرائع الدين.

{ وَذَلِكَ } أي التوحيد والإخلاص في الدين، هو { دِينُ الْقَيِّمَةِ } أي: الدين المستقيم، الموصل إلى جنات النعيم، وما سواه فطرق موصلة إلى الجحيم.
