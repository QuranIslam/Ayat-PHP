إِذْ أَرْسَلْنَا إِلَيْهِمُ اثْنَيْنِ فَكَذَّبُوهُمَا فَعَزَّزْنَا بِثَالِثٍ فَقَالُوا إِنَّا إِلَيْكُم مُّرْسَلُونَ|||

{ إِذْ أَرْسَلْنَا إِلَيْهِمُ اثْنَيْنِ فَكَذَّبُوهُمَا فَعَزَّزْنَا بِثَالِثٍ } أي: قويناهما بثالث، فصاروا ثلاثة رسل، اعتناء من اللّه بهم، وإقامة للحجة بتوالي الرسل إليهم، { فَقَالُوا } لهم: { إِنَّا إِلَيْكُمْ مُرْسَلُونَ }
