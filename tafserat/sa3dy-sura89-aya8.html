الَّتِي لَمْ يُخْلَقْ مِثْلُهَا فِي الْبِلَادِ|||

{ الَّتِي لَمْ يُخْلَقْ مِثْلُهَا } أي: مثل عاد { فِي الْبِلَادِ } أي: في جميع البلدان [في القوة والشدة]، كما قال لهم نبيهم هود عليه السلام: { وَاذْكُرُوا إِذْ جَعَلَكُمْ خُلَفَاءَ مِنْ بَعْدِ قَوْمِ نُوحٍ وَزَادَكُمْ فِي الْخَلْقِ بَسْطَةً فَاذْكُرُوا آلَاءَ اللَّهِ لَعَلَّكُمْ تُفْلِحُونَ } .
