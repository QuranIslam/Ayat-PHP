قَالَ مَعَاذَ اللَّهِ أَن نَّأْخُذَ إِلَّا مَن وَجَدْنَا مَتَاعَنَا عِندَهُ إِنَّا إِذًا لَّظَالِمُونَ|||

فـ { قَالَ } يوسف { مَعَاذَ اللَّهِ أَنْ نَأْخُذَ إِلَّا مَنْ وَجَدْنَا مَتَاعَنَا عِنْدَهُ } أي: هذا ظلم منا، لو أخذنا البريء بذنب من وجدنا متاعنا عنده، ولم يقل "من سرق" كل هذا تحرز من الكذب، { إِنَّا إِذًا } أي: إن أخذنا غير من وجد في رحله { لَظَالِمُونَ } حيث وضعنا العقوبة في غير موضعها.
