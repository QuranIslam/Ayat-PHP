أَلَمْ تَكُنْ آيَاتِي تُتْلَىٰ عَلَيْكُمْ فَكُنتُم بِهَا تُكَذِّبُونَ|||

فيقال لهم - توبيخا ولوما -: { أَلَمْ تَكُنْ آيَاتِي تُتْلَى عَلَيْكُمْ } تدعون بها، لتؤمنوا، وتعرض عليكم لتنظروا، { فَكُنْتُمْ بِهَا تُكَذِّبُونَ } ظلما منكم وعنادا، وهي آيات بينات، دالات على الحق والباطل، مبينات للمحق والمبطل
