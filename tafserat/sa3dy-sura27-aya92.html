وَأَنْ أَتْلُوَ الْقُرْآنَ ۖ فَمَنِ اهْتَدَىٰ فَإِنَّمَا يَهْتَدِي لِنَفْسِهِ ۖ وَمَن ضَلَّ فَقُلْ إِنَّمَا أَنَا مِنَ الْمُنذِرِينَ|||

{ و } أمرت أيضا { أن أَتْلُوَ } عليكم { الْقُرْآنُ } لتهتدوا به وتقتدوا وتعلموا ألفاظه ومعانيه فهذا الذي علي وقد أديته، { فَمَنِ اهْتَدَى فَإِنَّمَا يَهْتَدِي لِنَفْسِهِ } نفعه يعود عليه وثمرته عائدة إليه { وَمَنْ ضَلَّ فَقُلْ إِنَّمَا أَنَا مِنَ الْمُنْذِرِينَ } وليس بيدي من الهداية شيء.
