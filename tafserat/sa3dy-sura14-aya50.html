سَرَابِيلُهُم مِّن قَطِرَانٍ وَتَغْشَىٰ وُجُوهَهُمُ النَّارُ|||

{ سَرَابِيلُهُمْ } أي: ثيابهم { مِنْ قَطِرَانٍ } وذلك لشدة اشتعال النار فيهم وحرارتها ونتن ريحها، { وَتَغْشَى وُجُوهَهُمُ } التي هي أشرف ما في أبدانهم { النَّارَ } أي: تحيط بها وتصلاها من كل جانب، وغير الوجوه من باب أولى وأحرى، وليس هذا ظلما من الله لهم وإنما هو جزاء لما قدموا وكسبوا
