وَلَوْلَا إِذْ سَمِعْتُمُوهُ قُلْتُم مَّا يَكُونُ لَنَا أَن نَّتَكَلَّمَ بِهَٰذَا سُبْحَانَكَ هَٰذَا بُهْتَانٌ عَظِيمٌ|||

{ ولَوْلَا إِذْ سَمِعْتُمُوهُ } أي: وهلا إذ سمعتم -أيها المؤمنون- كلام أهل الإفك { قُلْتُمْ } منكرين لذلك، معظمين لأمره: { مَا يَكُونُ لَنَا أَنْ نَتَكَلَّمَ بِهَذَا } أي: ما ينبغي لنا، وما يليق بنا الكلام، بهذا الإفك المبين، لأن المؤمن يمنعه إيمانه من ارتكاب القبائح { هَذَا بُهْتَانٌ } أي: كذب عظيم.
