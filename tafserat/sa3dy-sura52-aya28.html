إِنَّا كُنَّا مِن قَبْلُ نَدْعُوهُ ۖ إِنَّهُ هُوَ الْبَرُّ الرَّحِيمُ|||

{ إِنَّا كُنَّا مِنْ قَبْلُ نَدْعُوهُ } أن يقينا عذاب السموم، ويوصلنا إلى النعيم، وهذا شامل لدعاء العبادة ودعاء المسألة أي: لم نزل نتقرب إليه بأنواع القربات وندعوه في سائر الأوقات، { إِنَّهُ هُوَ الْبَرُّ الرَّحِيمُ } فمن بره بنا ورحمته إيانا، أنالنا رضاه والجنة، ووقانا سخطه والنار.
