قَالَ لَا تَخْتَصِمُوا لَدَيَّ وَقَدْ قَدَّمْتُ إِلَيْكُم بِالْوَعِيدِ|||

قال الله تعالى مجيبًا لاختصامهم: { لَا تَخْتَصِمُوا لَدَيَّ } أي: لا فائدة في اختصامكم عندي، { و } الحال أني قد { قَدَّمْتُ إِلَيْكُمْ بِالْوَعِيدِ } أي: جاءتكم رسلي بالآيات البينات، والحجج الواضحات، والبراهين الساطعات، فقامت عليكم حجتي، وانقطعت حجتكم، وقدمتم علي بما أسلفتم من الأعمال التي وجب جزاؤها.
