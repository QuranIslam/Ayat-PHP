هُوَ الَّذِي يُحْيِي وَيُمِيتُ ۖ فَإِذَا قَضَىٰ أَمْرًا فَإِنَّمَا يَقُولُ لَهُ كُن فَيَكُونُ|||

{ هُوَ الَّذِي يُحْيِي وَيُمِيتُ } أي هو المنفرد بالإحياء والإماتة، فلا تموت نفس بسبب أو بغير سبب، إلا بإذنه. { وَمَا يُعَمَّرُ مِنْ مُعَمَّرٍ وَلَا يُنْقَصُ مِنْ عُمُرِهِ إِلَّا فِي كِتَابٍ إِنَّ ذَلِكَ عَلَى اللَّهِ يَسِيرٌ }

{ فَإِذَا قَضَى أَمْرًا } جليلاً أو حقيرًا { فَإِنَّمَا يَقُولُ لَهُ كُنْ فَيَكُونُ } لا رد في ذلك، ولا مثنوية، ولا تمنع.
