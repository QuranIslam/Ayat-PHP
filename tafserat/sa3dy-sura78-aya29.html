وَكُلَّ شَيْءٍ أَحْصَيْنَاهُ كِتَابًا|||

{ وَكُلُّ شَيْءٍ } من قليل وكثير، وخير وشر { أَحْصَيْنَاهُ كِتَابًا } أي: كتبناه في اللوح المحفوظ، فلا يخشى المجرمون أنا عذبناهم بذنوب لم يعملوها، ولا يحسبوا أنه يضيع من أعمالهم شيء، أو ينسى منها مثقال ذرة، كما قال تعالى: { وَوُضِعَ الْكِتَابُ فَتَرَى الْمُجْرِمِينَ مُشْفِقِينَ مِمَّا فِيهِ وَيَقُولُونَ يَا وَيْلَتَنَا مَالِ هَذَا الْكِتَابِ لَا يُغَادِرُ صَغِيرَةً وَلَا كَبِيرَةً إِلَّا أَحْصَاهَا وَوَجَدُوا مَا عَمِلُوا حَاضِرًا وَلَا يَظْلِمُ رَبُّكَ أَحَدًا }
