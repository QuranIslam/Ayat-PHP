اللَّهُ يَعْلَمُ مَا تَحْمِلُ كُلُّ أُنثَىٰ وَمَا تَغِيضُ الْأَرْحَامُ وَمَا تَزْدَادُ ۖ وَكُلُّ شَيْءٍ عِندَهُ بِمِقْدَارٍ|||

يخبر تعالى بعموم علمه وسعة اطلاعه وإحاطته بكل شيء فقال: { اللَّهُ يَعْلَمُ مَا تَحْمِلُ كُلُّ أُنْثَى } من بني آدم وغيرهم، { وَمَا تَغِيضُ الْأَرْحَامُ } أي: تنقص مما فيها إما أن يهلك الحمل أو يتضاءل أو يضمحل { وَمَا تَزْدَادُ } الأرحام وتكبر الأجنة التي فيها، { وَكُلُّ شَيْءٍ عِنْدَهُ بِمِقْدَارٍ } لا يتقدم عليه ولا يتأخر ولا يزيد ولا ينقص إلا بما تقتضيه حكمته وعلمه.
