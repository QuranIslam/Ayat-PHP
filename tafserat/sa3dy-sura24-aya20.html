وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ وَرَحْمَتُهُ وَأَنَّ اللَّهَ رَءُوفٌ رَّحِيمٌ|||

{ وَلَوْلَا فَضْلُ اللَّهِ عَلَيْكُمْ } قد أحاط بكم من كل جانب { وَرَحْمَتُهُ } عليكم { وَأَنَّ اللَّهَ رَءُوفٌ رَحِيمٌ } لما بين لكم هذه الأحكام والمواعظ، والحكم الجليلة، ولما أمهل من خالف أمره، ولكن فضله ورحمته، وأن ذلك وصفه اللازم آثر لكم من الخير الدنيوي والأخروي، ما لن تحصوه، أو تعدوه.
