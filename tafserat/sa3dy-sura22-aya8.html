وَمِنَ النَّاسِ مَن يُجَادِلُ فِي اللَّهِ بِغَيْرِ عِلْمٍ وَلَا هُدًى وَلَا كِتَابٍ مُّنِيرٍ|||

المجادلة المتقدمة للمقلد، وهذه المجادلة للشيطان المريد، الداعي إلى البدع، فأخبر أنه { يُجَادِلُ فِي اللَّهِ } أي: يجادل رسل الله وأتباعهم بالباطل ليدحض به الحق، { بِغَيْرِ عِلْمٍ } صحيح { وَلَا هُدًى } أي: غير متبع في جداله هذا من يهديه، لا عقل مرشد، ولا متبوع مهتد، { وَلَا كِتَابٍ مُنِيرٍ } أي: واضح بين، أي: فلا له حجة عقلية ولا نقلية ، إن هي إلا شبهات، يوحيها إليه الشيطان { وَإِنَّ الشَّيَاطِينَ لَيُوحُونَ إِلَى أَوْلِيَائِهِمْ لِيُجَادِلُوكُمْ }
