وَالشَّمْسُ تَجْرِي لِمُسْتَقَرٍّ لَّهَا ۚ ذَٰلِكَ تَقْدِيرُ الْعَزِيزِ الْعَلِيمِ|||

{ وَالشَّمْسُ تَجْرِي لِمُسْتَقَرٍّ لَهَا } [أي: دائما تجري لمستقر لها] قدره اللّه لها، لا تتعداه، ولا تقصر عنه، وليس لها تصرف في نفسها، ولا استعصاء على قدرة اللّه تعالى. { ذَلِكَ تَقْدِيرُ الْعَزِيزِ } الذي بعزته دبر هذه المخلوقات العظيمة، بأكمل تدبير، وأحسن نظام. { الْعَلِيمُ } الذي بعلمه، جعلها مصالح لعباده، ومنافع في دينهم ودنياهم.
