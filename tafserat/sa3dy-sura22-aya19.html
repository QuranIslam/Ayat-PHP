۞ هَٰذَانِ خَصْمَانِ اخْتَصَمُوا فِي رَبِّهِمْ ۖ فَالَّذِينَ كَفَرُوا قُطِّعَتْ لَهُمْ ثِيَابٌ مِّن نَّارٍ يُصَبُّ مِن فَوْقِ رُءُوسِهِمُ الْحَمِيمُ|||

ثم فصل هذا الفصل بينهم بقوله: { هَذَانِ خَصْمَانِ اخْتَصَمُوا فِي رَبِّهِمْ } كل يدعي أنه المحق.

{ فَالَّذِينَ كَفَرُوا } يشمل كل كافر، من اليهود، والنصارى، والمجوس، والصابئين، والمشركين.

{ قُطِّعَتْ لَهُمْ ثِيَابٌ مِنْ نَارٍ } أي: يجعل لهم ثياب من قطران، وتشعل فيها النار، ليعمهم العذاب من جميع جوانبهم.{ يُصَبُّ مِنْ فَوْقِ رُءُوسِهِمُ الْحَمِيمُ } الماء الحار جدا
