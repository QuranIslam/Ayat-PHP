وَبَدَا لَهُمْ سَيِّئَاتُ مَا عَمِلُوا وَحَاقَ بِهِم مَّا كَانُوا بِهِ يَسْتَهْزِئُونَ|||

فهذه حالهم في الدنيا وحال البعث الإنكار له ورد قول من جاء به قال تعالى: { وَبَدَا لَهُمْ سَيِّئَاتُ مَا عَمِلُوا } أي: وظهر لهم يوم القيامة عقوبات أعمالهم، { وَحَاقَ بِهِمْ } أي: نزل { مَا كَانُوا بِهِ يَسْتَهْزِئُونَ } أي: نزل بهم العذاب الذي كانوا في الدنيا يستهزئون به وبوقوعه وبمن جاء به.
