قَالَ الَّذِينَ اسْتَكْبَرُوا لِلَّذِينَ اسْتُضْعِفُوا أَنَحْنُ صَدَدْنَاكُمْ عَنِ الْهُدَىٰ بَعْدَ إِذْ جَاءَكُم ۖ بَلْ كُنتُم مُّجْرِمِينَ|||

{ قَالَ الَّذِينَ اسْتَكْبَرُوا لِلَّذِينَ اسْتُضْعِفُوا } مستفهمين لهم ومخبرين أن الجميع مشتركون في الجرم: { أَنَحْنُ صَدَدْنَاكُمْ عَنِ الْهُدَى بَعْدَ إِذْ جَاءَكُمْ } أي: بقوتنا وقهرنا لكم. { بَلْ كُنْتُمْ مُجْرِمِينَ } أي: مختارين للإجرام, لستم مقهورين عليه, وإن كنا قد زينا لكم, فما كان لنا عليكم من سلطان.
