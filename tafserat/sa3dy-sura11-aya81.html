قَالُوا يَا لُوطُ إِنَّا رُسُلُ رَبِّكَ لَن يَصِلُوا إِلَيْكَ ۖ فَأَسْرِ بِأَهْلِكَ بِقِطْعٍ مِّنَ اللَّيْلِ وَلَا يَلْتَفِتْ مِنكُمْ أَحَدٌ إِلَّا امْرَأَتَكَ ۖ إِنَّهُ مُصِيبُهَا مَا أَصَابَهُمْ ۚ إِنَّ مَوْعِدَهُمُ الصُّبْحُ ۚ أَلَيْسَ الصُّبْحُ بِقَرِيبٍ|||

{ قَالُوا } له: { إِنَّا رُسُلُ رَبِّكَ } أي: أخبروه بحالهم ليطمئن قلبه، { لَنْ يَصِلُوا إِلَيْكَ } بسوء.

ثم قال جبريل بجناحه، فطمس أعينهم، فانطلقوا يتوعدون لوطا بمجيء الصبح، وأمر الملائكة لوطا، أن يسري بأهله { بِقِطْعٍ مِنَ اللَّيْلِ } أي: بجانب منه قبل الفجر بكثير، ليتمكنوا من البعد عن قريتهم.

{ وَلَا يَلْتَفِتْ مِنْكُمْ أَحَدٌ } أي: بادروا بالخروج، وليكن همكم النجاة ولا تلتفتوا إلى ما وراءكم.

{ إِلَّا امْرَأَتَكَ إِنَّهُ مُصِيبُهَا } من العذاب { مَا أَصَابَهُمُ } لأنها تشارك قومها في الإثم، فتدلهم على أضياف لوط، إذا نزل به أضياف.

{ إِنَّ مَوْعِدَهُمُ الصُّبْحُ } فكأن لوطا، استعجل ذلك، فقيل له: { أَلَيْسَ الصُّبْحُ بِقَرِيبٍ }
