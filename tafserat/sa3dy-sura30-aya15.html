فَأَمَّا الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ فَهُمْ فِي رَوْضَةٍ يُحْبَرُونَ|||

{ فَأَمَّا الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ } آمنوا بقلوبهم وصدقوا ذلك بالأعمال الصالحة { فَهُمْ فِي رَوْضَةٍ } فيها سائر أنواع النبات وأصناف المشتهيات، { يُحْبَرُونَ } أي: يسرون وينعمون بالمآكل اللذيذة والأشربة والحور الحسان والخدم والولدان والأصوات المطربات والسماع المشجي والمناظر العجيبة والروائح الطيبة والفرح والسرور واللذة والحبور مما لا يقدر أحد أن يصفه.
