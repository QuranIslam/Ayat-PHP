وَلَا تَتَّخِذُوا أَيْمَانَكُمْ دَخَلًا بَيْنَكُمْ فَتَزِلَّ قَدَمٌ بَعْدَ ثُبُوتِهَا وَتَذُوقُوا السُّوءَ بِمَا صَدَدتُّمْ عَن سَبِيلِ اللَّهِ ۖ وَلَكُمْ عَذَابٌ عَظِيمٌ|||

أي: { وَلَا تَتَّخِذُوا أَيْمَانَكُمْ } وعهودكم ومواثيقكم تبعا لأهوائكم متى شئتم وفيتم بها، ومتى شئتم نقضتموها، فإنكم إذا فعلتم ذلك تزل أقدامكم بعد ثبوتها على الصراط المستقيم، { وَتَذُوقُوا السُّوءَ } أي: العذاب الذي يسوءكم ويحزنكم { بِمَا صَدَدْتُمْ عَنْ سَبِيلِ اللَّهِ } حيث ضللتم وأضللتم غيركم { وَلَكُمْ عَذَابٌ عَظِيمٌ } مضاعف.
