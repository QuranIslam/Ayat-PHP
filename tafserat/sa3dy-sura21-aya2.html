مَا يَأْتِيهِم مِّن ذِكْرٍ مِّن رَّبِّهِم مُّحْدَثٍ إِلَّا اسْتَمَعُوهُ وَهُمْ يَلْعَبُونَ|||

وأن الله تعالى لا يزال يجدد لهم التذكير والوعظ، ولا يزالون في غفلتهم وإعراضهم، ولهذا قال: { مَا يَأْتِيهِمْ مِنْ ذِكْرٍ مِنْ رَبِّهِمْ مُحْدَثٍ } يذكرهم ما ينفعهم ويحثهم عليه وما يضرهم، ويرهبهم منه { إِلَّا اسْتَمَعُوهُ } سماعا، تقوم عليهم به الحجة، { وَهُمْ يَلْعَبُونَ }
