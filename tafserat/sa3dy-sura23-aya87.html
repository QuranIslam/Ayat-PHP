سَيَقُولُونَ لِلَّهِ ۚ قُلْ أَفَلَا تَتَّقُونَ|||

{ سَيَقُولُونَ لِلَّهِ } أي: سيقرون بأن الله رب ذلك كله. قل لهم حين يقرون بذلك: { أَفَلَا تَتَّقُونَ } عبادة المخلوقات العاجزة، وتتقون الرب العظيم، كامل القدرة، عظيم السلطان؟ وفي هذا من لطف الخطاب، من قوله: { أَفَلَا تذكرون } { أفلا تَتَّقُونَ } والوعظ بأداة العرض الجاذبة للقلوب
