خَالِدِينَ فِيهَا ۖ لَا يُخَفَّفُ عَنْهُمُ الْعَذَابُ وَلَا هُمْ يُنظَرُونَ|||

و { خَالِدِينَ فِيهَا } أي: في اللعنة, أو في العذاب والمعنيان { لَا يُخَفَّفُ عَنْهُمُ الْعَذَابُ } بل عذابهم دائم شديد مستمر { وَلَا هُمْ يُنْظَرُونَ } أي: يمهلون, لأن وقت الإمهال وهو الدنيا قد مضى, ولم يبق لهم عذر فيعتذرون.
