لَا إِلَٰهَ إِلَّا هُوَ يُحْيِي وَيُمِيتُ ۖ رَبُّكُمْ وَرَبُّ آبَائِكُمُ الْأَوَّلِينَ|||

{ لَا إِلَهَ إِلَّا هُوَ } أي: لا معبود إلا وجهه، { يُحْيِي وَيُمِيتُ } أي: هو المتصرف وحده بالإحياء والإماتة وسيجمعكم بعد موتكم فيجزيكم بعملكم إن خيرا فخير وإن شرا فشر، { رَبُّكُمْ وَرَبُّ آبَائِكُمُ الْأَوَّلِينَ } أي: رب الأولين والآخرين مربيهم بالنعم الدافع عنهم النقم.
