أَفَمَنْ هُوَ قَائِمٌ عَلَىٰ كُلِّ نَفْسٍ بِمَا كَسَبَتْ ۗ وَجَعَلُوا لِلَّهِ شُرَكَاءَ قُلْ سَمُّوهُمْ ۚ أَمْ تُنَبِّئُونَهُ بِمَا لَا يَعْلَمُ فِي الْأَرْضِ أَم بِظَاهِرٍ مِّنَ الْقَوْلِ ۗ بَلْ زُيِّنَ لِلَّذِينَ كَفَرُوا مَكْرُهُمْ وَصُدُّوا عَنِ السَّبِيلِ ۗ وَمَن يُضْلِلِ اللَّهُ فَمَا لَهُ مِنْ هَادٍ|||

يقول تعالى: { أَفَمَنْ هُوَ قَائِمٌ عَلَى كُلِّ نَفْسٍ بِمَا كَسَبَتْ } بالجزاء العاجل والآجل، بالعدل والقسط، وهو الله تبارك وتعالى كمن ليس كذلك؟

ولهذا قال: { وَجَعَلُوا لِلَّهِ شُرَكَاءَ } وهو الله الأحد الفرد الصمد، الذي لا شريك له ولا ند ولا نظير، { قُلْ } لهم إن كانوا صادقين: { سَمُّوهُمْ } لتعلم حالهم { أَمْ تُنَبِّئُونَهُ بِمَا لَا يَعْلَمُ فِي الْأَرْضِ } فإنه إذا كان عالم الغيب والشهادة وهو لا يعلم له شريكا، علم بذلك بطلان دعوى الشريك له، وأنكم بمنزلة الذي يُعَلِّمُ الله أن له شريكا وهو لا يعلمه، وهذا أبطل ما يكون؛ ولهذا قال: { أَمْ بِظَاهِرٍ مِنَ الْقَوْلِ } أي: غاية ما يمكن من دعوى الشريك له تعالى أنه بظاهر أقوالكم.

وأما في الحقيقة، فلا إله إلا الله، وليس أحد من الخلق يستحق شيئا من العبادة، ولكن { زُيِّنَ لِلَّذِينَ كَفَرُوا مَكْرُهُمْ } الذي مكروه وهو كفرهم وشركهم، وتكذيبهم لآيات الله { وَصُدُّوا عَنِ السَّبِيلِ } أي: عن الطريق المستقيمة الموصلة إلى الله وإلى دار كرامته، { وَمَنْ يُضْلِلِ اللَّهُ فَمَا لَهُ مِنْ هَادٍ } لأنه ليس لأحد من الأمر شيء.
