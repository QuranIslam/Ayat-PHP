إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ لَهُمْ أَجْرٌ غَيْرُ مَمْنُونٍ|||

ولما ذكر الكافرين، ذكر المؤمنين، ووصفهم وجزاءهم، فقال: { إِنَّ الَّذِينَ آمَنُوا } بهذا الكتاب، وما اشتمل عليه مما دعا إليه من الإيمان، وصدقوا إيمانهم بالأعمال الصالحة الجامعة للإخلاص، والمتابعة. { لَهُمْ أَجْرٌ } أي: عظيم { غَيْرُ مَمْنُونٍ } أي: غير مقطوع ولا نافد، بل هو مستمر مدى الأوقات، متزايد على الساعات، مشتمل على جميع اللذات والمشتهيات.
