ذَٰلِكَ بِأَنَّهُمْ آمَنُوا ثُمَّ كَفَرُوا فَطُبِعَ عَلَىٰ قُلُوبِهِمْ فَهُمْ لَا يَفْقَهُونَ|||

{ ذَلِكَ } الذي زين لهم النفاق { بـ } سبب أنهم لا يثبتون على الإيمان.

بل { آمَنُوا ثُمَّ كَفَرُوا فَطُبِعَ عَلَى قُلُوبِهِمْ } بحيث لا يدخلها الخير أبدًا، { فَهُمْ لَا يَفْقَهُونَ } ما ينفعهم، ولا يعون ما يعود بمصالحهم.
