وَيَوْمَ يَعَضُّ الظَّالِمُ عَلَىٰ يَدَيْهِ يَقُولُ يَا لَيْتَنِي اتَّخَذْتُ مَعَ الرَّسُولِ سَبِيلًا|||

{ وَيَوْمَ يَعَضُّ الظَّالِمُ } بشركه وكفره وتكذيبه للرسل { عَلَى يَدَيْهِ } تأسفا وتحسرا وحزنا وأسفا. { يَقُولُ يَا لَيْتَنِي اتَّخَذْتُ مَعَ الرَّسُولِ سَبِيلًا } أي طريقا بالإيمان به وتصديقه واتباعه.
