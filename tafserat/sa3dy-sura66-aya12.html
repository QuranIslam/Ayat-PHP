وَمَرْيَمَ ابْنَتَ عِمْرَانَ الَّتِي أَحْصَنَتْ فَرْجَهَا فَنَفَخْنَا فِيهِ مِن رُّوحِنَا وَصَدَّقَتْ بِكَلِمَاتِ رَبِّهَا وَكُتُبِهِ وَكَانَتْ مِنَ الْقَانِتِينَ|||

{ وَمَرْيَمَ ابْنَتَ عِمْرَانَ الَّتِي أَحْصَنَتْ فَرْجَهَا } أي: صانته وحفظته عن الفاحشة، لكمال ديانتها، وعفتها، ونزاهتها.

{ فَنَفَخْنَا فِيهِ مِنْ رُوحِنَا } بأن نفخ جبريل [عليه السلام] في جيب درعها فوصلت نفخته إلى مريم، فجاء منها عيسى ابن مريم [عليه السلام]، الرسول الكريم والسيد العظيم.

{ وَصَدَّقَتْ بِكَلِمَاتِ رَبِّهَا وَكُتُبِهِ } وهذا وصف لها بالعلم والمعرفة، فإن التصديق بكلمات الله، يشمل كلماته الدينية والقدرية، والتصديق بكتبه، يقتضي معرفة ما به يحصل التصديق، ولا يكون ذلك إلا بالعلم والعمل، [ولهذا قال] { وَكَانَتْ مِنَ الْقَانِتِينَ } أي: المطيعين لله، المداومين على طاعته بخشية وخشوع، وهذا وصف لها بكمال العمل، فإنها رضي الله عنها صديقة، والصديقية: هي كمال العلم والعمل.

تمت ولله الحمد
