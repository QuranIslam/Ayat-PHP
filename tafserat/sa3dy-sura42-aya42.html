إِنَّمَا السَّبِيلُ عَلَى الَّذِينَ يَظْلِمُونَ النَّاسَ وَيَبْغُونَ فِي الْأَرْضِ بِغَيْرِ الْحَقِّ ۚ أُولَٰئِكَ لَهُمْ عَذَابٌ أَلِيمٌ|||

{ إِنَّمَا السَّبِيلُ } أي: إنما تتوجه الحجة بالعقوبة الشرعية { عَلَى الَّذِينَ يَظْلِمُونَ النَّاسَ وَيَبْغُونَ فِي الْأَرْضِ بِغَيْرِ الْحَقِّ } وهذا شامل للظلم والبغي على الناس، في دمائهم وأموالهم وأعراضهم. { أُولَئِكَ لَهُمْ عَذَابٌ أَلِيمٌ } أي: موجع للقلوب والأبدان، بحسب ظلمهم وبغيهم.
