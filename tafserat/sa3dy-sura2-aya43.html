وَأَقِيمُوا الصَّلَاةَ وَآتُوا الزَّكَاةَ وَارْكَعُوا مَعَ الرَّاكِعِينَ|||

ثم قال: { وَأَقِيمُوا الصَّلَاةَ } أي: ظاهرا وباطنا { وَآتُوا الزَّكَاةَ } مستحقيها، { وَارْكَعُوا مَعَ الرَّاكِعِينَ } أي: صلوا مع المصلين، فإنكم إذا فعلتم ذلك مع الإيمان برسل الله وآيات الله, فقد جمعتم بين الأعمال الظاهرة والباطنة, وبين الإخلاص للمعبود, والإحسان إلى عبيده، وبين العبادات القلبية البدنية والمالية. وقوله: { وَارْكَعُوا مَعَ الرَّاكِعِينَ } أي: صلوا مع المصلين, ففيه الأمر بالجماعة للصلاة ووجوبها، وفيه أن الركوع ركن من أركان الصلاة لأنه عبّر عن الصلاة بالركوع، والتعبير عن العبادة بجزئها يدل على فرضيته فيها.
