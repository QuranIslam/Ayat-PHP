فَعَسَىٰ رَبِّي أَن يُؤْتِيَنِ خَيْرًا مِّن جَنَّتِكَ وَيُرْسِلَ عَلَيْهَا حُسْبَانًا مِّنَ السَّمَاءِ فَتُصْبِحَ صَعِيدًا زَلَقًا|||

{ فَعَسَى رَبِّي أَنْ يُؤْتِيَنِ خَيْرًا مِنْ جَنَّتِكَ وَيُرْسِلَ عَلَيْهَا } أي: على جنتك التي طغيت بها وغرتك { حُسْبَانًا مِنَ السَّمَاءِ } أي: عذابا، بمطر عظيم أو غيره، { فَتُصْبِحَ } بسبب ذلك { صَعِيدًا زَلَقًا } أي: قد اقتلعت أشجارها، وتلفت ثمارها، وغرق زرعها، وزال نفعها.
