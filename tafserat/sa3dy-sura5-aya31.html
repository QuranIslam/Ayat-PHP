فَبَعَثَ اللَّهُ غُرَابًا يَبْحَثُ فِي الْأَرْضِ لِيُرِيَهُ كَيْفَ يُوَارِي سَوْءَةَ أَخِيهِ ۚ قَالَ يَا وَيْلَتَا أَعَجَزْتُ أَنْ أَكُونَ مِثْلَ هَٰذَا الْغُرَابِ فَأُوَارِيَ سَوْءَةَ أَخِي ۖ فَأَصْبَحَ مِنَ النَّادِمِينَ|||

{ فَبَعَثَ اللَّهُ غُرَابًا يَبْحَثُ فِي الْأَرْضِ } أي: يثيرها ليدفن غرابا آخر ميتا. { لِيُرِيَهُ } بذلك { كَيْفَ يُوَارِي سَوْأَةَ أَخِيهِ } أي: بدنه، لأن بدن الميت يكون عورة { فَأَصْبَحَ مِنَ النَّادِمِينَ } وهكذا عاقبة المعاصي الندامة والخسارة.
