هَٰذَا عَطَاؤُنَا فَامْنُنْ أَوْ أَمْسِكْ بِغَيْرِ حِسَابٍ|||

وقلنا له: { هَذَا عَطَاؤُنَا } فَقَرَّ به عينا { فَامْنُنْ } على من شئت، { أَوْ أَمْسِكْ } من شئت { بِغَيْرِ حِسَابٍ } أي: لا حرج عليك في ذلك ولا حساب، لعلمه تعالى بكمال عدله، وحسن أحكامه، ولا تحسبن هذا لسليمان في الدنيا دون الآخرة، بل له في الآخرة خير عظيم.
