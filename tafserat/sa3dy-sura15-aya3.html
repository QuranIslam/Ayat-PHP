ذَرْهُمْ يَأْكُلُوا وَيَتَمَتَّعُوا وَيُلْهِهِمُ الْأَمَلُ ۖ فَسَوْفَ يَعْلَمُونَ|||

فـ { ذَرْهُمْ يَأْكُلُوا وَيَتَمَتَّعُوا } بلذاتهم { وَيُلْهِهِمُ الْأَمَلُ } أي: يؤملون البقاء في الدنيا فيلهيهم عن الآخرة، { فَسَوْفَ يَعْلَمُونَ } أن ما هم عليه باطل وأن أعمالهم ذهبت خسرانا عليهم ولا يغتروا بإمهال الله تعالى فإن هذه سنته في الأمم.
