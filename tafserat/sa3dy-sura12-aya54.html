وَقَالَ الْمَلِكُ ائْتُونِي بِهِ أَسْتَخْلِصْهُ لِنَفْسِي ۖ فَلَمَّا كَلَّمَهُ قَالَ إِنَّكَ الْيَوْمَ لَدَيْنَا مَكِينٌ أَمِينٌ|||

فلما تحقق الملك والناس براءة يوسف التامة، أرسل إليه الملك وقال: { ائْتُونِي بِهِ أَسْتَخْلِصْهُ لِنَفْسِي } أي: أجعله خصيصة لي ومقربا لديَّ فأتوه به مكرما محترما، { فَلَمَّا كَلَّمَهُ } أعجبه كلامه، وزاد موقعه عنده فقال له: { إِنَّكَ الْيَوْمَ لَدَيْنَا } أي: عندنا { مَكِينٌ أَمِينٌ } أي: متمكن، أمين على الأسرار
