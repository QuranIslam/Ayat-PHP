وَمَن نُّعَمِّرْهُ نُنَكِّسْهُ فِي الْخَلْقِ ۖ أَفَلَا يَعْقِلُونَ|||

يقول تعالى: { وَمَنْ نُعَمِّرْهُ } من بني آدم { نُنَكِّسْهُ فِي الْخَلْقِ } أي: يعود إلى الحالة التي ابتدأ حالة الضعف، ضعف العقل، وضعف القوة. { أَفَلَا يَعْقِلُونَ } أن الآدمي ناقص من كل وجه، فيتداركوا قوتهم وعقولهم، فيستعملونها في طاعة ربهم.
