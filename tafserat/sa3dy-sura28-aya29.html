۞ فَلَمَّا قَضَىٰ مُوسَى الْأَجَلَ وَسَارَ بِأَهْلِهِ آنَسَ مِن جَانِبِ الطُّورِ نَارًا قَالَ لِأَهْلِهِ امْكُثُوا إِنِّي آنَسْتُ نَارًا لَّعَلِّي آتِيكُم مِّنْهَا بِخَبَرٍ أَوْ جَذْوَةٍ مِّنَ النَّارِ لَعَلَّكُمْ تَصْطَلُونَ|||

{ فَلَمَّا قَضَى مُوسَى الْأَجَلَ } يحتمل أنه قضى الأجل الواجب، أو الزائد عليه، كما هو الظن بموسى ووفائه، اشتاق إلى الوصول إلى أهله ووالدته وعشيرته، ووطنه، وعلم من طول المدة، أنهم قد تناسوا ما صدر منه. { سَارَ بِأَهْلِهِ } قاصدا مصر، { آنَسَ } أي: أبصر { مِنْ جَانِبِ الطُّورِ نَارًا قَالَ لِأَهْلِهِ امْكُثُوا إِنِّي آنَسْتُ نَارًا لَعَلِّي آتِيكُمْ مِنْهَا بِخَبَرٍ أَوْ جَذْوَةٍ مِنَ النَّارِ لَعَلَّكُمْ تَصْطَلُونَ } وكان قد أصابهم البرد، وتاهوا الطريق.
