إِنَّ الَّذِينَ لَا يُؤْمِنُونَ بِآيَاتِ اللَّهِ لَا يَهْدِيهِمُ اللَّهُ وَلَهُمْ عَذَابٌ أَلِيمٌ|||

{ إِنَّ الَّذِينَ لَا يُؤْمِنُونَ بِآيَاتِ اللَّهِ } الدالة دلالة صريحة على الحق المبين فيردونها ولا يقبلونها، { لَا يَهْدِيهِمُ اللَّهُ } حيث جاءهم الهدى فردوه فعوقبوا بحرمانه وخذلان الله لهم. { وَلَهُمْ } في الآخرة { عَذَابٌ أَلِيمٌ }
