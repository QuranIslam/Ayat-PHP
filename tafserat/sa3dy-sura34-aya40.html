وَيَوْمَ يَحْشُرُهُمْ جَمِيعًا ثُمَّ يَقُولُ لِلْمَلَائِكَةِ أَهَٰؤُلَاءِ إِيَّاكُمْ كَانُوا يَعْبُدُونَ|||

{ وَيَوْمَ يَحْشُرُهُمْ جَمِيعًا } أي: العابدين لغير اللّه والمعبودين من دونه, من الملائكة. { ثُمَّ يَقُولُ } الله { لِلْمَلَائِكَةِ } على وجه التوبيخ لمن عبدهم { أَهَؤُلَاءِ إِيَّاكُمْ كَانُوا يَعْبُدُونَ } فتبرأوا من عبادتهم.
