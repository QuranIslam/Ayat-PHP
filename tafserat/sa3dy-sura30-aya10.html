ثُمَّ كَانَ عَاقِبَةَ الَّذِينَ أَسَاءُوا السُّوأَىٰ أَن كَذَّبُوا بِآيَاتِ اللَّهِ وَكَانُوا بِهَا يَسْتَهْزِئُونَ|||

{ ثُمَّ كَانَ عَاقِبَةَ الَّذِينَ أَسَاءُوا السوأى } أي: الحالة السيئة الشنيعة، وصار ذلك داعيا لهم لأن { كَذَّبُوا بِآيَاتِ اللَّهِ وَكَانُوا بِهَا يَسْتَهْزِئُونَ } فهذا عقوبة لسوئهم وذنوبهم.

ثم ذلك الاستهزاء والتكذيب يكون سببا لأعظم العقوبات وأعضل المثلات.
