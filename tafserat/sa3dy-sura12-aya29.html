يُوسُفُ أَعْرِضْ عَنْ هَٰذَا ۚ وَاسْتَغْفِرِي لِذَنبِكِ ۖ إِنَّكِ كُنتِ مِنَ الْخَاطِئِينَ|||

ثم إن سيدها لما تحقق الأمر، قال ليوسف: { يُوسُفُ أَعْرِضْ عَنْ هَذَا } أي: اترك الكلام فيه وتناسه ولا تذكره لأحد، طلبا للستر على أهله، { وَاسْتَغْفِرِي } أيتها المرأة { لِذَنْبِكِ إِنَّكِ كُنْتِ مِنَ الْخَاطِئِينَ } فأمر يوسف بالإعراض، وهي بالاستغفار والتوبة.
