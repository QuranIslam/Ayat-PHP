وَلَمَّا جَهَّزَهُم بِجَهَازِهِمْ قَالَ ائْتُونِي بِأَخٍ لَّكُم مِّنْ أَبِيكُمْ ۚ أَلَا تَرَوْنَ أَنِّي أُوفِي الْكَيْلَ وَأَنَا خَيْرُ الْمُنزِلِينَ|||

{ وَلَمَّا جَهَّزَهُمْ بِجَهَازِهِمْ } أي: كال لهم كما كان يكيل لغيرهم، وكان من تدبيره الحسن أنه لا يكيل لكل واحد أكثر من حمل بعير، وكان قد سألهم عن حالهم، فأخبروه أن لهم أخا عند أبيه، وهو بنيامين.

فـ { قَالَ } لهم: { ائْتُونِي بِأَخٍ لَكُمْ مِنْ أَبِيكُمْ } ثم رغبهم في الإتيان به فقال: { أَلَا تَرَوْنَ أَنِّي أُوفِي الْكَيْلَ وَأَنَا خَيْرُ الْمُنْزِلِينَ } في الضيافة والإكرام.
