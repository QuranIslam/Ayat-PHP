فَوَسْوَسَ لَهُمَا الشَّيْطَانُ لِيُبْدِيَ لَهُمَا مَا وُورِيَ عَنْهُمَا مِن سَوْآتِهِمَا وَقَالَ مَا نَهَاكُمَا رَبُّكُمَا عَنْ هَٰذِهِ الشَّجَرَةِ إِلَّا أَن تَكُونَا مَلَكَيْنِ أَوْ تَكُونَا مِنَ الْخَالِدِينَ|||

فلم يزالا ممتثلين لأمر اللّه، حتى تغلغل إليهما عدوهما إبليس بمكره، فوسوس لهما وسوسة خدعهما بها، وموه عليهما وقال: { مَا نَهَاكُمَا رَبُّكُمَا عَنْ هَذِهِ الشَّجَرَةِ إِلَّا أَنْ تَكُونَا مَلَكَيْنِ } أي: من جنس الملائكة { أَوْ تَكُونَا مِنَ الْخَالِدِينَ } كما قال في الآية الأخرى: { هَلْ أَدُلُّكَ عَلَى شَجَرَةِ الْخُلْدِ وَمُلْكٍ لَا يَبْلَى }
