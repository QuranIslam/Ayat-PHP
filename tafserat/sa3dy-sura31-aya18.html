وَلَا تُصَعِّرْ خَدَّكَ لِلنَّاسِ وَلَا تَمْشِ فِي الْأَرْضِ مَرَحًا ۖ إِنَّ اللَّهَ لَا يُحِبُّ كُلَّ مُخْتَالٍ فَخُورٍ|||

{ وَلَا تُصَعِّرْ خَدَّكَ لِلنَّاسِ } أي: لا تُمِلْهُ وتعبس بوجهك الناس، تكبُّرًا عليهم، وتعاظما.

{ وَلَا تَمْشِ فِي الْأَرْضِ مَرَحًا } أي: بطرا، فخرا بالنعم، ناسيا المنعم، معجبا بنفسك. { إِنَّ اللَّهَ لَا يُحِبُّ كُلَّ مُخْتَالٍ } في نفسه وهيئته وتعاظمه { فَخُور } بقوله.
