قَالَ إِنَّمَا أَشْكُو بَثِّي وَحُزْنِي إِلَى اللَّهِ وَأَعْلَمُ مِنَ اللَّهِ مَا لَا تَعْلَمُونَ|||

{ قَالَ } يعقوب { إِنَّمَا أَشْكُو بَثِّي } أي: ما أبث من الكلام { وَحُزْنِي } الذي في قلبي { إِلَى اللَّهِ } وحده، لا إليكم ولا إلى غيركم من الخلق، فقولوا ما شئتم { وَأَعْلَمُ مِنَ اللَّهِ مَا لَا تَعْلَمُونَ } من أنه سيردهم علي ويقر عيني بالاجتماع بهم.
