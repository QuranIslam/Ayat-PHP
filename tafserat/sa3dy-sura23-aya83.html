لَقَدْ وُعِدْنَا نَحْنُ وَآبَاؤُنَا هَٰذَا مِن قَبْلُ إِنْ هَٰذَا إِلَّا أَسَاطِيرُ الْأَوَّلِينَ|||

{ لَقَدْ وُعِدْنَا نَحْنُ وَآبَاؤُنَا هَذَا مِنْ قَبْلُ } أي: ما زلنا نوعد بأن البعث كائن، نحن وآباؤنا، ولم نره، ولم يأت بعد، { إِنْ هَذَا إِلَّا أَسَاطِيرُ الْأَوَّلِينَ } أي: قصصهم وأسمارهم، التي يتحدث بها وتلهى، وإلا فليس لها حقيقة، وكذبوا -قبحهم الله- فإن الله أراهم، من آياته أكبر من البعث، ومثله، { لَخَلْقُ السَّمَاوَاتِ وَالْأَرْضِ أَكْبَرُ مِنْ خَلْقِ النَّاسِ }

{ وَضَرَبَ لَنَا مَثَلًا وَنَسِيَ خَلْقَهُ قَالَ مَنْ يُحْيِي الْعِظَامَ وَهِيَ رَمِيمٌ } الآيات { وَتَرَى الْأَرْضَ هَامِدَةً فَإِذَا أَنْزَلْنَا عَلَيْهَا الْمَاءَ اهْتَزَّتْ وَرَبَتْ } الآيات.
