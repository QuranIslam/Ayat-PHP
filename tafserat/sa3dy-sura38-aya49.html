هَٰذَا ذِكْرٌ ۚ وَإِنَّ لِلْمُتَّقِينَ لَحُسْنَ مَآبٍ|||

{ هَذَا } أي: ذكر هؤلاء الأنبياء الصفوة وذكر أوصافهم، { ذكر } في هذا القرآن ذي الذكر، يتذكر بأحوالهم المتذكرون، ويشتاق إلى الاقتداء بأوصافهم الحميدة المقتدون، ويعرف ما منَّ اللّه عليهم به من الأوصاف الزكية، وما نشر لهم من الثناء بين البرية.

فهذا نوع من أنواع الذكر، وهو ذكر أهل الخير، ومن أنواع الذكر، ذزْقُنَا مَا لَهُ مِنْ نَفَادٍ }

أي: { وَإِنَّ لِلْمُتَّقِينَ } ربهم، بامتثال الأوامر واجتناب النواهي، من كل مؤمن ومؤمنة، { لَحُسْنَ مَآبٍ } أي: لمآبا حسنا، ومرجعا مستحسنا.
