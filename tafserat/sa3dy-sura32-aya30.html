فَأَعْرِضْ عَنْهُمْ وَانتَظِرْ إِنَّهُم مُّنتَظِرُونَ|||

{ فَأَعْرِضْ عَنْهُمْ } لما وصل خطابهم إلى حالة الجهل، واستعجال العذاب. { وَانْتَظِرْ } الأمر الذي يحل بهم، فإنه لا بد منه، ولكن له أجل، إذا جاء لا يتقدم ولا يتأخر. { إِنَّهُمْ مُنْتَظِرُونَ } بك ريب المنون، ومتربصون بكم دوائر السوء، والعاقبة للتقوى.

تم تفسير سورة السجدة - بحول اللّه ومنه فله تعالى كمال الحمد والثناء والمجد.
