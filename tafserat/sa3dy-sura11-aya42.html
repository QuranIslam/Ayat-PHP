وَهِيَ تَجْرِي بِهِمْ فِي مَوْجٍ كَالْجِبَالِ وَنَادَىٰ نُوحٌ ابْنَهُ وَكَانَ فِي مَعْزِلٍ يَا بُنَيَّ ارْكَب مَّعَنَا وَلَا تَكُن مَّعَ الْكَافِرِينَ|||

ثم وصف جريانها كأنا نشاهدها فقال: { وَهِيَ تَجْرِي بِهِمْ } أي: بنوح، ومن ركب معه { فِي مَوْجٍ كَالْجِبَالِ } والله حافظها وحافظ أهلها { وَنَادَى نُوحٌ ابْنَهُ } لما ركب، ليركب معه { وَكَانَ } ابنه { فِي مَعْزِلٍ } عنهم، حين ركبوا، أي: مبتعدا وأراد منه، أن يقرب ليركب، فقال له: { يَا بُنَيَّ ارْكَبْ مَعَنَا وَلَا تَكُنْ مَعَ الْكَافِرِينَ } فيصيبك ما يصيبهم.
