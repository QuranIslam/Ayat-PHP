وَنَادَىٰ أَصْحَابُ الْجَنَّةِ أَصْحَابَ النَّارِ أَن قَدْ وَجَدْنَا مَا وَعَدَنَا رَبُّنَا حَقًّا فَهَلْ وَجَدتُّم مَّا وَعَدَ رَبُّكُمْ حَقًّا ۖ قَالُوا نَعَمْ ۚ فَأَذَّنَ مُؤَذِّنٌ بَيْنَهُمْ أَن لَّعْنَةُ اللَّهِ عَلَى الظَّالِمِينَ|||

يقول تعالى لما ذكر استقرار كل من الفريقين في الدارين، ووجدوا ما أخبرت به الرسل ونطقت به الكتب من الثواب والعقاب: أن أهل الجنة نادوا أصحاب النار بأن قالوا: { أَنْ قَدْ وَجَدْنَا مَا وَعَدَنَا رَبُّنَا حَقًّا } حين وعدنا على الإيمان والعمل الصالح الجنة فأدخلناها وأرانا ما وصفه لنا { فَهَلْ وَجَدْتُمْ مَا وَعَدَ رَبُّكُمْ } على الكفر والمعاصي { حَقًّا قالوا نعم } قد وجدناه حقا، فبين للخلق كلهم، بيانا لا شك فيه، صدق وعد اللّه، ومن أصدق من اللّه قيلا، وذهبت عنهم الشكوك والشبه، وصار الأمر حق اليقين، وفرح المؤمنون بوعد اللّه واغتبطوا، وأيس الكفار من الخير، وأقروا على أنفسهم بأنهم مستحقون للعذاب. { فَأَذَّنَ مُؤَذِّنٌ بَيْنَهُمْ } أي: بين أهل النار وأهل الجنة، بأن قال: { أَنْ لَعْنَةُ اللَّهِ } أي: بُعْدُه وإقصاؤه عن كل خير { عَلَى الظَّالِمِينَ } إذ فتح اللّه لهم أبواب رحمته، فصدفوا أنفسهم عنها ظلما، وصدوا عن سبيل اللّه بأنفسهم، وصدوا غيرهم، فضلوا وأضلوا.
