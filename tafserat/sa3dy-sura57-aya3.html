هُوَ الْأَوَّلُ وَالْآخِرُ وَالظَّاهِرُ وَالْبَاطِنُ ۖ وَهُوَ بِكُلِّ شَيْءٍ عَلِيمٌ|||

{ هُوَ الْأَوَّلُ } الذي ليس قبله شيء، { وَالْآخِرُ } الذي ليس بعده شيء { وَالظَّاهِرُ } الذي ليس فوقه شيء، { وَالْبَاطِنُ } الذي ليس دونه شيء. { وَهُوَ بِكُلِّ شَيْءٍ عَلِيمٌ } قد أحاط علمه بالظواهر والبواطن، والسرائر والخفايا، والأمور المتقدمة والمتأخرة.
