۞ أَوَلَمْ يَسِيرُوا فِي الْأَرْضِ فَيَنظُرُوا كَيْفَ كَانَ عَاقِبَةُ الَّذِينَ كَانُوا مِن قَبْلِهِمْ ۚ كَانُوا هُمْ أَشَدَّ مِنْهُمْ قُوَّةً وَآثَارًا فِي الْأَرْضِ فَأَخَذَهُمُ اللَّهُ بِذُنُوبِهِمْ وَمَا كَانَ لَهُم مِّنَ اللَّهِ مِن وَاقٍ|||

يقول تعالى: { أَوَلَمْ يَسِيرُوا فِي الْأَرْضِ } أي: بقلوبهم وأبدانهم سير نظر واعتبار، وتفكر في الآثار، { فَيَنْظُرُوا كَيْفَ كَانَ عَاقِبَةُ الَّذِينَ كَانُوا مِنْ قَبْلِهِمْ } من المكذبين، فسيجدونها شر العواقب، عاقبة الهلاك والدمار والخزي والفضيحة، وقد كانوا أشد قوة من هؤلاء في الْعَدَد والْعُدَد وكبر الأجسام. { و } أشد { آثارا في الأرض } من البناء والغرس، وقوة الآثار تدل على قوة المؤثر فيها وعلى تمنعه بها. { فَأَخَذَهُمُ اللَّهُ } بعقوبته بذنوبهم حين أصروا واستمروا عليها.
