وَالَّذِي نَزَّلَ مِنَ السَّمَاءِ مَاءً بِقَدَرٍ فَأَنشَرْنَا بِهِ بَلْدَةً مَّيْتًا ۚ كَذَٰلِكَ تُخْرَجُونَ|||

{ وَالَّذِي نَزَّلَ مِنَ السَّمَاءِ مَاءً بِقَدَرٍ } لا يزيد ولا ينقص، ويكون أيضا بمقدار الحاجة، لا ينقص بحيث لا يكون فيه نفع، ولا يزيد بحيث يضر العباد والبلاد، بل أغاث به العباد، وأنقذ به البلاد من الشدة، ولهذا قال: { فَأَنْشَرْنَا بِهِ بَلْدَةً مَيْتًا } أي: أحييناها بعد موتها، { كَذَلِكَ تُخْرَجُونَ } أي: فكما أحيا الأرض الميتة الهامدة بالماء، كذلك يحييكم بعد ما تستكملون في البرزخ، ليجازيكم بأعمالكم.
