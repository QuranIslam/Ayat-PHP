وَوَهَبْنَا لَهُ أَهْلَهُ وَمِثْلَهُم مَّعَهُمْ رَحْمَةً مِّنَّا وَذِكْرَىٰ لِأُولِي الْأَلْبَابِ|||

{ وَوَهَبْنَا لَهُ أَهْلَهُ } قيل: إن اللّه تعالى أحياهم له { وَمِثْلَهُمْ مَعَهُمْ } في الدنيا، وأغناه اللّه، وأعطاه مالا عظيما { رَحْمَةً مِنَّا } بعبدنا أيوب، حيث صبر فأثبناه من رحمتنا ثوابا عاجلا وآجلا. { وَذِكْرَى لِأُولِي الْأَلْبَابِ } أي: وليتذكر أولو العقول بحالة أيوب ويعتبروا، فيعلموا أن من صبر على الضر، أن اللّه تعالى يثيبه ثوابا عاجلا وآجلا، ويستجيب دعاءه إذا دعاه.
