قُلْ إِنَّمَا يُوحَىٰ إِلَيَّ أَنَّمَا إِلَٰهُكُمْ إِلَٰهٌ وَاحِدٌ ۖ فَهَلْ أَنتُم مُّسْلِمُونَ|||

{ قُلْ } يا محمد { إِنَّمَا يُوحَى إِلَيَّ أَنَّمَا إِلَهُكُمْ إِلَهٌ وَاحِدٌ } الذي لا يستحق العبادة إلا هو، ولهذا قال: { فَهَلْ أَنْتُمْ مُسْلِمُونَ } أي: منقادون لعبوديته مستسلمون لألوهيته، فإن فعلوا فليحمدوا ربهم على ما منَّ عليهم بهذه النعمة التي فاقت المنن.
