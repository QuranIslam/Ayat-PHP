قُلْ أَذَٰلِكَ خَيْرٌ أَمْ جَنَّةُ الْخُلْدِ الَّتِي وُعِدَ الْمُتَّقُونَ ۚ كَانَتْ لَهُمْ جَزَاءً وَمَصِيرًا|||

أي: قل لهم -مبينا لسفاهة رأيهم واختيارهم الضار على النافع-: { أَذَلِكَ } الذي وصفت لكم من العذاب { خَيْرٌ أَمْ جَنَّةُ الْخُلْدِ الَّتِي وُعِدَ الْمُتَّقُونَ } التي زادها تقوى الله فمن قام بالتقوى فالله قد وعده إياها، { كَانَتْ لَهُمْ جَزَاءً } على تقواهم { وَمَصِيرًا } موئلا يرجعون إليها، ويستقرون فيها ويخلدون دائما أبدا.
