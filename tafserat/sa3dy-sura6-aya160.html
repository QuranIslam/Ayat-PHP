مَن جَاءَ بِالْحَسَنَةِ فَلَهُ عَشْرُ أَمْثَالِهَا ۖ وَمَن جَاءَ بِالسَّيِّئَةِ فَلَا يُجْزَىٰ إِلَّا مِثْلَهَا وَهُمْ لَا يُظْلَمُونَ|||

ثم ذكر صفة الجزاء فقال: { مَنْ جَاءَ بِالْحَسَنَةِ } القولية والفعلية، الظاهرة والباطنة، المتعلقة بحق الله أو حق خلقه { فَلَهُ عَشْرُ أَمْثَالِهَا } هذا أقل ما يكون من التضعيف. { وَمَنْ جَاءَ بِالسَّيِّئَةِ فَلَا يُجْزَى إِلَّا مِثْلَهَا } وهذا من تمام عدله تعالى وإحسانه، وأنه لا يظلم مثقال ذرة، ولهذا قال: { وَهُمْ لَا يُظْلَمُونَ }
