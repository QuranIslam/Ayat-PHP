وَإِذَا جَاءُوكُمْ قَالُوا آمَنَّا وَقَد دَّخَلُوا بِالْكُفْرِ وَهُمْ قَدْ خَرَجُوا بِهِ ۚ وَاللَّهُ أَعْلَمُ بِمَا كَانُوا يَكْتُمُونَ|||

{ وَإِذَا جَاءُوكُمْ قَالُوا آمَنَّا ْ} نفاقا ومكرا { و ْ} هم { قد دَّخَلُوا ْ} مشتملين على الكفر { وَهُمْ قَدْ خَرَجُوا بِهِ ْ} فمدخلهم ومخرجهم بالكفر -وهم يزعمون أنهم مؤمنون، فهل أشر من هؤلاء وأقبح حالا منهم؟" { وَاللَّهُ أَعْلَمُ بِمَا كَانُوا يَكْتُمُونَ ْ} فيجازيهم بأعمالهم خيرها وشرها.
