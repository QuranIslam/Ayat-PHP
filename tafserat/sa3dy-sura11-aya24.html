۞ مَثَلُ الْفَرِيقَيْنِ كَالْأَعْمَىٰ وَالْأَصَمِّ وَالْبَصِيرِ وَالسَّمِيعِ ۚ هَلْ يَسْتَوِيَانِ مَثَلًا ۚ أَفَلَا تَذَكَّرُونَ|||

{ مَثَلُ الْفَرِيقَيْنِ } أي: فريق الأشقياء، وفريق السعداء. { كَالْأَعْمَى وَالْأَصَمِّ } هؤلاء الأشقياء، { وَالْبَصِيرِ وَالسَّمِيعِ } مثل السعداء.

{ هَلْ يَسْتَوِيَانِ مَثَلًا } لا يستوون مثلا، بل بينهما من الفرق ما لا يأتي عليه الوصف، { أَفَلَا تَذَكَّرُونَ } الأعمال، التي تنفعكم، فتفعلونها، والأعمال التي تضركم، فتتركونها.
