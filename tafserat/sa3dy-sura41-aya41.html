إِنَّ الَّذِينَ كَفَرُوا بِالذِّكْرِ لَمَّا جَاءَهُمْ ۖ وَإِنَّهُ لَكِتَابٌ عَزِيزٌ|||

ثم قال تعالى: { إِنَّ الَّذِينَ كَفَرُوا بِالذِّكْرِ } أي: يجحدون القرآن الكريم المذكر للعباد جميع مصالحهم الدينية والدنيوية والأخروية، المُعلي لقدر من اتبعه، { لَمَّا جَاءَهُمْ } نعمة من ربهم على يد أفضل الخلق وأكملهم. { و } الحال { إِنَّهُ لَكِتَابٌ } جامع لأوصاف الكمال { عَزِيزٌ } أي: منيع من كل من أراده بتحريف أو سوء.
