۞ أَوَلَمْ يَرَوْا أَنَّ اللَّهَ الَّذِي خَلَقَ السَّمَاوَاتِ وَالْأَرْضَ قَادِرٌ عَلَىٰ أَن يَخْلُقَ مِثْلَهُمْ وَجَعَلَ لَهُمْ أَجَلًا لَّا رَيْبَ فِيهِ فَأَبَى الظَّالِمُونَ إِلَّا كُفُورًا|||

{ أَوَلَمْ يَرَوْا أَنَّ اللَّهَ الَّذِي خَلَقَ السَّمَاوَاتِ وَالْأَرْضَ } وهي أكبر من خلق الناس. { قَادِرٌ عَلَى أَنْ يَخْلُقَ مِثْلَهُمْ } بلى، إنه على ذلك قدير.

{ و } لكنه قد { وَجَعَل } َ لذلك { أَجَلًا لَا رَيْبَ فِيهِ } ولا شك، وإلا فلو شاء لجاءهم به بغتة، ومع إقامته الحجج والأدلة على البعث.

{ فَأَبَى الظَّالِمُونَ إِلَّا كُفُورًا } ظلمًا منهم وافتراء.
