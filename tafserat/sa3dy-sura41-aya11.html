ثُمَّ اسْتَوَىٰ إِلَى السَّمَاءِ وَهِيَ دُخَانٌ فَقَالَ لَهَا وَلِلْأَرْضِ ائْتِيَا طَوْعًا أَوْ كَرْهًا قَالَتَا أَتَيْنَا طَائِعِينَ|||

{ ثُمَّ } بعد أن خلق الأرض { اسْتَوَى } أي: قصد { إِلَى } خلق { السَّمَاءِ وَهِيَ دُخَانٌ } قد ثار على وجه الماء، { فَقَالَ لَهَا } ولما كان هذا التخصيص يوهم الاختصاص، عطف عليه بقوله: { وَلِلْأَرْضِ اِئْتِيَا طَوْعًا أَوْ كَرْهًا } أي: انقادا لأمري، طائعتين أو مكرهتين، فلا بد من نفوذه. { قَالَتَا أَتَيْنَا طَائِعِينَ } ليس لنا إرادة تخالف إرادتك.
