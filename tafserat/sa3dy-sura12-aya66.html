قَالَ لَنْ أُرْسِلَهُ مَعَكُمْ حَتَّىٰ تُؤْتُونِ مَوْثِقًا مِّنَ اللَّهِ لَتَأْتُنَّنِي بِهِ إِلَّا أَن يُحَاطَ بِكُمْ ۖ فَلَمَّا آتَوْهُ مَوْثِقَهُمْ قَالَ اللَّهُ عَلَىٰ مَا نَقُولُ وَكِيلٌ|||

فـ { قَالَ } لهم يعقوب: { لَنْ أُرْسِلَهُ مَعَكُمْ حَتَّى تُؤْتُونِ مَوْثِقًا مِنْ اللَّهِ } أي: عهدا ثقيلا، وتحلفون بالله { لَتَأْتُنَّنِي بِهِ إِلَّا أَنْ يُحَاطَ بِكُمْ } أي: إلا أن يأتيكم أمر لا قبل لكم به، ولا تقدرون دفعه، { فَلَمَّا آتَوْهُ مَوْثِقَهُمْ } على ما قال وأراد { قَالَ اللَّهُ عَلَى مَا نَقُولُ وَكِيلٌ } أي: تكفينا شهادته علينا وحفظه وكفالته.
