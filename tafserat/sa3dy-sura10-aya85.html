فَقَالُوا عَلَى اللَّهِ تَوَكَّلْنَا رَبَّنَا لَا تَجْعَلْنَا فِتْنَةً لِّلْقَوْمِ الظَّالِمِينَ|||

{ فَقَالُوا } ممتثلين لذلك { عَلَى اللَّهِ تَوَكَّلْنَا رَبَّنَا لَا تَجْعَلْنَا فِتْنَةً لِلْقَوْمِ الظَّالِمِينَ } أي: لا تسلطهم علينا، فيفتنونا، أو يغلبونا، فيفتتنون بذلك، ويقولون: لو كانوا على حق لما غلبوا.
