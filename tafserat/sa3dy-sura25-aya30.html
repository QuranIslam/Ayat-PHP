وَقَالَ الرَّسُولُ يَا رَبِّ إِنَّ قَوْمِي اتَّخَذُوا هَٰذَا الْقُرْآنَ مَهْجُورًا|||

{ وَقَالَ الرَّسُولُ } مناديا لربه وشاكيا له إعراض قومه عما جاء به، ومتأسفا على ذلك منهم: { يَا رَبِّ إِنَّ قَوْمِي } الذي أرسلتني لهدايتهم وتبليغهم، { اتَّخَذُوا هَذَا الْقُرْآنَ مَهْجُورًا } أي: قد أعرضوا عنه وهجروه وتركوه مع أن الواجب عليهم الانقياد لحكمه والإقبال على أحكامه، والمشي خلفه
