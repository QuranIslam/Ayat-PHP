ذُوقُوا فِتْنَتَكُمْ هَٰذَا الَّذِي كُنتُم بِهِ تَسْتَعْجِلُونَ|||

ويقال [لهم ]: { ذُوقُوا فِتْنَتَكُمْ } أي: العذاب والنار، الذي هو أثر ما افتتنوا به، من الابتلاء الذي صيرهم إلى الكفر، والضلال، { هَذَا } العذاب، الذي وصلتم إليه، [هو] { الَّذِي كُنْتُمْ بِهِ تَسْتَعْجِلُونَ } فالآن، تمتعوا بأنواع العقاب والنكال والسلاسل والأغلال، والسخط والوبال.
