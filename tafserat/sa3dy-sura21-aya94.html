فَمَن يَعْمَلْ مِنَ الصَّالِحَاتِ وَهُوَ مُؤْمِنٌ فَلَا كُفْرَانَ لِسَعْيِهِ وَإِنَّا لَهُ كَاتِبُونَ|||

ثم فصل جزاءه فيهم، منطوقا ومفهوما، فقال: { فَمَنْ يَعْمَلْ مِنَ الصَّالِحَاتِ } أي: الأعمال التي شرعتها الرسل وحثت عليها الكتب { وَهُوَ مُؤْمِنٌ } بالله وبرسله، وما جاءوا به { فَلَا كُفْرَانَ لِسَعْيِهِ } أي: لا نضيع سعيه ولا نبطله، بل نضاعفه له أضعافا كثيرة.

{ وَإِنَّا لَهُ كَاتِبُونَ } أي: مثبتون له في اللوح المحفوظ، وفي الصحف التي مع الحفظة. أي: ومن لم يعمل من الصالحات، أو عملها وهو ليس بمؤمن، فإنه محروم، خاسر في دينه، ودنياه.
