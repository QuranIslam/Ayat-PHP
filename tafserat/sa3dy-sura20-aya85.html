قَالَ فَإِنَّا قَدْ فَتَنَّا قَوْمَكَ مِن بَعْدِكَ وَأَضَلَّهُمُ السَّامِرِيُّ|||

{ فَإِنَّا قَدْ فَتَنَّا قَوْمَكَ مِنْ بَعْدِكَ } أي: بعبادتهم للعجل، ابتليناهم، واختبرناهم، فلم يصبروا، وحين وصلت إليهم المحنة، كفروا { وَأَضَلَّهُمُ السَّامِرِيُّ }

{ فَأَخْرَجَ لَهُمْ عِجْلًا جَسَدًا } وصاغه فصار { لَهُ خُوَارٌ فَقَالُوا } لهم { هَذَا إِلَهُكُمْ وَإِلَهُ مُوسَى } فنسيه موسى، فافتتن به بنو إسرائيل، فعبدوه، ونهاهم هارون فلم ينتهوا.
