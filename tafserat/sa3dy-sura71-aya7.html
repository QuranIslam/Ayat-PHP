وَإِنِّي كُلَّمَا دَعَوْتُهُمْ لِتَغْفِرَ لَهُمْ جَعَلُوا أَصَابِعَهُمْ فِي آذَانِهِمْ وَاسْتَغْشَوْا ثِيَابَهُمْ وَأَصَرُّوا وَاسْتَكْبَرُوا اسْتِكْبَارًا|||

{ وَإِنِّي كُلَّمَا دَعَوْتُهُمْ لِتَغْفِرَ لَهُمْ } أي: لأجل أن يستجيبوا فإذا استجابوا غفرت لهم فكان هذا محض مصلحتهم، ولكنهم أبوا إلا تماديا على باطلهم، ونفورا عن الحق، { جَعَلُوا أَصَابِعَهُمْ فِي آذَانِهِمْ } حذر سماع ما يقول لهم نبيهم نوح عليه السلام، { وَاسْتَغْشَوْا ثِيَابَهُمْ } أي تغطوا بها غطاء يغشاهم بعدا عن الحق وبغضا له، { وَأَصَرُّوا } على كفرهم وشرهم { وَاسْتَكْبَرُوا } على الحق { اسْتِكْبَارًا } فشرهم ازداد، وخيرهم بعد.
