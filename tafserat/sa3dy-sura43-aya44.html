وَإِنَّهُ لَذِكْرٌ لَّكَ وَلِقَوْمِكَ ۖ وَسَوْفَ تُسْأَلُونَ|||

{ وَإِنَّهُ } أي: هذا القرآن الكريم { لَذِكْرٌ لَكَ وَلِقَوْمِكَ } أي: فخر لكم، ومنقبة جليلة، ونعمة لا يقادر قدرها، ولا يعرف وصفها، ويذكركم أيضا ما فيه الخير الدنيوي والأخروي، ويحثكم عليه، ويذكركم الشر ويرهبكم عنه، { وَسَوْفَ تُسْأَلُونَ } عنه، هل قمتم به فارتفعتم وانتفعتم، أم لم تقوموا به فيكون حجة عليكم، وكفرا منكم بهذه النعمة؟
