مِّن وَرَائِهِمْ جَهَنَّمُ ۖ وَلَا يُغْنِي عَنْهُم مَّا كَسَبُوا شَيْئًا وَلَا مَا اتَّخَذُوا مِن دُونِ اللَّهِ أَوْلِيَاءَ ۖ وَلَهُمْ عَذَابٌ عَظِيمٌ|||

وأخبر أن له عذابا أليما وأن { مِنْ وَرَائِهِمْ جَهَنَّمُ } تكفي في عقوبتهم البليغة.

وأنه { لا يُغْنِي عَنْهُمْ مَا كَسَبُوا } من الأموال { وَلَا مَا اتَّخَذُوا مِنْ دُونِ اللَّهِ أَوْلِيَاءَ } يستنصرون بهم فخذلوهم أحوج ما كانوا إليهم لو نفعوا.
