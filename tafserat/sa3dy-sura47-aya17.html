وَالَّذِينَ اهْتَدَوْا زَادَهُمْ هُدًى وَآتَاهُمْ تَقْوَاهُمْ|||

ثم بين حال المهتدين، فقال: { وَالَّذِينَ اهْتَدَوْا } بالإيمان والانقياد، واتباع ما يرضي الله { زَادَهُمْ هُدًى } شكرا منه تعالى لهم على ذلك، { وَآتَاهُمْ تَقْوَاهُمْ } أي: وفقهم للخير، وحفظهم من الشر، فذكر للمهتدين جزاءين: العلم النافع، والعمل الصالح.
