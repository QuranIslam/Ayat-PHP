وَأَسِرُّوا قَوْلَكُمْ أَوِ اجْهَرُوا بِهِ ۖ إِنَّهُ عَلِيمٌ بِذَاتِ الصُّدُورِ|||

هذا إخبار من الله بسعة علمه، وشمول لطفه فقال: { وَأَسِرُّوا قَوْلَكُمْ أَوِ اجْهَرُوا بِهِ } أي: كلها سواء لديه، لا يخفى عليه منها خافية، فـ { إِنَّهُ عَلِيمٌ بِذَاتِ الصُّدُورِ } أي: بما فيها من النيات، والإرادات، فكيف بالأقوال والأفعال، التي تسمع وترى؟!
