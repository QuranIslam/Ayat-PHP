فَأَنجَيْنَاهُ وَأَصْحَابَ السَّفِينَةِ وَجَعَلْنَاهَا آيَةً لِّلْعَالَمِينَ|||

{ فَأَنْجَيْنَاهُ وَأَصْحَابَ السَّفِينَةِ } الذين ركبوا معه، أهله ومن آمن به. { وَجَعَلْنَاهَا } أي: السفينة، أو قصة نوح { آيَةً لِلْعَالَمِينَ } يعتبرون بها، على أن من كذب الرسل، آخر أمره الهلاك، وأن المؤمنين سيجعل اللّه لهم من كل هم فرجا، ومن كل ضيق مخرجا.

وجعل اللّه أيضا السفينة، أي: جنسها آية للعالمين، يعتبرون بها رحمة ربهم، الذي قيض لهم أسبابها، ويسر لهم أمرها، وجعلها تحملهم وتحمل متاعهم من محل إلى محل ومن قُطرٍ إلى قُطرٍ.
