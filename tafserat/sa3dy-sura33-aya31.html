۞ وَمَن يَقْنُتْ مِنكُنَّ لِلَّهِ وَرَسُولِهِ وَتَعْمَلْ صَالِحًا نُّؤْتِهَا أَجْرَهَا مَرَّتَيْنِ وَأَعْتَدْنَا لَهَا رِزْقًا كَرِيمًا|||

{ وَمَنْ يَقْنُتْ مِنْكُنَّ } أي: تطيع { لِلَّهِ وَرَسُولِهِ وَتَعْمَلْ صَالِحًا } قليلا أو كثيرًا، { نُؤْتِهَا أَجْرَهَا مَرَّتَيْنِ } أي: مثل ما نعطي غيرها مرتين، { وَأَعْتَدْنَا لَهَا رِزْقًا كَرِيمًا } وهي الجنة، فقنتن للّه ورسوله، وعملن صالحًا، فعلم بذلك أجرهن.
