وَتَرَىٰ كُلَّ أُمَّةٍ جَاثِيَةً ۚ كُلُّ أُمَّةٍ تُدْعَىٰ إِلَىٰ كِتَابِهَا الْيَوْمَ تُجْزَوْنَ مَا كُنتُمْ تَعْمَلُونَ|||

ثم وصف تعالى شدة يوم القيامة وهوله ليحذره العباد ويستعد له العباد

فقال { وَتَرَى } أيها الرائي لذلك اليوم { كُلَّ أُمَّةٍ جَاثِيَةً } على ركبها خوفا وذعرا وانتظارا لحكم الملك الرحمن.

{ كُلُّ أُمَّةٍ تُدْعَى إِلَى كِتَابِهَا } أي: إلى شريعة نبيهم الذي جاءهم من عند الله، وهل قاموا بها فيحصل لهم الثواب والنجاة؟ أم ضيعوها فيحصل لهم الخسران؟ فأمة موسى يدعون إلى شريعة موسى وأمة عيسى كذلك وأمة محمد كذلك، وهكذا غيرهم كل أمة تدعى إلى شرعها الذي كلفت به، هذا أحد الاحتمالات في الآية وهو معنى صحيح في نفسه غير مشكوك فيه، ويحتمل أن المراد بقوله: { كُلُّ أُمَّةٍ تُدْعَى إِلَى كِتَابِهَا } أي: إلى كتاب أعمالها وما سطر عليها من خير وشر وأن كل أحد يجازى بما عمله بنفسه كقوله تعالى: { مَنْ عَمِلَ صَالِحًا فَلِنَفْسِهِ وَمَنْ أَسَاءَ فَعَلَيْهَا }

ويحتمل أن المعنيين كليهما مراد من الآية ويدل على هذا قوله: { هَذَا كِتَابُنَا يَنْطِقُ عَلَيْكُمْ بِالْحَقِّ }
