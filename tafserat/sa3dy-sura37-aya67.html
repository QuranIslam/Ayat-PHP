ثُمَّ إِنَّ لَهُمْ عَلَيْهَا لَشَوْبًا مِّنْ حَمِيمٍ|||

ثم ذكر شرابهم فقال: { ثُمَّ إِنَّ لَهُمْ عَلَيْهَا } أي: على أثر هذا الطعام { لَشَوْبًا مِنْ حَمِيمٍ } أي: ماء حارا، قد انتهى، كما قال تعالى: { وَإِنْ يَسْتَغِيثُوا يُغَاثُوا بِمَاءٍ كَالْمُهْلِ يَشْوِي الْوُجُوهَ بِئْسَ الشَّرَابُ وَسَاءَتْ مُرْتَفَقًا } وكما قال تعالى: { وَسُقُوا مَاءً حَمِيمًا فَقَطَّعَ أَمْعَاءَهُمْ }
