ثُمَّ رَدَدْنَا لَكُمُ الْكَرَّةَ عَلَيْهِمْ وَأَمْدَدْنَاكُم بِأَمْوَالٍ وَبَنِينَ وَجَعَلْنَاكُمْ أَكْثَرَ نَفِيرًا|||

{ ثُمَّ رَدَدْنَا لَكُمُ الْكَرَّةَ عَلَيْهِمْ } أي: على هؤلاء الذين سلطوا عليكم فأجليتموهم من دياركم. { وَأَمْدَدْنَاكُمْ بِأَمْوَالٍ وَبَنِينَ } أي: أكثرنا أرزاقكم وكثرناكم وقويناكم عليهم، { وَجَعَلْنَاكُمْ أَكْثَرَ نَفِيرًا } منهم وذلك بسبب إحسانكم وخضوعكم لله.
