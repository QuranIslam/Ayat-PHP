رَبَّنَا وَأَدْخِلْهُمْ جَنَّاتِ عَدْنٍ الَّتِي وَعَدتَّهُمْ وَمَن صَلَحَ مِنْ آبَائِهِمْ وَأَزْوَاجِهِمْ وَذُرِّيَّاتِهِمْ ۚ إِنَّكَ أَنتَ الْعَزِيزُ الْحَكِيمُ|||

{ رَبَّنَا وَأَدْخِلْهُمْ جَنَّاتِ عَدْنٍ الَّتِي وَعَدْتَهُمْ } على ألسنة رسلك { وَمَنْ صَلَحَ } أي: صلح بالإيمان والعمل الصالح { مِنْ آبَائِهِمْ وَأَزْوَاجِهِمْ } زوجاتهم وأزواجهن وأصحابهم ورفقائهم { وَذُرِّيَّاتِهِمْ } { إِنَّكَ أَنْتَ الْعَزِيزُ } القاهر لكل شيء، فبعزتك تغفر ذنوبهم، وتكشف عنهم المحذور، وتوصلهم بها إلى كل خير { الْحَكِيمُ } الذي يضع الأشياء مواضعها، فلا نسألك يا ربنا أمرا تقتضي حكمتك خلافه، بل من حكمتك التي أخبرت بها على ألسنة رسلك، واقتضاها فضلك، المغفرة للمؤمنين.
