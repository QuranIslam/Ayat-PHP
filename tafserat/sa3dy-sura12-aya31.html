فَلَمَّا سَمِعَتْ بِمَكْرِهِنَّ أَرْسَلَتْ إِلَيْهِنَّ وَأَعْتَدَتْ لَهُنَّ مُتَّكَأً وَآتَتْ كُلَّ وَاحِدَةٍ مِّنْهُنَّ سِكِّينًا وَقَالَتِ اخْرُجْ عَلَيْهِنَّ ۖ فَلَمَّا رَأَيْنَهُ أَكْبَرْنَهُ وَقَطَّعْنَ أَيْدِيَهُنَّ وَقُلْنَ حَاشَ لِلَّهِ مَا هَٰذَا بَشَرًا إِنْ هَٰذَا إِلَّا مَلَكٌ كَرِيمٌ|||

{ فَلَمَّا سَمِعَتْ بِمَكْرِهِنَّ أَرْسَلَتْ إِلَيْهِنَّ } تدعوهن إلى منزلها للضيافة.

{ وَأَعْتَدَتْ لَهُنَّ مُتَّكَأً } أي: محلا مهيأ بأنواع الفرش والوسائد، وما يقصد بذلك من المآكل اللذيذة، وكان في جملة ما أتت به وأحضرته في تلك الضيافة، طعام يحتاج إلى سكين، إما أترج، أو غيره، { وَآتَتْ كُلَّ وَاحِدَةٍ مِنْهُنَّ سِكِّينًا } ليقطعن فيها ذلك الطعام { وَقَالَتِ } ليوسف: { اخْرُجْ عَلَيْهِنَّ } في حالة جماله وبهائه.

{ فَلَمَّا رَأَيْنَهُ أَكْبَرْنَهُ } أي: أعظمنه في صدورهن، ورأين منظرا فائقا لم يشاهدن مثله، { وَقَطَّعْنَ } من الدهش { أَيْدِيَهُنَّ } بتلك السكاكين اللاتي معهن، { وَقُلْنَ حَاشَ لِلَّهِ } أي: تنزيها لله { مَا هَذَا بَشَرًا إِنْ هَذَا إِلَّا مَلَكٌ كَرِيمٌ } وذلك أن يوسف أعطي من الجمال الفائق والنور والبهاء، ما كان به آية للناظرين، وعبرة للمتأملين.
