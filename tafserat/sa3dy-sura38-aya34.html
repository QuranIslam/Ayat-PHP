وَلَقَدْ فَتَنَّا سُلَيْمَانَ وَأَلْقَيْنَا عَلَىٰ كُرْسِيِّهِ جَسَدًا ثُمَّ أَنَابَ|||

{ وَلَقَدْ فَتَنَّا سُلَيْمَانَ } أي: ابتليناه واختبرناه بذهاب ملكه وانفصاله عنه بسبب خلل اقتضته الطبيعة البشرية، { وَأَلْقَيْنَا عَلَى كُرْسِيِّهِ جَسَدًا } أي: شيطانا قضى اللّه وقدر أن يجلس على كرسي ملكه، ويتصرف في الملك في مدة فتنة سليمان، { ثُمَّ أَنَابَ } سليمان إلى اللّه تعالى وتاب.
