قَالَ لَقَدْ عَلِمْتَ مَا أَنزَلَ هَٰؤُلَاءِ إِلَّا رَبُّ السَّمَاوَاتِ وَالْأَرْضِ بَصَائِرَ وَإِنِّي لَأَظُنُّكَ يَا فِرْعَوْنُ مَثْبُورًا|||

فـ { قَالَ } له موسى { لَقَدْ عَلِمْتَ } يا فرعون { مَا أَنْزَلَ هَؤُلَاءِ } الآيات { إِلَّا رَبُّ السَّمَاوَاتِ وَالْأَرْضِ بَصَائِرَ } منه لعباده، فليس قولك هذا بالحقيقة، وإنما قلت ذلك ترويجًا على قومك، واستخفافًا لهم.

{ وَإِنِّي لَأَظُنُّكَ يَا فِرْعَوْنُ مَثْبُورًا } أي: ممقوتًا، ملقى في العذاب، لك الويل والذم واللعنة.
