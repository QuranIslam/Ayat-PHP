قُلْ يَوْمَ الْفَتْحِ لَا يَنفَعُ الَّذِينَ كَفَرُوا إِيمَانُهُمْ وَلَا هُمْ يُنظَرُونَ|||

{ قُلْ يَوْمَ الْفَتْحِ } الذي يحصل به عقابكم، لا تستفيدون به شيئًا، فلو كان إذا حصل، حصل إمهالكم، لتستدركوا ما فاتكم، حين صار الأمر عندكم يقينًا، لكان لذلك وجه، ولكن إذا جاء يوم الفتح، انقضى الأمر، ولم يبق للمحنة محل فـ { لَا يَنْفَعُ الَّذِينَ كَفَرُوا إِيمَانُهُمْ } لأنه صار إيمان ضرورة، { وَلَا هُمْ يُنْظَرُونَ } أي: يمهلون، فيؤخر عنهم العذاب، فيستدركون أمرهم.
