حَتَّىٰ إِذَا أَخَذْنَا مُتْرَفِيهِم بِالْعَذَابِ إِذَا هُمْ يَجْأَرُونَ|||

{ حَتَّى إِذَا أَخَذْنَا مُتْرَفِيهِمْ } أي: متنعميهم، الذين ما اعتادوا إلا الترف والرفاهية والنعيم، ولم تحصل لهم المكاره، فإذا أخذناهم { بِالْعَذَابِ } ووجدوا مسه { إِذَا هُمْ يَجْأَرُونَ } يصرخون ويتوجعون، لأنه أصابهم أمر خالف ما هم عليه
