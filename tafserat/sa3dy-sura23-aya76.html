وَلَقَدْ أَخَذْنَاهُم بِالْعَذَابِ فَمَا اسْتَكَانُوا لِرَبِّهِمْ وَمَا يَتَضَرَّعُونَ|||

{ وَلَقَدْ أَخَذْنَاهُمْ بِالْعَذَابِ } قال المفسرون: المراد بذلك: الجوع الذي أصابهم سبع سنين، وأن الله ابتلاهم بذلك، ليرجعوا إليه بالذل والاستسلام، فلم ينجع فيهم، ولا نجح منهم أحد، { فَمَا اسْتَكَانُوا لِرَبِّهِمْ } أي: خضعوا وذلوا { وَمَا يَتَضَرَّعُونَ } إليه ويفتقرون، بل مر عليهم ذلك ثم زال، كأنه لم يصبهم، لم يزالوا في غيهم وكفرهم، ولكن وراءهم العذاب الذي لا يرد
