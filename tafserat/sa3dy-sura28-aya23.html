وَلَمَّا وَرَدَ مَاءَ مَدْيَنَ وَجَدَ عَلَيْهِ أُمَّةً مِّنَ النَّاسِ يَسْقُونَ وَوَجَدَ مِن دُونِهِمُ امْرَأَتَيْنِ تَذُودَانِ ۖ قَالَ مَا خَطْبُكُمَا ۖ قَالَتَا لَا نَسْقِي حَتَّىٰ يُصْدِرَ الرِّعَاءُ ۖ وَأَبُونَا شَيْخٌ كَبِيرٌ|||

{ وَلَمَّا وَرَدَ مَاءَ مَدْيَنَ وَجَدَ عَلَيْهِ أُمَّةً مِنَ النَّاسِ يَسْقُونَ } مواشيهم، وكانوا أهل ماشية كثيرة { وَوَجَدَ مِنْ دُونِهِمُ } أي: دون تلك الأمة { امْرَأتَيْنِ تَذُودَانِ } غنمهما عن حياض الناس، لعجزهما عن مزاحمة الرجال وبخلهم، وعدم مروءتهم عن السقي لهما.

{ قَالَ } لهما موسى { مَا خَطْبُكُمَا } أي: ما شأنكما بهذه الحالة، { قَالَتَا لَا نَسْقِي حَتَّى يُصْدِرَ الرِّعَاءُ } أي: قد جرت العادة أنه لا يحصل لنا سقي حتى يصدر الرعاء مواشيهم، فإذا خلا لنا الجو سقينا، { وَأَبُونَا شَيْخٌ كَبِيرٌ } أي: لا قوة له على السقي، فليس فينا قوة، نقتدر بها، ولا لنا رجال يزاحمون الرعاء.
