وَلَقَدْ مَكَّنَّاكُمْ فِي الْأَرْضِ وَجَعَلْنَا لَكُمْ فِيهَا مَعَايِشَ ۗ قَلِيلًا مَّا تَشْكُرُونَ|||

يقول تعالى ممتنا على عباده بذكر المسكن والمعيشة: { وَلَقَدْ مَكَّنَّاكُمْ فِي الْأَرْضِ } أي: هيأناها لكم، بحيث تتمكنون من البناء عليها وحرثها، ووجوه الانتفاع بها { وَجَعَلْنَا لَكُمْ فِيهَا مَعَايِشَ } مما يخرج من الأشجار والنبات، ومعادن الأرض، وأنواع الصنائع والتجارات، فإنه هو الذي هيأها، وسخر أسبابها. { قَلِيلًا مَا تَشْكُرُونَ } اللّه، الذي أنعم عليكم بأصناف النعم، وصرف عنكم النقم.
