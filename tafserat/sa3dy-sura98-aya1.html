لَمْ يَكُنِ الَّذِينَ كَفَرُوا مِنْ أَهْلِ الْكِتَابِ وَالْمُشْرِكِينَ مُنفَكِّينَ حَتَّىٰ تَأْتِيَهُمُ الْبَيِّنَةُ|||

يقول تعالى: { لَمْ يَكُنِ الَّذِينَ كَفَرُوا مِنْ أَهْلِ الْكِتَابِ } أي: [من] اليهود والنصارى { وَالْمُشْرِكِينَ } من سائر أصناف الأمم.

{ مُنْفَكِّينَ } عن كفرهم وضلالهم الذي هم عليه، أي: لا يزالون في غيهم وضلالهم، لا يزيدهم مرور السنين إلا كفرًا.

{ حَتَّى تَأْتِيَهُمُ الْبَيِّنَةُ } الواضحة، والبرهان الساطع.
