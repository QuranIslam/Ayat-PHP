قَالُوا ادْعُ لَنَا رَبَّكَ يُبَيِّن لَّنَا مَا هِيَ ۚ قَالَ إِنَّهُ يَقُولُ إِنَّهَا بَقَرَةٌ لَّا فَارِضٌ وَلَا بِكْرٌ عَوَانٌ بَيْنَ ذَٰلِكَ ۖ فَافْعَلُوا مَا تُؤْمَرُونَ|||

فلما قال لهم موسى ذلك, علموا أن ذلك صدق فقالوا: { ادْعُ لَنَا رَبَّكَ يُبَيِّنْ لَنَا مَا هِيَ } أي: ما سنها؟ { قَالَ إِنَّهُ يَقُولُ إِنَّهَا بَقَرَةٌ لَا فَارِضٌ } أي: كبيرة { وَلَا بِكْرٌ } أي: صغيرة { عَوَانٌ بَيْنَ ذَلِكَ فَافْعَلُوا مَا تُؤْمَرُونَ } واتركوا التشديد والتعنت.
