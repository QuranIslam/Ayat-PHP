وَأُحِيطَ بِثَمَرِهِ فَأَصْبَحَ يُقَلِّبُ كَفَّيْهِ عَلَىٰ مَا أَنفَقَ فِيهَا وَهِيَ خَاوِيَةٌ عَلَىٰ عُرُوشِهَا وَيَقُولُ يَا لَيْتَنِي لَمْ أُشْرِكْ بِرَبِّي أَحَدًا|||

فاستجاب الله دعاءه { وَأُحِيطَ بِثَمَرِهِ } أي: أصابه عذاب، أحاط به، واستهلكه، فلم يبق منه شيء، والإحاطة بالثمر يستلزم تلف جميع أشجاره، وثماره، وزرعه، فندم كل الندامة، واشتد لذلك أسفه، { فَأَصْبَحَ يُقَلِّبُ كَفَّيْهِ عَلَى مَا أَنْفَقَ فِيهَا } أي على كثرة نفقاته الدنيوية عليها، حيث اضمحلت وتلاشت، فلم يبق لها عوض، وندم أيضا على شركه، وشره، ولهذا قال: { وَيَقُولُ يَا لَيْتَنِي لَمْ أُشْرِكْ بِرَبِّي أَحَدًا }
