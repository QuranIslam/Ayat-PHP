وَجَعَلُوا لِلَّهِ أَندَادًا لِّيُضِلُّوا عَن سَبِيلِهِ ۗ قُلْ تَمَتَّعُوا فَإِنَّ مَصِيرَكُمْ إِلَى النَّارِ|||

{ وَجَعَلُوا لِلَّهِ أَنْدَادًا } أي: نظراء وشركاء { لِيُضِلُّوا عَنْ سَبِيلِهِ } أي: ليضلوا العباد عن سبيل الله بسبب ما جعلوا لله من الأنداد ودعوهم إلى عبادتها، { قُلْ } لهم متوعدا: { تَمَتَّعُوا } بكفركم وضلالكم قليلا، فليس ذلك بنافعكم { فَإِنَّ مَصِيرَكُمْ إِلَى النَّارِ } أي: مآلكم ومقركم ومأواكم فيها وبئس المصير.
