وَأَنزَلَ الَّذِينَ ظَاهَرُوهُم مِّنْ أَهْلِ الْكِتَابِ مِن صَيَاصِيهِمْ وَقَذَفَ فِي قُلُوبِهِمُ الرُّعْبَ فَرِيقًا تَقْتُلُونَ وَتَأْسِرُونَ فَرِيقًا|||

{ وَأَنْزَلَ الَّذِينَ ظَاهَرُوهُمْ } أي عاونوهم { مِنْ أَهْلِ الْكِتَابِ } أي: اليهود { مِنْ صَيَاصِيهِمْ } أي: أنزلهم من حصونهم، نزولاً مظفورًا بهم، مجعولين تحت حكم الإسلام.

{ وَقَذَفَ فِي قُلُوبِهِمُ الرُّعْبَ } فلم يقووا على القتال، بل استسلموا وخضعوا وذلوا. { فَرِيقًا تَقْتُلُونَ } وهم الرجال المقاتلون { وَتَأْسِرُونَ فَرِيقًا } مَنْ عداهم من النساء والصبيان.
