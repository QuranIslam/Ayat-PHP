فَقَضَاهُنَّ سَبْعَ سَمَاوَاتٍ فِي يَوْمَيْنِ وَأَوْحَىٰ فِي كُلِّ سَمَاءٍ أَمْرَهَا ۚ وَزَيَّنَّا السَّمَاءَ الدُّنْيَا بِمَصَابِيحَ وَحِفْظًا ۚ ذَٰلِكَ تَقْدِيرُ الْعَزِيزِ الْعَلِيمِ|||

{ فَقَضَاهُنَّ سَبْعَ سَمَاوَاتٍ فِي يَوْمَيْنِ } فَتَمَّ خلق السماوات والأرض في ستة أيام، أولها يوم الأحد، وآخرها يوم الجمعة، مع أن قدرة اللّه ومشيئته صالحة لخلق الجميع في لحظة واحدة، ولكن مع أنه قدير، فهو حكيم رفيق، فمن حكمته ورفقه، أن جعل خلقها في هذه المدة المقدرة.

واعلم أن ظاهر هذه الآية، مع قوله تعالى في النازعات، لما ذكر خلق السماوات قال: { وَالْأَرْضَ بَعْدَ ذَلِكَ دَحَاهَا } يظهر منهما التعارض، مع أن كتاب اللّه، لا تعارض فيه ولا اختلاف.

والجواب عن ذلك، ما قاله كثير من السلف، أن خلق الأرض وصورتها متقدم على خلق السماوات كما هنا، ودحي الأرض بأن { أَخْرَجَ مِنْهَا مَاءَهَا وَمَرْعَاهَا وَالْجِبَالَ أَرْسَاهَا } متأخر عن خلق السماوات كما في سورة النازعات، ولهذا قال فيها: { وَالْأَرْضَ بَعْدَ ذَلِكَ دَحَاهَا أَخْرَجَ مِنْهَا } إلى آخره ولم يقل: "والأرض بعد ذلك خلقها"

وقوله: { وَأَوْحَى فِي كُلِّ سَمَاءٍ أَمْرَهَا } أي: الأمر والتدبير اللائق بها، الذي اقتضته حكمة أحكم الحاكمين.

{ وَزَيَّنَّا السَّمَاءَ الدُّنْيَا بِمَصَابِيحَ } هي: النجوم، يستنار، بها، ويهتدى، وتكون زينة وجمالاً للسماء ظاهرًا، وجمالاً لها، باطنًا، بجعلها رجومًا للشياطين، لئلا يسترق السمع فيها. { ذَلِكَ } المذكور، من الأرض وما فيها، والسماء وما فيها { تَقْدِيرُ الْعَزِيزِ } الذي عزته، قهر بها الأشياء ودبرها، وخلق بها المخلوقات. { الْعَلِيمِ } الذي أحاط علمه بالمخلوقات، الغائب والشاهد.

فَتَرْكُ المشركين الإخلاص لهذا الرب العظيم الواحد القهار، الذي انقادت المخلوقات لأمره ونفذ فيها قدره، من أعجب الأشياء، واتخاذهم له أندادًا يسوونهم به، وهم ناقصون في أوصافهم وأفعالهم، أعجب، وأعجب، ولا دواء لهؤلاء، إن استمر إعراضهم، إلا العقوبات الدنيوية والأخروية.
