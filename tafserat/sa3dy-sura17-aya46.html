وَجَعَلْنَا عَلَىٰ قُلُوبِهِمْ أَكِنَّةً أَن يَفْقَهُوهُ وَفِي آذَانِهِمْ وَقْرًا ۚ وَإِذَا ذَكَرْتَ رَبَّكَ فِي الْقُرْآنِ وَحْدَهُ وَلَّوْا عَلَىٰ أَدْبَارِهِمْ نُفُورًا|||

{ وَجَعَلْنَا عَلَى قُلُوبِهِمْ أَكِنَّةً } أي: أغطية وأغشية لا يفقهون معها القرآن بل يسمعونه سماعا تقوم به عليهم الحجة، { وَفِي آذَانِهِمْ وَقْرًا } أي: صمما عن سماعه، { وَإِذَا ذَكَرْتَ رَبَّكَ فِي الْقُرْآن } داعيا لتوحيده ناهيا عن الشرك به.

{ وَلَّوْا عَلَى أَدْبَارِهِمْ نُفُورًا } من شدة بغضهم له ومحبتهم لما هم عليه من الباطل، كما قال تعالى: { وَإِذَا ذُكِرَ اللَّهُ وَحْدَهُ اشْمَأَزَّتْ قُلُوبُ الَّذِينَ لَا يُؤْمِنُونَ بِالْآخِرَةِ وَإِذَا ذُكِرَ الَّذِينَ مِنْ دُونِهِ إِذَا هُمْ يَسْتَبْشِرُونَ }
