وَجِيءَ يَوْمَئِذٍ بِجَهَنَّمَ ۚ يَوْمَئِذٍ يَتَذَكَّرُ الْإِنسَانُ وَأَنَّىٰ لَهُ الذِّكْرَىٰ|||

{ وَجِيءَ يَوْمَئِذٍ بِجَهَنَّمَ } تقودها الملائكة بالسلاسل.

فإذا وقعت هذه الأمور فـ { يَوْمَئِذٍ يَتَذَكَّرُ الْإِنْسَانُ } ما قدمه من خير وشر.

{ وَأَنَّى لَهُ الذِّكْرَى } فقد فات أوانها، وذهب زمانها.
