وَجَعَلَهَا كَلِمَةً بَاقِيَةً فِي عَقِبِهِ لَعَلَّهُمْ يَرْجِعُونَ|||

{ وَجَعَلَهَا } أي: هذه الخصلة الحميدة، التي هي أم الخصال وأساسها، وهي إخلاص العبادة للّه وحده، والتبرِّي من عبادة ما سواه.

{ كَلِمَةً بَاقِيَةً فِي عَقِبِهِ } أي: ذريته { لَعَلَّهُمْ } إليها { يَرْجِعُونَ } لشهرتها عنه، وتوصيته لذريته، وتوصية بعض بنيه -كإسحاق ويعقوب- لبعض، كما قال تعالى: { وَمَنْ يَرْغَبُ عَنْ مِلَّةِ إِبْرَاهِيمَ إِلَّا مَنْ سَفِهَ نَفْسَهُ } إلى آخر الآيات.

فلم تزل هذه الكلمة موجودة في ذريته عليه السلام حتى دخلهم الترف والطغيان.
