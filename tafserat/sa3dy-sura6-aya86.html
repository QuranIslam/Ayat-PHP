وَإِسْمَاعِيلَ وَالْيَسَعَ وَيُونُسَ وَلُوطًا ۚ وَكُلًّا فَضَّلْنَا عَلَى الْعَالَمِينَ|||

{ وَإِسْمَاعِيلَ } بن إبراهيم أبو الشعب الذي هو أفضل الشعوب، وهو الشعب العربي، ووالد سيد ولد آدم، محمد صلى الله عليه وسلم. { وَيُونُسَ } بن متى { وَلُوطًا } بن هاران، أخي إبراهيم. { وَكُلَا } من هؤلاء الأنبياء والمرسلين { فَضَّلْنَا عَلَى الْعَالَمِينَ } لأن درجات الفضائل أربع – وهي التي ذكرها الله بقوله: { وَمَنْ يُطِعِ اللَّهَ وَالرَّسُولَ فَأُولَئِكَ مَعَ الَّذِينَ أَنْعَمَ اللَّهُ عَلَيْهِمْ مِنَ النَّبِيِّينَ وَالصِّدِّيقِينَ وَالشُّهَدَاءِ وَالصَّالِحِينَ } فهؤلاء من الدرجة العليا، بل هم أفضل الرسل على الإطلاق، فالرسل الذين قصهم الله في كتابه، أفضل ممن لم يقص علينا نبأهم بلا شك.
