۞ أَلَمْ أَعْهَدْ إِلَيْكُمْ يَا بَنِي آدَمَ أَن لَّا تَعْبُدُوا الشَّيْطَانَ ۖ إِنَّهُ لَكُمْ عَدُوٌّ مُّبِينٌ|||

{ أَلَمْ أَعْهَدْ إِلَيْكُمْ } أي: آمركم وأوصيكم، على ألسنة رسلي، [وأقول لكم:] { يَا بَنِي آدَمَ أَنْ لَا تَعْبُدُوا الشَّيْطَانَ } أي: لا تطيعوه؟ وهذا التوبيخ، يدخل فيه التوبيخ عن جميع أنواع الكفر والمعاصي، لأنها كلها طاعة للشيطان وعبادة له، { إِنَّهُ لَكُمْ عَدُوٌّ مُبِينٌ } فحذرتكم منه غاية التحذير، وأنذرتكم عن طاعته، وأخبرتكم بما يدعوكم إليه.
