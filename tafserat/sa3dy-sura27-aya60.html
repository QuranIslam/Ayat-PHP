أَمَّنْ خَلَقَ السَّمَاوَاتِ وَالْأَرْضَ وَأَنزَلَ لَكُم مِّنَ السَّمَاءِ مَاءً فَأَنبَتْنَا بِهِ حَدَائِقَ ذَاتَ بَهْجَةٍ مَّا كَانَ لَكُمْ أَن تُنبِتُوا شَجَرَهَا ۗ أَإِلَٰهٌ مَّعَ اللَّهِ ۚ بَلْ هُمْ قَوْمٌ يَعْدِلُونَ|||

أي: أمن خلق السماوات وما فيها من الشمس والقمر والنجوم والملائكة والأرض وما فيها من جبال وبحار وأنهار وأشجار وغير ذلك.

{ وَأَنْزَلَ لَكُمْ ْ} أي: لأجلكم { مِنَ السَّمَاءِ مَاءً فَأَنْبَتْنَا بِهِ حَدَائِقَ ْ} أي: بساتين { ذَاتَ بَهْجَةٍ } أي: حسن منظر من كثرة أشجارها وتنوعها وحسن ثمارها، { مَا كَانَ لَكُمْ أَنْ تُنْبِتُوا شَجَرَهَا ْ} لولا منة الله عليكم بإنزال المطر. { أَإِلَهٌ مَعَ اللَّهِ ْ} فعل هذه الأفعال حتى يعبد معه ويشرك به؟ { بَلْ هُمْ قَوْمٌ يَعْدِلُونَ ْ} به غيره ويسوون به سواه مع علمهم أنه وحده خالق العالم العلوي والسفلي ومنزل الرزق.
