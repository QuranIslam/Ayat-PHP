يُسَبِّحُ لِلَّهِ مَا فِي السَّمَاوَاتِ وَمَا فِي الْأَرْضِ الْمَلِكِ الْقُدُّوسِ الْعَزِيزِ الْحَكِيمِ|||

أي: يسبح لله، وينقاد لأمره، ويتألهه، ويعبده، جميع ما في السماوات والأرض، لأنه الكامل الملك، الذي له ملك العالم العلوي والسفلي، فالجميع مماليكه، وتحت تدبيره، { الْقُدُّوسُ } المعظم، المنزه عن كل آفة ونقص، { الْعَزِيزُ } القاهر للأشياء كلها، { الْحَكِيمُ } في خلقه وأمره.

فهذه الأوصاف العظيمة مما تدعو إلى عبادة الله وحده لا شريك له.
