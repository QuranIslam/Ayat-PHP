قَالَ أَرَأَيْتَكَ هَٰذَا الَّذِي كَرَّمْتَ عَلَيَّ لَئِنْ أَخَّرْتَنِ إِلَىٰ يَوْمِ الْقِيَامَةِ لَأَحْتَنِكَنَّ ذُرِّيَّتَهُ إِلَّا قَلِيلًا|||

فلما تبين لإبليس تفضيل الله لآدم { قَالَ } مخاطبا لله: { أَرَأَيْتَكَ هَذَا الَّذِي كَرَّمْتَ عَلَيَّ لَئِنْ أَخَّرْتَنِ إِلَى يَوْمِ الْقِيَامَةِ لَأَحْتَنِكَنَّ ذُرِّيَّتَهُ } أي: لأستأصلنهم بالإضلال ولأغوينهم { إِلَّا قَلِيلًا } عرف الخبيث أنه لا بد أن يكون منهم من يعاديه ويعصيه.
