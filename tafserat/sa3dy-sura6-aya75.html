وَكَذَٰلِكَ نُرِي إِبْرَاهِيمَ مَلَكُوتَ السَّمَاوَاتِ وَالْأَرْضِ وَلِيَكُونَ مِنَ الْمُوقِنِينَ|||

{ وَكَذَلِكَ } حين وفقناه للتوحيد والدعوة إليه { نُرِي إِبْرَاهِيمَ مَلَكُوتَ السَّمَاوَاتِ وَالْأَرْضِ } أي: ليرى ببصيرته، ما اشتملت عليه من الأدلة القاطعة، والبراهين الساطعة { وَلِيَكُونَ مِنَ الْمُوقِنِينَ } فإنه بحسب قيام الأدلة، يحصل له الإيقان والعلم التام بجميع المطالب.
