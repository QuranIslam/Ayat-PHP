وَأَنْ أَقِيمُوا الصَّلَاةَ وَاتَّقُوهُ ۚ وَهُوَ الَّذِي إِلَيْهِ تُحْشَرُونَ|||

{ وَأَنْ أَقِيمُوا الصَّلَاةَ } أي: وأمرنا أن نقيم الصلاة بأركانها وشروطها وسننها ومكملاتها. { وَاتَّقُوهُ } بفعل ما أمر به، واجتناب ما عنه نهى. { وَهُوَ الَّذِي إِلَيْهِ تُحْشَرُونَ } أي: تُجْمَعون ليوم القيامة، فيجازيكم بأعمالكم، خيرها وشرها.
