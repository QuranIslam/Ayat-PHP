فَلَمَّا أَن جَاءَ الْبَشِيرُ أَلْقَاهُ عَلَىٰ وَجْهِهِ فَارْتَدَّ بَصِيرًا ۖ قَالَ أَلَمْ أَقُل لَّكُمْ إِنِّي أَعْلَمُ مِنَ اللَّهِ مَا لَا تَعْلَمُونَ|||

{ فَلَمَّا أَنْ جَاءَ الْبَشِيرُ } بقرب الاجتماع بيوسف وإخوته وأبيهم، { أَلْقَاهُ } أي: القميص { عَلَى وَجْهِهِ فَارْتَدَّ بَصِيرًا } أي: رجع على حاله الأولى بصيرا، بعد أن ابيضت عيناه من الحزن، فقال لمن حضره من أولاده وأهله الذين كانوا يفندون رأيه، ويتعجبون منه منتصرا عليهم، متبجحا بنعمة الله عليه: { أَلَمْ أَقُلْ لَكُمْ إِنِّي أَعْلَمُ مِنَ اللَّهِ مَا لَا تَعْلَمُونَ } حيث كنت مترجيا للقاء يوسف، مترقبا لزوال الهم والغم والحزن.
