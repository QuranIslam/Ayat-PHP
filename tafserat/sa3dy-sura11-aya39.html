فَسَوْفَ تَعْلَمُونَ مَن يَأْتِيهِ عَذَابٌ يُخْزِيهِ وَيَحِلُّ عَلَيْهِ عَذَابٌ مُّقِيمٌ|||

تفسير الآيتين 38 و 39 :ـ

فامتثل أمر ربه، وجعل يصنع الفلك { وَكُلَّمَا مَرَّ عَلَيْهِ مَلَأٌ مِنْ قَوْمِهِ } ورأوا ما يصنع { سَخِرُوا مِنْهُ قَالَ إِنْ تَسْخَرُوا مِنَّا } الآن { فَإِنَّا نَسْخَرُ مِنْكُمْ كَمَا تَسْخَرُونَ * فَسَوْفَ تَعْلَمُونَ مَنْ يَأْتِيهِ عَذَابٌ يُخْزِيهِ وَيَحِلُّ عَلَيْهِ عَذَابٌ مُقِيمٌ } نحن أم أنتم. وقد علموا ذلك، حين حل بهم العقاب.
