وَأُخْرَىٰ لَمْ تَقْدِرُوا عَلَيْهَا قَدْ أَحَاطَ اللَّهُ بِهَا ۚ وَكَانَ اللَّهُ عَلَىٰ كُلِّ شَيْءٍ قَدِيرًا|||

{ وَأُخْرَى } أي: وعدكم أيضا غنيمة أخرى { لَمْ تَقْدِرُوا عَلَيْهَا } وقت هذا الخطاب، { قَدْ أَحَاطَ اللَّهُ بِهَا } أي: هو قادر عليها، وتحت تدبيره وملكه، وقد وعدكموها، فلا بد من وقوع ما وعد به، لكمال اقتدار الله تعالى، ولهذا قال: { وَكَانَ اللَّهُ عَلَى كُلِّ شَيْءٍ قَدِيرًا }
