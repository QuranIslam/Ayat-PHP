يَسْأَلُكَ النَّاسُ عَنِ السَّاعَةِ ۖ قُلْ إِنَّمَا عِلْمُهَا عِندَ اللَّهِ ۚ وَمَا يُدْرِيكَ لَعَلَّ السَّاعَةَ تَكُونُ قَرِيبًا|||

أي: يستخبرك الناس عن الساعة، استعجالاً لها، وبعضهم، تكذيبًا لوقوعها، وتعجيزًا للذي أخبر بها. { قُلْ } لهم: { إِنَّمَا عِلْمُهَا عِنْدَ اللَّهِ } أي: لا يعلمها إلا اللّه، فليس لي، ولا لغيرى بها علم، ومع هذا، فلا تستبطؤوها.

{ وَمَا يُدْرِيكَ لَعَلَّ السَّاعَةَ تَكُونُ قَرِيبًا } ومجرد مجيء الساعة، قرباً وبعدًا، ليس تحته نتيجة ولا فائدة، وإنما النتيجة والخسار، والربح، والشقا والسعادة، هل يستحق العبد العذاب، أو يستحق الثواب؟ فهذه سأخبركم بها، وأصف لكم مستحقها.

فوصف مستحق العذاب، ووصف العذاب، لأن الوصف المذكور، منطبق على هؤلاء المكذبين بالساعة .
