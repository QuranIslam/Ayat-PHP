۞ يَوْمَ تَأْتِي كُلُّ نَفْسٍ تُجَادِلُ عَن نَّفْسِهَا وَتُوَفَّىٰ كُلُّ نَفْسٍ مَّا عَمِلَتْ وَهُمْ لَا يُظْلَمُونَ|||

في يوم القيامة حين { تَأْتِي كُلُّ نَفْسٍ تُجَادِلُ عَنْ نَفْسِهَا } كلٌّ يقول نفسي نفسي لا يهمه سوى نفسه، ففي ذلك اليوم يفتقر العبد إلى حصول مثقال ذرة من الخير.

{ وَتُوَفَّى كُلُّ نَفْسٍ مَا عَمِلَتْ } من خير وشر { وَهُمْ لَا يُظْلَمُونَ } فلا يزاد في سيئاتهم ولا ينقص من حسناتهم { فَالْيَوْمَ لَا تُظْلَمُ نَفْسٌ شَيْئًا وَلَا تُجْزَوْنَ إِلَّا مَا كُنْتُمْ تَعْمَلُونَ }
