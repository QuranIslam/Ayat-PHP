مُّتَّكِئِينَ فِيهَا عَلَى الْأَرَائِكِ ۖ لَا يَرَوْنَ فِيهَا شَمْسًا وَلَا زَمْهَرِيرًا|||

{ مُتَّكِئِينَ فِيهَا عَلَى الْأَرَائِكِ } الاتكاء: التمكن من الجلوس، في حال الرفاهية والطمأنينة [الراحة]، والأرائك هي السرر التي عليها اللباس المزين، { لَا يَرَوْنَ فِيهَا } أي: في الجنة { شَمْسًا } يضرهم حرها { وَلَا زَمْهَرِيرًا } أي: بردا شديدا، بل جميع أوقاتهم في ظل ظليل، لا حر ولا برد، بحيث تلتذ به الأجساد، ولا تتألم من حر ولا برد.
