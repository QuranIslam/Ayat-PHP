قَالَ الْمَلَأُ الَّذِينَ اسْتَكْبَرُوا مِن قَوْمِهِ لِلَّذِينَ اسْتُضْعِفُوا لِمَنْ آمَنَ مِنْهُمْ أَتَعْلَمُونَ أَنَّ صَالِحًا مُّرْسَلٌ مِّن رَّبِّهِ ۚ قَالُوا إِنَّا بِمَا أُرْسِلَ بِهِ مُؤْمِنُونَ|||

{ قَالَ الْمَلَأُ الَّذِينَ اسْتَكْبَرُوا مِنْ قَوْمِهِ } أي: الرؤساء والأشراف الذين تكبروا عن الحق، { لِلَّذِينَ اسْتُضْعِفُوا } ولما كان المستضعفون ليسوا كلهم مؤمنين، قالوا { لِمَنْ آمَنَ مِنْهُمْ أَتَعْلَمُونَ أَنَّ صَالِحًا مُرْسَلٌ مِنْ رَبِّهِ } أي: أهو صادق أم كاذب؟. فقال المستضعفون: { إِنَّا بِمَا أُرْسِلَ بِهِ مُؤْمِنُونَ } من توحيد اللّه والخبر عنه وأمره ونهيه.
