لَّا ظَلِيلٍ وَلَا يُغْنِي مِنَ اللَّهَبِ|||

{ لَا ظَلِيلٍ } ذلك الظل أي: لا راحة فيه ولا طمأنينة، { وَلَا يُغْنِي } من مكث فيه { مِنَ اللَّهَبِ } بل اللهب قد أحاط به، يمنة ويسرة ومن كل جانب، كما قال تعالى: { لَهُمْ مِنْ فَوْقِهِمْ ظُلَلٌ مِنَ النَّارِ وَمِنْ تَحْتِهِمْ ظُلَلٌ }

{ لهم من جهنم مهاد ومن فوقهم غواش وكذلك نجزي الظالمين }
