يَوْمَ يَخْرُجُونَ مِنَ الْأَجْدَاثِ سِرَاعًا كَأَنَّهُمْ إِلَىٰ نُصُبٍ يُوفِضُونَ|||

ثم ذكر حال الخلق حين يلاقون يومهم الذي يوعدون، فقال: { يَوْمَ يَخْرُجُونَ مِنَ الْأَجْدَاثِ } أي: القبور، { سِرَاعًا } مجيبين لدعوة الداعي، مهطعين إليها { كَأَنَّهُمْ إِلَى نُصُبٍ يُوفِضُونَ } أي: [كأنهم إلى علم] يؤمون ويسرعون أي: فلا يتمكنون من الاستعصاء للداعي، والالتواء لنداء المنادي، بل يأتون أذلاء مقهورين للقيام بين يدي رب العالمين.
