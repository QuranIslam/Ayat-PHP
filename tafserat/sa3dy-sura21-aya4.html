قَالَ رَبِّي يَعْلَمُ الْقَوْلَ فِي السَّمَاءِ وَالْأَرْضِ ۖ وَهُوَ السَّمِيعُ الْعَلِيمُ|||

{ قَلَ رَبِّي يَعْلَمُ الْقَوْلَ } أي: الخفي والجلي { فِي السَّمَاءِ وَالْأَرْضِ } أي: في جميع ما احتوت عليه أقطارهما { وَهُوَ السَّمِيعُ } لسائر الأصوات، باختلاف اللغات، على تفنن الحاجات { الْعَلِيمُ } بما في الضمائر، وأكنته السرائر.
