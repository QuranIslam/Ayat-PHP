وَلَوْ دُخِلَتْ عَلَيْهِم مِّنْ أَقْطَارِهَا ثُمَّ سُئِلُوا الْفِتْنَةَ لَآتَوْهَا وَمَا تَلَبَّثُوا بِهَا إِلَّا يَسِيرًا|||

{ وَلَوْ دُخِلَتْ عَلَيْهِمْ } المدينة { مِنْ أَقْطَارِهَا } أي: لو دخل الكفار إليها من نواحيها، واستولوا عليها -لا كان ذلك- { ثُمَّ } سئل هؤلاء { الْفِتْنَة } أي: الانقلاب عن دينهم، والرجوع إلى دين المستولين المتغلبين { لَآتَوْهَا } أي: لأعطوها مبادرين.

{ وَمَا تَلَبَّثُوا بِهَا إِلَّا يَسِيرًا } أي: ليس لهم منعة ولا تَصلُّبٌ على الدين، بل بمجرد ما تكون الدولة للأعداء، يعطونهم ما طلبوا، ويوافقونهم على كفرهم، هذه حالهم.
