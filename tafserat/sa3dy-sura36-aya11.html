إِنَّمَا تُنذِرُ مَنِ اتَّبَعَ الذِّكْرَ وَخَشِيَ الرَّحْمَٰنَ بِالْغَيْبِ ۖ فَبَشِّرْهُ بِمَغْفِرَةٍ وَأَجْرٍ كَرِيمٍ|||

والقسم الثاني: الذين قبلوا النذارة، وقد ذكرهم بقوله: { إِنَّمَا تُنْذِرُ } أي: إنما تنفع نذارتك، ويتعظ بنصحك { مَنِ اتَّبَعَ الذِّكْرَ } [أي:] من قصده اتباع الحق وما ذكر به، { وَخَشِيَ الرَّحْمَنَ بِالْغَيْبِ } أي: من اتصف بهذين الأمرين، القصد الحسن في طلب الحق، وخشية اللّه تعالى، فهم الذين ينتفعون برسالتك، ويزكون بتعليمك، وهذا الذي وفق لهذين الأمرين { فَبَشِّرْهُ بِمَغْفِرَةٍ } لذنوبه، { وَأَجْرٍ كَرِيمٍ } لأعماله الصالحة، ونيته الحسنة.
