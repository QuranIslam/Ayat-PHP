ثُمَّ يَأْتِي مِن بَعْدِ ذَٰلِكَ سَبْعٌ شِدَادٌ يَأْكُلْنَ مَا قَدَّمْتُمْ لَهُنَّ إِلَّا قَلِيلًا مِّمَّا تُحْصِنُونَ|||

{ ثُمَّ يَأْتِي مِنْ بَعْدِ ذَلِكَ } أي: بعد تلك السنين السبع المخصبات. { سَبْعٌ شِدَادٌ } أي: مجدبات جدا { يَأْكُلْنَ مَا قَدَّمْتُمْ لَهُنَّ } أي: يأكلن جميع ما ادخرتموه ولو كان كثيرا. { إِلَّا قَلِيلًا مِمَّا تُحْصِنُونَ } أي: تمنعونه من التقديم لهن.
