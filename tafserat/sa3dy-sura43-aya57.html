۞ وَلَمَّا ضُرِبَ ابْنُ مَرْيَمَ مَثَلًا إِذَا قَوْمُكَ مِنْهُ يَصِدُّونَ|||

يقول تعالى: { وَلَمَّا ضُرِبَ ابْنُ مَرْيَمَ مَثَلًا } أي: نهي عن عبادته، وجعلت عبادته بمنزلة عبادة الأصنام والأنداد. { إِذَا قَوْمُكَ } المكذبون لك { مِنْهُ } أي: من أجل هذا المثل المضروب، { يَصِدُّونَ } أي: يستلجون في خصومتهم لك، ويصيحون، ويزعمون أنهم قد غلبوا في حجتهم، وأفلجوا.
