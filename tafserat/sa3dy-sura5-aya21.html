يَا قَوْمِ ادْخُلُوا الْأَرْضَ الْمُقَدَّسَةَ الَّتِي كَتَبَ اللَّهُ لَكُمْ وَلَا تَرْتَدُّوا عَلَىٰ أَدْبَارِكُمْ فَتَنقَلِبُوا خَاسِرِينَ|||

{ يَا قَوْمِ ادْخُلُوا الْأَرْضَ الْمُقَدَّسَةَ } أي: المطهرة { الَّتِي كَتَبَ اللَّهُ لَكُمْ } فأخبرهم خبرا تطمئن به أنفسهم، إن كانوا مؤمنين مصدقين بخبر الله، وأنه قد كتب الله لهم دخولها، وانتصارهم على عدوهم. { وَلَا تَرْتَدُّوا } أي: ترجعوا { عَلَى أَدْبَارِكُمْ فَتَنْقَلِبُوا خَاسِرِينَ } قد خسرتم دنياكم بما فاتكم من النصر على الأعداء وفتح بلادكم. وآخرتكم بما فاتكم من الثواب، وما استحققتم -بمعصيتكم- من العقاب،
