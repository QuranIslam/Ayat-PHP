وَقَالَ مُوسَىٰ يَا قَوْمِ إِن كُنتُمْ آمَنتُم بِاللَّهِ فَعَلَيْهِ تَوَكَّلُوا إِن كُنتُم مُّسْلِمِينَ|||

{ وَقَالَ مُوسَى } موصيًا لقومه بالصبر، ومذكرًا لهم ما يستعينون به على ذلك فقال: { يَا قَوْمِ إِنْ كُنْتُمْ آمَنْتُمْ بِاللَّهِ } فقوموا بوظيفة الإيمان.

{ فَعَلَيْهِ تَوَكَّلُوا إِنْ كُنْتُمْ مُسْلِمِينَ } أي: اعتمدوا عليه، والجؤوا إليه واستنصروه.
