عَالِمُ الْغَيْبِ وَالشَّهَادَةِ الْعَزِيزُ الْحَكِيمُ|||

{ عَالِمُ الْغَيْبِ وَالشَّهَادَةِ } أي: ما غاب عن العباد من الجنود التي لا يعلمها إلا هو، وما يشاهدونه من المخلوقات، { الْعَزِيزُ } الذي لا يغالب ولا يمانع، الذي قهر كل الأشياء، { الْحَكِيمُ } في خلقه وأمره، الذي يضع الأشياء مواضعها.

تم تفسير سورة التغابن [ولله الحمد].
