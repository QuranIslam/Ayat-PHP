أُولَٰئِكَ الَّذِينَ حَقَّ عَلَيْهِمُ الْقَوْلُ فِي أُمَمٍ قَدْ خَلَتْ مِن قَبْلِهِم مِّنَ الْجِنِّ وَالْإِنسِ ۖ إِنَّهُمْ كَانُوا خَاسِرِينَ|||

{ أُولَئِكَ الَّذِينَ } بهذه الحالة الذميمة { حَقَّ عَلَيْهِمُ الْقَوْلُ } أي: حقت عليهم كلمة العذاب { فِي } جملة { أُمَمٍ قَدْ خَلَتْ مِنْ قَبْلِهِمْ مِنَ الْجِنِّ وَالْإِنْسِ } على الكفر والتكذيب فسيدخل هؤلاء في غمارهم وسيغرقون في تيارهم.

{ إِنَّهُمْ كَانُوا خَاسِرِينَ } والخسران فوات رأس مال الإنسان، وإذا فقد رأس ماله فالأرباح من باب أولى وأحرى، فهم قد فاتهم الإيمان ولم يحصلوا على شيء من النعيم ولا سلموا من عذاب الجحيم.
