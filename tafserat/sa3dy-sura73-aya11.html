وَذَرْنِي وَالْمُكَذِّبِينَ أُولِي النَّعْمَةِ وَمَهِّلْهُمْ قَلِيلًا|||

{ وَذَرْنِي وَالْمُكَذِّبِينَ } أي: اتركني وإياهم، فسأنتقم منهم، وإن أمهلتهم فلا أهملهم، وقوله: { أُولِي النَّعْمَةِ } أي: أصحاب النعمة والغنى، الذين طغوا حين وسع الله عليهم من رزقه، وأمدهم من فضله كما قال تعالى: { كَلَّا إِنَّ الْإِنْسَانَ لَيَطْغَى أَنْ رَآهُ اسْتَغْنَى } .
