إِن يَثْقَفُوكُمْ يَكُونُوا لَكُمْ أَعْدَاءً وَيَبْسُطُوا إِلَيْكُمْ أَيْدِيَهُمْ وَأَلْسِنَتَهُم بِالسُّوءِ وَوَدُّوا لَوْ تَكْفُرُونَ|||

ثم بين تعالى شدة عداوتهم، تهييجا للمؤمنين على عداوتهم، { إِنْ يَثْقَفُوكُمْ } أي: يجدوكم، وتسنح لهم الفرصة في أذاكم، { يَكُونُوا لَكُمْ أَعْدَاءً } ظاهرين { وَيَبْسُطُوا إِلَيْكُمْ أَيْدِيَهُمْ } بالقتل والضرب، ونحو ذلك.

{ وَأَلْسِنَتَهُمْ بِالسُّوءِ } أي: بالقول الذي يسوء، من شتم وغيره، { وَوَدُّوا لَوْ تَكْفُرُونَ } فإن هذا غاية ما يريدون منكم.
