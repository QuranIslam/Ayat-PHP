هُوَ اللَّهُ الَّذِي لَا إِلَٰهَ إِلَّا هُوَ الْمَلِكُ الْقُدُّوسُ السَّلَامُ الْمُؤْمِنُ الْمُهَيْمِنُ الْعَزِيزُ الْجَبَّارُ الْمُتَكَبِّرُ ۚ سُبْحَانَ اللَّهِ عَمَّا يُشْرِكُونَ|||

ثم كرر [ذكر] عموم إلهيته وانفراده بها، وأنه المالك لجميع الممالك، فالعالم العلوي والسفلي وأهله، الجميع، مماليك لله، فقراء مدبرون.

{ الْقُدُّوسُ السَّلَامُ } أي: المقدس السالم من كل عيب وآفة ونقص، المعظم الممجد، لأن القدوس يدل على التنزيه عن كل نقص، والتعظيم لله في أوصافه وجلاله.

{ الْمُؤْمِنُ } أي: المصدق لرسله وأنبيائه بما جاءوا به، بالآيات البينات، والبراهين القاطعات، والحجج الواضحات.

{ الْعَزِيزُ } الذي لا يغالب ولا يمانع، بل قد قهر كل شيء، وخضع له كل شيء، { الْجَبَّارُ } الذي قهر جميع العباد، وأذعن له سائر الخلق، الذي يجبر الكسير، ويغني الفقير، { الْمُتَكَبِّرُ } الذي له الكبرياء والعظمة، المتنزه عن جميع العيوب والظلم والجور.

{ سُبْحَانَ اللَّهِ عَمَّا يُشْرِكُونَ } وهذا تنزيه عام عن كل ما وصفه به من أشرك به وعانده.
