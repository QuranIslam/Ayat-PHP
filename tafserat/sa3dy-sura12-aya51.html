قَالَ مَا خَطْبُكُنَّ إِذْ رَاوَدتُّنَّ يُوسُفَ عَن نَّفْسِهِ ۚ قُلْنَ حَاشَ لِلَّهِ مَا عَلِمْنَا عَلَيْهِ مِن سُوءٍ ۚ قَالَتِ امْرَأَتُ الْعَزِيزِ الْآنَ حَصْحَصَ الْحَقُّ أَنَا رَاوَدتُّهُ عَن نَّفْسِهِ وَإِنَّهُ لَمِنَ الصَّادِقِينَ|||

فأحضرهن الملك، وقال: { مَا خَطْبُكُنَّ } أي: شأنكن { إِذْ رَاوَدْتُنَّ يُوسُفَ عَنْ نَفْسِهِ } فهل رأيتن منه ما يريب؟

فبرَّأنه و { قُلْنَ حَاشَ لِلَّهِ مَا عَلِمْنَا عَلَيْهِ مِنْ سُوءٍ } أي: لا قليل ولا كثير، فحينئذ زال السبب الذي تنبني عليه التهمة، ولم يبق إلا ما عند امرأة العزيز، فـ { قَالَتِ امْرَأَةُ الْعَزِيزِ الْآنَ حَصْحَصَ الْحَقُّ } أي: تمحض وتبين، بعد ما كنا ندخل معه من السوء والتهمة، ما أوجب له السجن { أَنَا رَاوَدْتُهُ عَنْ نَفْسِهِ وَإِنَّهُ لَمِنَ الصَّادِقِينَ } في أقواله وبراءته.
