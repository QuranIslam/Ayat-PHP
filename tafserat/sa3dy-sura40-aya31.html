مِثْلَ دَأْبِ قَوْمِ نُوحٍ وَعَادٍ وَثَمُودَ وَالَّذِينَ مِن بَعْدِهِمْ ۚ وَمَا اللَّهُ يُرِيدُ ظُلْمًا لِّلْعِبَادِ|||

{ مِثْلَ دَأْبِ قَوْمِ نُوحٍ وَعَادٍ وَثَمُودَ وَالَّذِينَ مِنْ بَعْدِهِمْ } أي: مثل عادتهم في الكفر والتكذيب وعادة الله فيهم بالعقوبة العاجلة في الدنيا قبل الآخرة، { وَمَا اللَّهُ يُرِيدُ ظُلْمًا لِلْعِبَادِ } فيعذبهم بغير ذنب أذنبوه، ولا جرم أسلفوه.
