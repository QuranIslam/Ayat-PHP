وَقَالُوا آمَنَّا بِهِ وَأَنَّىٰ لَهُمُ التَّنَاوُشُ مِن مَّكَانٍ بَعِيدٍ|||

{ وَقَالُوا } في تلك الحال: { آمَنَّا } بالله وصدقنا ما به كذبنا { و } لكن { أَنَّى لَهُمُ التَّنَاوُشُ } أي: تناول الإيمان { مِنْ مَكَانٍ بَعِيدٍ } قد حيل بينهم وبينه, وصار من الأمور المحالة في هذه الحالة، فلو أنهم آمنوا وقت الإمكان, لكان إيمانهم مقبولا، ولكنهم { كَفَرُوا بِهِ مِنْ قَبْلُ وَيَقْذِفُونَ }
