وَأَقْسَمُوا بِاللَّهِ جَهْدَ أَيْمَانِهِمْ لَئِن جَاءَهُمْ نَذِيرٌ لَّيَكُونُنَّ أَهْدَىٰ مِنْ إِحْدَى الْأُمَمِ ۖ فَلَمَّا جَاءَهُمْ نَذِيرٌ مَّا زَادَهُمْ إِلَّا نُفُورًا|||

أي وأقسم هؤلاء، الذين كذبوك يا رسول اللّه، قسما اجتهدوا فيه بالأيمان الغليظة. { لَئِنْ جَاءَهُمْ نَذِيرٌ لَيَكُونُنَّ أَهْدَى مِنْ إِحْدَى الْأُمَمِ } أي: أهدى من اليهود والنصارى [أهل الكتب]، فلم يفوا بتلك الإقسامات والعهود.

{ فَلَمَّا جَاءَهُمْ نَذِيرٌ } لم يهتدوا، ولم يصيروا أهدى من إحدى الأمم، بل لم يدوموا على ضلالهم الذي كان، بل { مَا زَادَهُمْ } ذلك { إِلَّا نُفُورًا } وزيادة ضلال وبغي وعناد.
