وَقَالَ إِنِّي ذَاهِبٌ إِلَىٰ رَبِّي سَيَهْدِينِ|||

{ وَ } لما فعلوا فيه هذا الفعل، وأقام عليهم الحجة، وأعذر منهم، { قال إِنِّي ذَاهِبٌ إِلَى رَبِّي } أي: مهاجر إليه، قاصد إلى الأرض المباركة أرض الشام. { سَيَهْدِينِ } يدلني إلى ما فيه الخير لي، من أمر ديني ودنياي، وقال في الآية الأخرى: { وَأَعْتَزِلُكُمْ وَمَا تَدْعُونَ مِنْ دُونِ اللَّهِ وَأَدْعُو رَبِّي عَسَى أَلَّا أَكُونَ بِدُعَاءِ رَبِّي شَقِيًّا }
