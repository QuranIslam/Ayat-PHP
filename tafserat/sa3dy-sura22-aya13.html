يَدْعُو لَمَن ضَرُّهُ أَقْرَبُ مِن نَّفْعِهِ ۚ لَبِئْسَ الْمَوْلَىٰ وَلَبِئْسَ الْعَشِيرُ|||

{ يَدْعُو لَمَنْ ضَرُّهُ أَقْرَبُ مِنْ نَفْعِهِ } فإن ضرره في العقل والبدن والدنيا والآخرة معلوم { لَبِئْسَ الْمَوْلَى } أي: هذا المعبود { وَلَبِئْسَ الْعَشِيرُ } أي: القرين الملازم على صحبته، فإن المقصود من المولى والعشير، حصول النفع، ودفع الضرر، فإذا لم يحصل شيء من هذا، فإنه مذموم ملوم.
