إِنَّمَا يُؤْمِنُ بِآيَاتِنَا الَّذِينَ إِذَا ذُكِّرُوا بِهَا خَرُّوا سُجَّدًا وَسَبَّحُوا بِحَمْدِ رَبِّهِمْ وَهُمْ لَا يَسْتَكْبِرُونَ ۩|||

لما ذكر تعالى الكافرين بآياته، وما أعد لهم من العذاب، ذكر المؤمنين بها، ووصفهم، وما أعد لهم من الثواب، فقال: { إِنَّمَا يُؤْمِنُ بِآيَاتِنَا } [أي] إيمانًا حقيقيًا، من يوجد منه شواهد الإيمان، وهم: { الَّذِينَ إِذَا ذُكِّرُوا } بآيات ربهم فتليت عليهم آيات القرآن، وأتتهم النصائح على أيدي رسل اللّه، ودُعُوا إلى التذكر، سمعوها فقبلوها، وانقادوا، و { خَرُّوا سُجَّدًا } أي: خاضعين لها، خضوع ذكر للّه، وفرح بمعرفته.

{ وَسَبَّحُوا بِحَمْدِ رَبِّهِمْ وَهُمْ لَا يَسْتَكْبِرُونَ } لا بقلوبهم، ولا بأبدانهم، فيمتنعون من الانقياد لها، بل متواضعون لها، قد تلقوها بالقبول، والتسليم، وقابلوها بالانشراح والتسليم، وتوصلوا بها إلى مرضاة الرب الرحيم، واهتدوا بها إلى الصراط المستقيم.
