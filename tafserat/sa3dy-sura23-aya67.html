مُسْتَكْبِرِينَ بِهِ سَامِرًا تَهْجُرُونَ|||

{ مُسْتَكْبِرِينَ بِهِ سَامِرًا تَهْجُرُونَ } قال المفسرون معناه: مستكبرين به، الضمير يعود إلى البيت، المعهود عند المخاطبين، أو الحرم، أي: متكبرين على الناس بسببه، تقولون: نحن أهل الحرم، فنحن أفضل من غيرنا وأعلى، { سَامِرًا } أي: جماعة يتحدثون بالليل حول البيت { تَهْجُرُونَ } [أي: تقولون الكلام الهجر الذي هو القبيح في] هذا القرآن.

فالمكذبون كانت طريقتهم في القرآن، الإعراض عنه، ويوصي بعضهم بعضا بذلك { وَقَالَ الَّذِينَ كَفَرُوا لَا تَسْمَعُوا لِهَذَا الْقُرْآنِ وَالْغَوْا فِيهِ لَعَلَّكُمْ تَغْلِبُونَ } وقال الله عنهم: { أَفَمِنْ هَذَا الْحَدِيثِ تَعْجَبُونَ* وَتَضْحَكُونَ وَلَا تَبْكُونَ* وَأَنْتُمْ سَامِدُونَ } { أَمْ يَقُولُونَ تَقَوَّلَهُ } فلما كانوا جامعين لهذه الرذائل، لا جرم حقت عليهم العقوبة، ولما وقعوا فيها، لم يكن لهم ناصر ينصرهم، ولا مغيث ينقذهم، ويوبخون عند ذلك بهذه الأعمال الساقطة
