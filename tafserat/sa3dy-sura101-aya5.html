وَتَكُونُ الْجِبَالُ كَالْعِهْنِ الْمَنفُوشِ|||

وأما الجبال الصم الصلاب، فتكون { كَالْعِهْنِ الْمَنْفُوشِ } أي: كالصوف المنفوش، الذي بقي ضعيفًا جدًا، تطير به أدنى ريح، قال تعالى: { وَتَرَى الْجِبَالَ تَحْسَبُهَا جَامِدَةً وَهِيَ تَمُرُّ مَرَّ السَّحَابِ } ثم بعد ذلك، تكون هباء منثورًا، فتضمحل ولا يبقى منها شيء يشاهد، فحينئذ تنصب الموازين، وينقسم الناس قسمين: سعداء وأشقياء،
