قَالُوا مَا أَنتُمْ إِلَّا بَشَرٌ مِّثْلُنَا وَمَا أَنزَلَ الرَّحْمَٰنُ مِن شَيْءٍ إِنْ أَنتُمْ إِلَّا تَكْذِبُونَ|||

فأجابوهم بالجواب الذي ما زال مشهورا عند من رد دعوة الرسل: فـ { قَالُوا مَا أَنْتُمْ إِلَّا بَشَرٌ مِثْلُنَا } أي: فما الذي فضلكم علينا وخصكم من دوننا؟ قالت الرسل لأممهم: { إِنْ نَحْنُ إِلَّا بَشَرٌ مِثْلُكُمْ وَلَكِنَّ اللَّهَ يَمُنُّ عَلَى مَنْ يَشَاءُ مِنْ عِبَادِهِ }

{ وَمَا أَنْزَلَ الرَّحْمَنُ مِنْ شَيْءٍ } أي: أنكروا عموم الرسالة، ثم أنكروا أيضا المخاطبين لهم، فقالوا: { إِنْ أَنْتُمْ إِلَّا تَكْذِبُونَ }
