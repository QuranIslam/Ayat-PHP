فَلَمَّا اسْتَيْأَسُوا مِنْهُ خَلَصُوا نَجِيًّا ۖ قَالَ كَبِيرُهُمْ أَلَمْ تَعْلَمُوا أَنَّ أَبَاكُمْ قَدْ أَخَذَ عَلَيْكُم مَّوْثِقًا مِّنَ اللَّهِ وَمِن قَبْلُ مَا فَرَّطتُمْ فِي يُوسُفَ ۖ فَلَنْ أَبْرَحَ الْأَرْضَ حَتَّىٰ يَأْذَنَ لِي أَبِي أَوْ يَحْكُمَ اللَّهُ لِي ۖ وَهُوَ خَيْرُ الْحَاكِمِينَ|||

أي: فلما استيأس إخوة يوسف من يوسف أن يسمح لهم بأخيهم { خَلَصُوا نَجِيًّا } أي: اجتمعوا وحدهم، ليس معهم غيرهم، وجعلوا يتناجون فيما بينهم، فـ { قَالَ كَبِيرُهُمْ أَلَمْ تَعْلَمُوا أَنَّ أَبَاكُمْ قَدْ أَخَذَ عَلَيْكُمْ مَوْثِقًا مِنَ اللَّهِ } في حفظه، وأنكم تأتون به إلا أن يحاط بكم { وَمِنْ قَبْلُ مَا فَرَّطْتُمْ فِي يُوسُفَ } ، فاجتمع عليكم الأمران، تفريطكم في يوسف السابق، وعدم إتيانكم بأخيه باللاحق، فليس لي وجه أواجه به أبي.

{ فَلَنْ أَبْرَحَ الْأَرْضَ } أي: سأقيم في هذه الأرض ولا أزال بها { حَتَّى يَأْذَنَ لِي أَبِي أَوْ يَحْكُمَ اللَّهُ لِي } أي: يقدر لي المجيء وحدي، أو مع أخي { وَهُوَ خَيْرُ الْحَاكِمِينَ }
