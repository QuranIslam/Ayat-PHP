فَلَمَّا رَأَىٰ أَيْدِيَهُمْ لَا تَصِلُ إِلَيْهِ نَكِرَهُمْ وَأَوْجَسَ مِنْهُمْ خِيفَةً ۚ قَالُوا لَا تَخَفْ إِنَّا أُرْسِلْنَا إِلَىٰ قَوْمِ لُوطٍ|||

{ فَلَمَّا رَأَى أَيْدِيَهُمْ لَا تَصِلُ إِلَيْهِ } أي: إلى تلك الضيافة { نَكِرَهُمْ وَأَوْجَسَ مِنْهُمْ خِيفَةً } وظن أنهم أتوه بشر ومكروه، وذلك قبل أن يعرف أمرهم.

فـ { قَالُوا لَا تَخَفْ إِنَّا أُرْسِلْنَا إِلَى قَوْمِ لُوطٍ } أي: إنا رسل الله, أرسلنا الله إلى إهلاك قوم لوط.
