أَن تَقُولَ نَفْسٌ يَا حَسْرَتَا عَلَىٰ مَا فَرَّطتُ فِي جَنبِ اللَّهِ وَإِن كُنتُ لَمِنَ السَّاخِرِينَ|||

ثم حذرهم { أَن } يستمروا على غفلتهم، حتى يأتيهم يوم يندمون فيه، ولا تنفع الندامة.و { تَقُولَ نَفْسٌ يَا حَسْرَتَى عَلَى مَا فَرَّطْتُ فِي جَنْبِ اللَّهِ } أي: في جانب حقه. { وَإِنْ كُنْت } في الدنيا { لَمِنَ السَّاخِرِينَ } في إتيان الجزاء، حتى رأيته عيانا.
