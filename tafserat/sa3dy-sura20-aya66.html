قَالَ بَلْ أَلْقُوا ۖ فَإِذَا حِبَالُهُمْ وَعِصِيُّهُمْ يُخَيَّلُ إِلَيْهِ مِن سِحْرِهِمْ أَنَّهَا تَسْعَىٰ|||

فقال لهم موسى: { بَلْ أَلْقُوا } فألقوا حبالهم وعصيهم، { فَإِذَا حِبَالُهُمْ وَعِصِيُّهُمْ يُخَيَّلُ إِلَيْهِ } أي: إلى موسى { مِنْ سِحْرِهِمْ } البليغ { أَنَّهَا تَسْعَى } أي: أنها حيات تسعى فلما خيل إلى موسى ذلك.
