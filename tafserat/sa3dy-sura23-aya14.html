ثُمَّ خَلَقْنَا النُّطْفَةَ عَلَقَةً فَخَلَقْنَا الْعَلَقَةَ مُضْغَةً فَخَلَقْنَا الْمُضْغَةَ عِظَامًا فَكَسَوْنَا الْعِظَامَ لَحْمًا ثُمَّ أَنشَأْنَاهُ خَلْقًا آخَرَ ۚ فَتَبَارَكَ اللَّهُ أَحْسَنُ الْخَالِقِينَ|||

{ ثُمَّ خَلَقْنَا النُّطْفَةَ } التي قد استقرت قبل { عَلَقَةً } أي: دما أحمر، بعد مضي أربعين يوما من النطفة، { فخَلَقْنَا الْعَلَقَةَ } بعد أربعين يوما { مُضْغَةً } أي: قطعة لحم صغيرة، بقدر ما يمضغ من صغرها.

{ فَخَلَقْنَا الْمُضْغَةَ } اللينة { عِظَامًا } صلبة، قد تخللت اللحم، بحسب حاجة البدن إليها، { فَكَسَوْنَا الْعِظَامَ لَحْمًا } أي: جعلنا اللحم، كسوة للعظام، كما جعلنا العظام، عمادا للحم، وذلك في الأربعين الثالثة، { ثُمَّ أَنْشَأْنَاهُ خَلْقًا آخَرَ } نفخ فيه الروح، فانتقل من كونه جمادا، إلى أن صار حيوانا، { فَتَبَارَكَ اللَّهُ } أي: تعالى وتعاظم وكثر خيره { أَحْسَنُ الْخَالِقِينَ } { الذي أحسن كل شيء خلقه وبدأ خلق الإنسان من طين ثم جعل نسله من سلالة من ماء مهين ثم سواه ونفخ فيه من روحه وجعل لكم السمع والأبصار والأفئدة قليلا ما تشكرون } فخلقه كله حسن، والإنسان من أحسن مخلوقاته، بل هو أحسنها على الإطلاق، كما قال تعالى: { لَقَدْ خَلَقْنَا الْإِنْسَانَ فِي أَحْسَنِ تَقْوِيمٍ } ولهذا كان خواصه أفضل المخلوقات وأكملها.
