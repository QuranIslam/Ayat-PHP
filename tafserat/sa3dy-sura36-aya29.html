إِن كَانَتْ إِلَّا صَيْحَةً وَاحِدَةً فَإِذَا هُمْ خَامِدُونَ|||

{ إِنْ كَانَتْ } أي: كانت عقوبتهم { إِلَّا صَيْحَةً وَاحِدَةً } أي: صوتا واحدا، تكلم به بعض ملائكة اللّه، { فَإِذَا هُمْ خَامِدُونَ } قد تقطعت قلوبهم في أجوافهم، وانزعجوا لتلك الصيحة، فأصبحوا خامدين، لا صوت ولا حركة، ولا حياة بعد ذلك العتو والاستكبار، ومقابلة أشرف الخلق بذلك الكلام القبيح، وتجبرهم عليهم.
