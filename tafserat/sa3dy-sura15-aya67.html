وَجَاءَ أَهْلُ الْمَدِينَةِ يَسْتَبْشِرُونَ|||

{ وَجَاءَ أَهْلُ الْمَدِينَةِ } أي: المدينة التي فيها قوم لوط { يَسْتَبْشِرُونَ } أي: يبشر بعضهم بعضا بأضياف لوط وصباحة وجوههم واقتدارهم عليهم، وذلك لقصدهم فعل الفاحشة فيهم، فجاءوا حتى وصلوا إلى بيت لوط فجعلوا يعالجون لوطا على أضيافه، ولوط يستعيذ منهم ويقول: { إِنَّ هَؤُلَاءِ ضَيْفِي فَلَا تَفْضَحُون وَاتَّقُوا اللَّهَ وَلَا تُخْزُون }
