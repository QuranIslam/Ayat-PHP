ثُمَّ نُنَجِّي الَّذِينَ اتَّقَوا وَّنَذَرُ الظَّالِمِينَ فِيهَا جِثِيًّا|||

{ ثُمَّ نُنَجِّي الَّذِينَ اتَّقَوْا } الله تعالى بفعل المأمور، واجتناب المحظور { وَنَذَرُ الظَّالِمِينَ } أنفسهم بالكفر والمعاصي { فِيهَا جِثِيًّا } وهذا بسبب ظلمهم وكفرهم، وجب لهم الخلود، وحق عليهم العذاب، وتقطعت بهم الأسباب.
