وَلَئِنْ أَخَّرْنَا عَنْهُمُ الْعَذَابَ إِلَىٰ أُمَّةٍ مَّعْدُودَةٍ لَّيَقُولُنَّ مَا يَحْبِسُهُ ۗ أَلَا يَوْمَ يَأْتِيهِمْ لَيْسَ مَصْرُوفًا عَنْهُمْ وَحَاقَ بِهِم مَّا كَانُوا بِهِ يَسْتَهْزِئُونَ|||

{ وَلَئِنْ أَخَّرْنَا عَنْهُمُ الْعَذَابَ إِلَى أُمَّةٍ مَعْدُودَةٍ } أي: إلى وقت مقدر فتباطأوه، لقالوا من جهلهم وظلمهم { مَا يَحْبِسُهُ } ومضمون هذا تكذيبهم به، فإنهم يستدلون بعدم وقوعه بهم عاجلا على كذب الرسول المخبر بوقوع العذاب، فما أبعد هذا الاستدلال"

{ أَلَا يَوْمَ يَأْتِيهِمْ } العذاب { لَيْسَ مَصْرُوفًا عَنْهُمْ } فيتمكنون من النظر في أمرهم.

{ وَحَاقَ بِهِمْ } أي: نزل { مَا كَانُوا بِهِ يَسْتَهْزِئُونَ } من العذاب، حيث تهاونوا به، حتى جزموا بكذب من جاء به.
