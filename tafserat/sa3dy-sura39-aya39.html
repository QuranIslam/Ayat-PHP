قُلْ يَا قَوْمِ اعْمَلُوا عَلَىٰ مَكَانَتِكُمْ إِنِّي عَامِلٌ ۖ فَسَوْفَ تَعْلَمُونَ|||

أي: { قُلْ } لهم يا أيها الرسول: { يَا قَوْمِ اعْمَلُوا عَلَى مَكَانَتِكُمْ } أي: على حالتكم التي رضيتموها لأنفسكم، من عبادة من لا يستحق من العبادة شيئا ولا له من الأمر شيء.

{ إِنِّي عَامِلٌ } على ما دعوتكم إليه، من إخلاص الدين للّه تعالى وحده. { فَسَوْفَ تَعْلَمُونَ } لمن العاقبة
