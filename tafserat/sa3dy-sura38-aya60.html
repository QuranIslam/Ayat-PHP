قَالُوا بَلْ أَنتُمْ لَا مَرْحَبًا بِكُمْ ۖ أَنتُمْ قَدَّمْتُمُوهُ لَنَا ۖ فَبِئْسَ الْقَرَارُ|||

{ قَالُوا } أي: الفوج المقبل المقتحم: { بَلْ أَنْتُمْ لَا مَرْحَبًا بِكُمْ أَنْتُمْ قَدَّمْتُمُوهُ } أي: العذاب { لَنَا } بدعوتكم لنا، وفتنتكم وإضلالكم وتسببكم. { فَبِئْسَ الْقَرَارُ } قرار الجميع، قرار السوء والشر.
