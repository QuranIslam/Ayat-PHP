وَتَوَكَّلْ عَلَى الْحَيِّ الَّذِي لَا يَمُوتُ وَسَبِّحْ بِحَمْدِهِ ۚ وَكَفَىٰ بِهِ بِذُنُوبِ عِبَادِهِ خَبِيرًا|||

ثم أمره أن يتوكل عليه ويستعين به فقال: { وَتَوَكَّلْ عَلَى الْحَيِّ } الذي له الحياة الكاملة المطلقة { الَّذِي لَا يَمُوتُ وَسَبِّحْ بِحَمْدِهِ } أي: اعبده وتوكل عليه في الأمور المتعلقة بك والمتعلقة بالخلق. { وَكَفَى بِهِ بِذُنُوبِ عِبَادِهِ خَبِيرًا } يعلمها ويجازي عليها.
