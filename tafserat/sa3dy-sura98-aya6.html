إِنَّ الَّذِينَ كَفَرُوا مِنْ أَهْلِ الْكِتَابِ وَالْمُشْرِكِينَ فِي نَارِ جَهَنَّمَ خَالِدِينَ فِيهَا ۚ أُولَٰئِكَ هُمْ شَرُّ الْبَرِيَّةِ|||

ثم ذكر جزاء الكافرين بعدما جاءتهم البينة، فقال: { إِنَّ الَّذِينَ كَفَرُوا مِنْ أَهْلِ الْكِتَابِ وَالْمُشْرِكِينَ فِي نَارِ جَهَنَّمَ } قد أحاط بهم عذابها، واشتد عليهم عقابها، { خَالِدِينَ فِيهَا } لا يفتر عنهم العذاب، وهم فيها مبلسون، { أُولَئِكَ هُمْ شَرُّ الْبَرِيَّةِ } لأنهم عرفوا الحق وتركوه، وخسروا الدنيا والآخرة.
