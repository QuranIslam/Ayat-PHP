وَالَّذِي خَلَقَ الْأَزْوَاجَ كُلَّهَا وَجَعَلَ لَكُم مِّنَ الْفُلْكِ وَالْأَنْعَامِ مَا تَرْكَبُونَ|||

{ وَالَّذِي خَلَقَ الْأَزْوَاجَ كُلَّهَا } أي: الأصناف جميعها، مما تنبت الأرض ومن أنفسهم ومما لا يعلمون، من ليل ونهار، وحر وبرد، وذكر وأنثى، وغير ذلك. { وَجَعَلَ لَكُمْ مِنَ الْفُلْكِ } أي: السفن البحرية، الشراعية والنارية، مَا تَرْكَبُونَ { و } من { الأنعام ما تركبون لِتَسْتَوُوا عَلَى ظُهُورِهِ }
