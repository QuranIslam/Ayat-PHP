قُرْآنًا عَرَبِيًّا غَيْرَ ذِي عِوَجٍ لَّعَلَّهُمْ يَتَّقُونَ|||

{ قُرآنًا عَرَبِيًّا غَيْرَ ذِي عِوَجٍ } أي: جعلناه قرآنا عربيا، واضح الألفاظ، سهل المعاني، خصوصا على العرب. { غَيْرَ ذِي عِوَجٍ } أي: ليس فيه خلل ولا نقص بوجه من الوجوه، لا في ألفاظه ولا في معانيه، وهذا يستلزم كمال اعتداله واستقامته كما قال تعالى: { الْحَمْدُ لِلَّهِ الَّذِي أَنْزَلَ عَلَى عَبْدِهِ الْكِتَابَ وَلَمْ يَجْعَلْ لَهُ عِوَجَا قَيِّمًا }

{ لَعَلَّهُمْ يَتَّقُونَ } الله تعالى، حيث سهلنا عليهم طرق التقوى العلمية والعملية، بهذا القرآن العربي المستقيم، الذي ضرب اللّه فيه من كل مثل.
