وَيَا قَوْمِ لَا أَسْأَلُكُمْ عَلَيْهِ مَالًا ۖ إِنْ أَجْرِيَ إِلَّا عَلَى اللَّهِ ۚ وَمَا أَنَا بِطَارِدِ الَّذِينَ آمَنُوا ۚ إِنَّهُم مُّلَاقُو رَبِّهِمْ وَلَٰكِنِّي أَرَاكُمْ قَوْمًا تَجْهَلُونَ|||

{ وَيَا قَوْمِ لَا أَسْأَلُكُمْ عَلَيْهِ } أي: على دعوتي إياكم { مَا لَا } فستستثقلون المغرم.

{ إِنْ أَجْرِيَ إِلَّا عَلَى اللَّهِ } وكأنهم طلبوا منه طرد المؤمنين الضعفاء، فقال لهم: { وَمَا أَنَا بِطَارِدِ الَّذِينَ آمَنُوا } أي: ما ينبغي لي، ولا يليق بي ذلك، بل أتلقاهم بالرحب والإكرام، والإعزاز والإعظام { إِنَّهُمْ مُلَاقُو رَبِّهِمْ } فمثيبهم على إيمانهم وتقواهم بجنات النعيم.

{ وَلَكِنِّي أَرَاكُمْ قَوْمًا تَجْهَلُونَ } حيث تأمرونني، بطرد أولياء الله, وإبعادهم عني. وحيث رددتم الحق، لأنهم أتباعه، وحيث استدللتم على بطلان الحق بقولكم إني بشر مثلكم وإنه ليس لنا عليكم من فضل.
