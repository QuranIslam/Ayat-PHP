فِي بِضْعِ سِنِينَ ۗ لِلَّهِ الْأَمْرُ مِن قَبْلُ وَمِن بَعْدُ ۚ وَيَوْمَئِذٍ يَفْرَحُ الْمُؤْمِنُونَ|||

{ فِي بِضْعِ سِنِينَ } تسع أو ثمان ونحو ذلك مما لا يزيد على العشر، ولا ينقص عن الثلاث، وأن غلبة الفرس للروم ثم غلبة الروم للفرس كل ذلك بمشيئته وقدره ولهذا قال: { لِلَّهِ الْأَمْرُ مِنْ قَبْلُ وَمِنْ بَعْدُ } فليس الغلبة والنصر لمجرد وجود الأسباب، وإنما هي لا بد أن يقترن بها القضاء والقدر.

{ وَيَوْمَئِذٍ } أي: يوم يغلب الروم الفرس ويقهرونهم { يَفْرَحُ الْمُؤْمِنُونَ{
