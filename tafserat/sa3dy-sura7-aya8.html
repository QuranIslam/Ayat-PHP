وَالْوَزْنُ يَوْمَئِذٍ الْحَقُّ ۚ فَمَن ثَقُلَتْ مَوَازِينُهُ فَأُولَٰئِكَ هُمُ الْمُفْلِحُونَ|||

ثم ذكر الجزاء على الأعمال، فقال: { وَالْوَزْنُ يَوْمَئِذٍ الْحَقُّ فَمَنْ ثَقُلَتْ مَوَازِينُهُ فَأُولَئِكَ هُمُ الْمُفْلِحُونَ وَمَنْ خَفَّتْ مَوَازِينُهُ فَأُولَئِكَ الَّذِينَ خَسِرُوا أَنْفُسَهُمْ بِمَا كَانُوا بِآياتِنَا يَظْلِمُونَ } أي: والوزن يوم القيامة يكون بالعدل والقسط، الذي لا جور فيه ولا ظلم بوجه. { فَمَنْ ثَقُلَتْ مَوَازِينُهُ } بأن رجحت كفة حسناته على سيئاته { فَأُولَئِكَ هُمُ الْمُفْلِحُونَ } أي: الناجون من المكروه، المدركون للمحبوب، الذين حصل لهم الربح العظيم، والسعادة الدائمة.
