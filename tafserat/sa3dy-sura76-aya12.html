وَجَزَاهُم بِمَا صَبَرُوا جَنَّةً وَحَرِيرًا|||

{ وَجَزَاهُمْ بِمَا صَبَرُوا } على طاعة الله، فعملوا ما أمكنهم منها، وعن معاصي الله، فتركوها، وعلى أقدار الله المؤلمة، فلم يتسخطوها، { جَنَّةً } جامعة لكل نعيم، سالمة من كل مكدر ومنغص، { وَحَرِيرًا } كما قال [تعالى:] { وَلِبَاسُهُمْ فِيهَا حَرِيرٌ } ولعل الله إنما خص الحرير، لأنه لباسهم الظاهر، الدال على حال صاحبه.
