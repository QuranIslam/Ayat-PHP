إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ لَهُمْ جَنَّاتٌ تَجْرِي مِن تَحْتِهَا الْأَنْهَارُ ۚ ذَٰلِكَ الْفَوْزُ الْكَبِيرُ|||

ولما ذكر عقوبة الظالمين، ذكر ثواب المؤمنين، فقال: { إِنَّ الَّذِينَ آمَنُوا } بقلوبهم { وَعَمِلُوا الصَّالِحَاتِ } بجوارحهم { لَهُمْ جَنَّاتٌ تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ ذَلِكَ الْفَوْزُ الْكَبِيرُ } الذي حصل به الفوز برضا الله ودار كرامته.
