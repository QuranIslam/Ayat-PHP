وَكَذَٰلِكَ نَجْزِي مَنْ أَسْرَفَ وَلَمْ يُؤْمِن بِآيَاتِ رَبِّهِ ۚ وَلَعَذَابُ الْآخِرَةِ أَشَدُّ وَأَبْقَىٰ|||

{ وَكَذَلِكَ } أي: هذا الجزاء { نَجْزِي } ه { مَنْ أَسْرَفَ } بأن تعدى الحدود، وارتكب المحارم وجاوز ما أذن له { وَلَمْ يُؤْمِنْ بِآيَاتِ رَبِّهِ } الدالة على جميع مطالب الإيمان دلالة واضحة صريحة، فالله لم يظلمه ولم يضع العقوبة في غير محلها، وإنما السبب إسرافه وعدم إيمانه.

{ وَلَعَذَابُ الْآخِرَةِ أَشَدُّ } من عذاب الدنيا أضعافا مضاعفة { وَأَبْقَى } لكونه لا ينقطع، بخلاف عذاب الدنيا فإنه منقطع، فالواجب الخوف والحذر من عذاب الآخرة.
