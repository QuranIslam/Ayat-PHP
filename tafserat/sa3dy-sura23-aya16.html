ثُمَّ إِنَّكُمْ يَوْمَ الْقِيَامَةِ تُبْعَثُونَ|||

{ ثُمَّ إِنَّكُمْ يَوْمَ الْقِيَامَةِ تُبْعَثُونَ } فتجازون بأعمالكم، حسنها وسيئها. قال تعالى: { أَيَحْسَبُ الْإِنْسَانُ أَنْ يُتْرَكَ سُدًى* أَلَمْ يَكُ نُطْفَةً مِنْ مَنِيٍّ يُمْنَى* ثُمَّ كَانَ عَلَقَةً فَخَلَقَ فَسَوَّى* فَجَعَلَ مِنْهُ الزَّوْجَيْنِ الذَّكَرَ وَالْأُنْثَى* أَلَيْسَ ذَلِكَ بِقَادِرٍ عَلَى أَنْ يُحْيِيَ الْمَوْتَى }
