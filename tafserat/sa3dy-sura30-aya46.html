وَمِنْ آيَاتِهِ أَن يُرْسِلَ الرِّيَاحَ مُبَشِّرَاتٍ وَلِيُذِيقَكُم مِّن رَّحْمَتِهِ وَلِتَجْرِيَ الْفُلْكُ بِأَمْرِهِ وَلِتَبْتَغُوا مِن فَضْلِهِ وَلَعَلَّكُمْ تَشْكُرُونَ|||

أي: ومن الأدلة الدالة على رحمته وبعثه الموتى وأنه الإله المعبود والملك المحمود، { أَنْ يُرْسِلَ الرِّيَاحَ } أمام المطر { مُبَشِّرَاتٍ } بإثارتها للسحاب ثم جمعها فتبشر بذلك النفوس قبل نزوله.

{ وَلِيُذِيقَكُمْ مِنْ رَحْمَتِهِ } فينزل عليكم من رحمته مطرا تحيا به البلاد والعباد، وتذوقون من رحمته ما تعرفون أن رحمته هي المنقذة للعباد والجالبة لأرزاقهم، فتشتاقون إلى الإكثار من الأعمال الصالحة الفاتحة لخزائن الرحمة.

{ وَلِتَجْرِيَ الْفُلْكُ } في البحر { بِأَمْرِهِ } القدري { وَلِتَبْتَغُوا مِنْ فَضْلِهِ } بالتصرف في معايشكم ومصالحكم.

{ وَلَعَلَّكُمْ تَشْكُرُونَ } من سخر لكم الأسباب وسير لكم الأمور. فهذا المقصود من النعم أن تقابل بشكر اللّه تعالى ليزيدكم اللّه منها ويبقيها عليكم.

وأما مقابلة النعم بالكفر والمعاصي فهذه حال من بدَّل نعمة اللّه كفرا ونعمته محنة وهو معرض لها للزوال والانتقال منه إلى غيره.
