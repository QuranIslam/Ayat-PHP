قُلْ أَغَيْرَ اللَّهِ أَبْغِي رَبًّا وَهُوَ رَبُّ كُلِّ شَيْءٍ ۚ وَلَا تَكْسِبُ كُلُّ نَفْسٍ إِلَّا عَلَيْهَا ۚ وَلَا تَزِرُ وَازِرَةٌ وِزْرَ أُخْرَىٰ ۚ ثُمَّ إِلَىٰ رَبِّكُم مَّرْجِعُكُمْ فَيُنَبِّئُكُم بِمَا كُنتُمْ فِيهِ تَخْتَلِفُونَ|||

{ قُلْ أَغَيْرَ اللَّهِ } من المخلوقين { أَبْغِي رَبًّا } أي: يحسن ذلك ويليق بي، أن أتخذ غيره، مربيا ومدبرا والله رب كل شيء، فالخلق كلهم داخلون تحت ربوبيته، منقادون لأمره؟". فتعين علي وعلى غيري، أن يتخذ الله ربا، ويرضى به، وألا يتعلق بأحد من المربوبين الفقراء العاجزين. ثم رغب ورهب بذكر الجزاء فقال: { وَلَا تَكْسِبُ كُلُّ نَفْسٍ } من خير وشر { إِلَّا عَلَيْهَا } كما قال تعالى: { مَنْ عَمِلَ صَالِحًا فَلِنَفْسِهِ وَمَنْ أَسَاءَ فَعَلَيْهَا } { وَلَا تَزِرُ وَازِرَةٌ وِزْرَ أُخْرَى } بل كل عليه وزر نفسه، وإن كان أحد قد تسبب في ضلال غيره ووزره، فإن عليه وزر التسبب من غير أن ينقص من وزر المباشر شيء. { ثُمَّ إِلَى رَبِّكُمْ مَرْجِعُكُمْ } يوم القيامة { فَيُنَبِّئُكُمْ بِمَا كُنْتُمْ فِيهِ تَخْتَلِفُونَ } من خير وشر، ويجازيكم على ذلك، أوفى الجزاء.
