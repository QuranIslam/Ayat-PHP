وَمَا أَنتُم بِمُعْجِزِينَ فِي الْأَرْضِ ۖ وَمَا لَكُم مِّن دُونِ اللَّهِ مِن وَلِيٍّ وَلَا نَصِيرٍ|||

{ وَمَا أَنْتُمْ بِمُعْجِزِينَ فِي الْأَرْضِ } أي: معجزين قدرة اللّه عليكم، بل أنتم عاجزون في الأرض، ليس عندكم امتناع عما ينفذه اللّه فيكم. { وَمَا لَكُمْ مِنْ دُونِ اللَّهِ مِنْ وَلِيٍّ } يتولاكم، فيحصل لكم المنافع { وَلَا نَصِيرٍ } يدفع عنكم المضار.
