أَلَمْ تَرَ كَيْفَ ضَرَبَ اللَّهُ مَثَلًا كَلِمَةً طَيِّبَةً كَشَجَرَةٍ طَيِّبَةٍ أَصْلُهَا ثَابِتٌ وَفَرْعُهَا فِي السَّمَاءِ|||

يقول تعالى: { أَلَمْ تَرَ كَيْفَ ضَرَبَ اللَّهُ مَثَلًا كَلِمَةً طَيِّبَة } " وهي شهادة أن لا إله إلا الله، وفروعها { كَشَجَرَةٍ طَيِّبَةٍ } وهي النخلة { أَصْلُهَا ثَابِتٌ } في الأرض { وَفَرْعُهَا } منتشر { فِي السَّمَاءِ }
