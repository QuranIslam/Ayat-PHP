قَالَ الَّذِينَ حَقَّ عَلَيْهِمُ الْقَوْلُ رَبَّنَا هَٰؤُلَاءِ الَّذِينَ أَغْوَيْنَا أَغْوَيْنَاهُمْ كَمَا غَوَيْنَا ۖ تَبَرَّأْنَا إِلَيْكَ ۖ مَا كَانُوا إِيَّانَا يَعْبُدُونَ|||

ولهذا { قَالَ الَّذِينَ حَقَّ عَلَيْهِمُ الْقَوْلُ } الرؤساء والقادة، في الكفر والشر، مقرين بغوايتهم وإغوائهم: { رَبَّنَا هَؤُلَاءِ } التابعون { الَّذِينَ أَغْوَيْنَا أَغْوَيْنَاهُمْ كَمَا غَوَيْنَا } أي: كلنا قد اشترك في الغواية، وحق عليه كلمة العذاب.

{ تَبَرَّأْنَا إِلَيْكَ } من عبادتهم، أي: نحن برآء منهم ومن عملهم. { مَا كَانُوا إِيَّانَا يَعْبُدُونَ } وإنما كانوا يعبدون الشياطين.
