وَالَّذِينَ إِذَا أَنفَقُوا لَمْ يُسْرِفُوا وَلَمْ يَقْتُرُوا وَكَانَ بَيْنَ ذَٰلِكَ قَوَامًا|||

{ وَالَّذِينَ إِذَا أَنْفَقُوا } النفقات الواجبة والمستحبة { لَمْ يُسْرِفُوا } بأن يزيدوا على الحد فيدخلوا في قسم التبذير وإهمال الحقوق الواجبة، { وَلَمْ يَقْتُرُوا } فيدخلوا في باب البخل والشح { وَكَانَ } إنفاقهم { بَيْنَ ذَلِكَ } بين الإسراف والتقتير { قَوَامًا } يبذلون في الواجبات من الزكوات والكفارات والنفقات الواجبة، وفيما ينبغي على الوجه الذي ينبغي من غير ضرر ولا ضرار وهذا من عدلهم واقتصادهم.
