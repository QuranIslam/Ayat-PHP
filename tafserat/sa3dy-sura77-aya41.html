إِنَّ الْمُتَّقِينَ فِي ظِلَالٍ وَعُيُونٍ|||

لما ذكر عقوبة المكذبين، ذكر ثواب المحسنين، فقال: { إِنَّ الْمُتَّقِينَ } [أي:] للتكذيب، المتصفين بالتصديق في أقوالهم وأفعالهم وأعمالهم، ولا يكونون كذلك إلا بأدائهم الواجبات، وتركهم المحرمات.

{ فِي ظِلَالٍ } من كثرة الأشجار المتنوعة، الزاهية البهية. { وَعُيُونٍ } جارية من السلسبيل، والرحيق وغيرهما،
