قَالُوا يَا قَوْمَنَا إِنَّا سَمِعْنَا كِتَابًا أُنزِلَ مِن بَعْدِ مُوسَىٰ مُصَدِّقًا لِّمَا بَيْنَ يَدَيْهِ يَهْدِي إِلَى الْحَقِّ وَإِلَىٰ طَرِيقٍ مُّسْتَقِيمٍ|||

{ قَالُوا يَا قَوْمَنَا إِنَّا سَمِعْنَا كِتَابًا أُنْزِلَ مِنْ بَعْدِ مُوسَى } لأن كتاب موسى أصل للإنجيل وعمدة لبني إسرائيل في أحكام الشرع، وإنما الإنجيل متمم ومكمل ومغير لبعض الأحكام.

{ مُصَدِّقًا لِمَا بَيْنَ يَدَيْهِ يَهْدِي } هذا الكتاب الذي سمعناه { إِلَى الْحَقِّ } وهو الصواب في كل مطلوب وخبر { وَإِلَى طَرِيقٍ مُسْتَقِيمٍ } موصل إلى الله وإلى جنته من العلم بالله وبأحكامه الدينية وأحكام الجزاء.
