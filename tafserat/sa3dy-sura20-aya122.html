ثُمَّ اجْتَبَاهُ رَبُّهُ فَتَابَ عَلَيْهِ وَهَدَىٰ|||

{ وَعَصَى آدَمُ رَبَّهُ فَغَوَى } فبادرا إلى التوبة والإنابة، وقالا: { رَبَّنَا ظَلَمْنَا أَنْفُسَنَا وَإِنْ لَمْ تَغْفِرْ لَنَا وَتَرْحَمْنَا لَنَكُونَنَّ مِنَ الْخَاسِرِينَ } فاجتباه ربه، واختاره، ويسر له التوبة { فَتَابَ عَلَيْهِ وَهَدَى } فكان بعد التوبة أحسن منه قبلها، ورجع كيد العدو عليه، وبطل مكره، فتمت النعمة عليه وعلى ذريته، ووجب عليهم القيام بها والاعتراف، وأن يكونوا على حذر من هذا العدو المرابط الملازم لهم، ليلا ونهارا { يَا بَنِي آدَمَ لَا يَفْتِنَنَّكُمُ الشَّيْطَانُ كَمَا أَخْرَجَ أَبَوَيْكُمْ مِنَ الْجَنَّةِ يَنْزِعُ عَنْهُمَا لِبَاسَهُمَا لِيُرِيَهُمَا سَوْآتِهِمَا إِنَّهُ يَرَاكُمْ هُوَ وَقَبِيلُهُ مِنْ حَيْثُ لَا تَرَوْنَهُمْ إِنَّا جَعَلْنَا الشَّيَاطِينَ أَوْلِيَاءَ لِلَّذِينَ لَا يُؤْمِنُونَ }
