وَيَقُولُونَ أَئِنَّا لَتَارِكُو آلِهَتِنَا لِشَاعِرٍ مَّجْنُونٍ|||

{ وَيَقُولُونَ } معارضة لها { أَئِنَّا لَتَارِكُوا آلِهَتِنَا } التي لم نزل نعبدها نحن وآباؤنا { لـ } قول { شَاعِرٍ مَجْنُونٍ } يعنون محمدا صلى اللّه عليه وسلم. فلم يكفهم - قبحهم اللّه - الإعراض عنه، ولا مجرد تكذيبه، حتى حكموا عليه بأظلم الأحكام، وجعلوه شاعرا مجنونا، وهم يعلمون أنه لا يعرف الشعر والشعراء، ولا وصفه وصفهم، وأنه أعقل خلق اللّه، وأعظمهم رأيا.
