سُبْحَانَ الَّذِي خَلَقَ الْأَزْوَاجَ كُلَّهَا مِمَّا تُنبِتُ الْأَرْضُ وَمِنْ أَنفُسِهِمْ وَمِمَّا لَا يَعْلَمُونَ|||

{ سُبْحَانَ الَّذِي خَلَقَ الْأَزْوَاجَ كُلَّهَا } أي: الأصناف كلها، { مِمَّا تُنْبِتُ الْأَرْضُ } فنوع فيها من الأصناف ما يعسر تعداده. { وَمِنْ أَنْفُسِهِمْ } فنوعهم إلى ذكر وأنثى، وفاوت بين خلقهم وخُلُقِهمْ، وأوصافهم الظاهرة والباطنة. { وَمِمَّا لَا يَعْلَمُونَ } من المخلوقات التي قد خلقت وغابت عن علمنا، والتي لم تخلق بعد، فسبحانه وتعالى أن يكون له شريك، أو ظهير، أو عوين، أو وزير، أو صاحبة، أو ولد، أو سَمِيٌّ، أو شبيه، أو مثيل في صفات كماله ونعوت جلاله، أو يعجزه شيء يريده.
