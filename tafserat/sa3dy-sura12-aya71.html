قَالُوا وَأَقْبَلُوا عَلَيْهِم مَّاذَا تَفْقِدُونَ|||

{ قَالُوا } أي: إخوة يوسف { وَأَقْبَلُوا عَلَيْهِمْ } لإبعاد التهمة، فإن السارق ليس له همٌّ إلا البعد والانطلاق عمن سرق منه، لتسلم له سرقته، وهؤلاء جاءوا مقبلين إليهم، ليس لهم همٌّ إلا إزالة التهمة التي رموا بها عنهم، فقالوا في هذه الحال: { مَاذَا تَفْقِدُونَ } ولم يقولوا: "ما الذي سرقنا" لجزمهم بأنهم براء من السرقة.
