وَهَٰذَا كِتَابٌ أَنزَلْنَاهُ مُبَارَكٌ مُّصَدِّقُ الَّذِي بَيْنَ يَدَيْهِ وَلِتُنذِرَ أُمَّ الْقُرَىٰ وَمَنْ حَوْلَهَا ۚ وَالَّذِينَ يُؤْمِنُونَ بِالْآخِرَةِ يُؤْمِنُونَ بِهِ ۖ وَهُمْ عَلَىٰ صَلَاتِهِمْ يُحَافِظُونَ|||

أي: { وَهَذَا ْ} القرآن الذي { أَنْزَلْنَاهُ ْ} إليك { مُبَارَكٌ ْ} أي: وَصْفُه البركة، وذلك لكثرة خيراته، وسعة مبراته. { مُصَدِّقُ الَّذِي بَيْنَ يَدَيْهِ ْ} أي: موافق للكتب السابقة، وشاهد لها بالصدق. { وَلِتُنْذِرَ أُمَّ الْقُرَى وَمَنْ حَوْلَهَا ْ} أي: وأنزلناه أيضا لتنذر أم القرى، وهي: مكة المكرمة، ومن حولها، من ديار العرب، بل، ومن سائر البلدان. فتحذر الناس عقوبة الله، وأخذه الأمم، وتحذرهم مما يوجب ذلك. { وَالَّذِينَ يُؤْمِنُونَ بِالْآخِرَةِ يُؤْمِنُونَ بِهِ ْ} لأن الخوف إذا كان في القلب عمرت أركانه، وانقاد لمراضي الله. { وَهُمْ عَلَى صَلَاتِهِمْ يُحَافِظُونَ ْ} أي: يداومون عليها، ويحفظون أركانها وحدودها وشروطها وآدابها، ومكملاتها. جعلنا الله منهم.
