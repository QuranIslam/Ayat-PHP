كُلُوا مِن طَيِّبَاتِ مَا رَزَقْنَاكُمْ وَلَا تَطْغَوْا فِيهِ فَيَحِلَّ عَلَيْكُمْ غَضَبِي ۖ وَمَن يَحْلِلْ عَلَيْهِ غَضَبِي فَقَدْ هَوَىٰ|||

{ كُلُوا مِنْ طَيِّبَاتِ مَا رَزَقْنَاكُمْ } أي: واشكروه على ما أسدى إليكم من النعم { وَلَا تَطْغَوْا فِيهِ } أي: في رزقه، فتستعملونه في معاصيه، وتبطرون النعمة، فإنكم إن فعلتم ذلك، حل عليكم غضبي أي: غضبت عليكم، ثم عذبتكم، { وَمَنْ يَحْلِلْ عَلَيْهِ غَضَبِي فَقَدْ هَوَى } أي: ردى وهلك، وخاب وخسر، لأنه عدم الرضا والإحسان، وحل عليه الغضب والخسران.
