فَكَيْفَ إِذَا أَصَابَتْهُم مُّصِيبَةٌ بِمَا قَدَّمَتْ أَيْدِيهِمْ ثُمَّ جَاءُوكَ يَحْلِفُونَ بِاللَّهِ إِنْ أَرَدْنَا إِلَّا إِحْسَانًا وَتَوْفِيقًا|||

{ فَكَيْفَ } يكون حال هؤلاء الضالين { إِذَا أَصَابَتْهُمْ مُصِيبَةٌ بِمَا قَدَّمَتْ أَيْدِيهِمْ } من المعاصي ومنها تحكيم الطاغوت؟! { ثُمَّ جَاءُوكَ } معتذرين لما صدر منهم، ويقولون: { إِنْ أَرَدْنَا إِلَّا إِحْسَانًا وَتَوْفِيقًا } أي: ما قصدنا في ذلك إلا الإحسان إلى المتخاصمين والتوفيق بينهم، وهم كَذَبة في ذلك. فإن الإحسان كل الإحسان تحكيم الله ورسوله { ومَنْ أحْسَن من الله حكمًا لقوْمٍ يوقنون }
