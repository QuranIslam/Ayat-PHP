إِذْ قَالُوا لَيُوسُفُ وَأَخُوهُ أَحَبُّ إِلَىٰ أَبِينَا مِنَّا وَنَحْنُ عُصْبَةٌ إِنَّ أَبَانَا لَفِي ضَلَالٍ مُّبِينٍ|||

{ إِذْ قَالُوا ْ} فيما بينهم: { لَيُوسُفُ وَأَخُوهُ ْ} بنيامين، أي: شقيقه، وإلا فكلهم إخوة. { أَحَبُّ إِلَى أَبِينَا مِنَّا وَنَحْنُ عُصْبَةٌْ} أي: جماعة، فكيف يفضلهما علينا بالمحبة والشفقة، { إِنَّ أَبَانَا لَفِي ضَلَالٍ مُبِينٍ ْ} أي: لفي خطأ بيِّن، حيث فضلهما علينا من غير موجب نراه، ولا أمر نشاهده.
