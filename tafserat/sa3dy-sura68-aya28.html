قَالَ أَوْسَطُهُمْ أَلَمْ أَقُل لَّكُمْ لَوْلَا تُسَبِّحُونَ|||

فـ { قَالَ أَوْسَطُهُمْ } أي: أعدلهم، وأحسنهم طريقة { أَلَمْ أَقُلْ لَكُمْ لَوْلَا تُسَبِّحُونَ } أي: تنزهون الله عما لا يليق به، ومن ذلك، ظنكم أن قدرتكم مستقلة، فلولا استثنيتم، فقلتم: { إِنْ شَاءَ اللَّهُ } وجعلتم مشيئتكم تابعة لمشيئتة الله، لما جرى عليكم ما جرى.
