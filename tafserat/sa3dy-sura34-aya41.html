قَالُوا سُبْحَانَكَ أَنتَ وَلِيُّنَا مِن دُونِهِم ۖ بَلْ كَانُوا يَعْبُدُونَ الْجِنَّ ۖ أَكْثَرُهُم بِهِم مُّؤْمِنُونَ|||

{ قَالُوا سُبْحَانَكَ } أي: تنزيها لك وتقديسا, أن يكون لك شريك, أو ند { أَنْتَ وَلِيُّنَا مِنْ دُونِهِمْ } فنحن مفتقرون إلى ولايتك, مضطرون إليها, فكيف ندعو غيرنا إلى عبادتنا؟ أم كيف نصلح لأن نتخذ من دونك أولياء وشركاء؟"

ولكن هؤلاء المشركون { كَانُوا يَعْبُدُونَ الْجِنَّ } أي: الشياطين, يأمرون بعبادتنا أو عبادة غيرنا, فيطيعونهم بذلك. وطاعتهم هي عبادتهم, لأن العبادة الطاعة, كما قال تعالى مخاطبا لكل من اتخذ معه آلهة { أَلَمْ أَعْهَدْ إِلَيْكُمْ يَا بَنِي آدَمَ أَنْ لَا تَعْبُدُوا الشَّيْطَانَ إِنَّهُ لَكُمْ عَدُوٌّ مُبِينٌ * وَأَنِ اعْبُدُونِي هَذَا صِرَاطٌ مُسْتَقِيمٌ }

{ أَكْثَرُهُمْ بِهِمْ مُؤْمِنُونَ } أي: مصدقون للجن, منقادون لهم, لأن الإيمان هو: التصديق الموجب للانقياد.
