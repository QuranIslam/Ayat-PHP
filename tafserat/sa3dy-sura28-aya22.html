وَلَمَّا تَوَجَّهَ تِلْقَاءَ مَدْيَنَ قَالَ عَسَىٰ رَبِّي أَن يَهْدِيَنِي سَوَاءَ السَّبِيلِ|||

{ وَلَمَّا تَوَجَّهَ تِلْقَاءَ مَدْيَنَ } أي: قاصدا بوجهه مدين، وهو جنوبي فلسطين، حيث لا ملك لفرعون، { قَالَ عَسَى رَبِّي أَنْ يَهْدِيَنِي سَوَاءَ السَّبِيلِ } أي: وسط الطريق المختصر، الموصل إليها بسهولة ورفق، فهداه اللّه سواء السبيل، فوصل إلى مدين.
