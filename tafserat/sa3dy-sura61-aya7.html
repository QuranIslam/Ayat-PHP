وَمَنْ أَظْلَمُ مِمَّنِ افْتَرَىٰ عَلَى اللَّهِ الْكَذِبَ وَهُوَ يُدْعَىٰ إِلَى الْإِسْلَامِ ۚ وَاللَّهُ لَا يَهْدِي الْقَوْمَ الظَّالِمِينَ|||

{ وَمَنْ أَظْلَمُ مِمَّنِ افْتَرَى عَلَى اللَّهِ الْكَذِبَ } بهذا وغيره، والحال أنه لا عذر له، وقد انقطعت حجته، لأنه { يُدْعَى إِلَى الْإِسْلَامِ } ويبين له ببراهينه وبيناته، { وَاللَّهُ لَا يَهْدِي الْقَوْمَ الظَّالِمِينَ } الذين لا يزالون على ظلمهم مستقيمين، لا تردهم عنه موعظة، ولا يزجرهم بيان ولا برهان، خصوصا هؤلاء الظلمة القائمين بمقابلة الحق ليردوه، ولينصروا الباطل.
