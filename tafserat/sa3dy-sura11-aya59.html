وَتِلْكَ عَادٌ ۖ جَحَدُوا بِآيَاتِ رَبِّهِمْ وَعَصَوْا رُسُلَهُ وَاتَّبَعُوا أَمْرَ كُلِّ جَبَّارٍ عَنِيدٍ|||

{ وَتِلْكَ عَادٌ } الذين أوقع الله بهم ما أوقع، بظلم منهم لأنهم { جَحَدُوا بِآيَاتِ رَبِّهِمْ } ولهذا قالوا لهود: { ما جئتنا ببينة } فتبين بهذا أنهم متيقنون لدعوته، وإنما عاندوا وجحدوا { وَعَصَوْا رُسُلَهُ } لأن من عصى رسولا، فقد عصى جميع المرسلين، لأن دعوتهم واحدة.

{ وَاتَّبَعُوا أَمْرَ كُلِّ جَبَّارٍ } أي: متسلط على عباد الله بالجبروت، { عنيد } أي: معاند لآيات الله، فعصوا كل ناصح ومشفق عليهم، واتبعوا كل غاش لهم، يريد إهلاكهم لا جرم أهلكهم الله.
