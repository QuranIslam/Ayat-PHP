سَخَّرَهَا عَلَيْهِمْ سَبْعَ لَيَالٍ وَثَمَانِيَةَ أَيَّامٍ حُسُومًا فَتَرَى الْقَوْمَ فِيهَا صَرْعَىٰ كَأَنَّهُمْ أَعْجَازُ نَخْلٍ خَاوِيَةٍ|||

{ سَخَّرَهَا عَلَيْهِمْ سَبْعَ لَيَالٍ وَثَمَانِيَةَ أَيَّامٍ حُسُومًا } أي: نحسا وشرا فظيعا عليهم فدمرتهم وأهلكتهم، { فَتَرَى الْقَوْمَ فِيهَا صَرْعَى } أي: هلكى موتى { كَأَنَّهُمْ أَعْجَازُ نَخْلٍ خَاوِيَةٍ } أي: كأنهم جذوع النخل التي قد قطعت رءوسها الخاوية الساقط بعضها على بعض.
