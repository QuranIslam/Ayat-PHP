وَأَنِيبُوا إِلَىٰ رَبِّكُمْ وَأَسْلِمُوا لَهُ مِن قَبْلِ أَن يَأْتِيَكُمُ الْعَذَابُ ثُمَّ لَا تُنصَرُونَ|||

ولهذا أمر تعالى بالإنابة إليه، والمبادرة إليها فقال: { وَأَنِيبُوا إِلَى رَبِّكُمْ } بقلوبكم { وَأَسْلِمُوا لَهُ } بجوارحكم، إذا أفردت الإنابة، دخلت فيها أعمال الجوارح، وإذا جمع بينهما، كما في هذا الموضع، كان المعنى ما ذكرنا.

وفي قوله { إِلَى رَبِّكُمْ وَأَسْلِمُوا لَهُ } دليل على الإخلاص، وأنه من دون إخلاص، لا تفيد الأعمال الظاهرة والباطنة شيئا. { مِنْ قَبْلِ أَنْ يَأْتِيَكُمُ الْعَذَابُ } مجيئا لا يدفع { ثُمَّ لَا تُنْصَرُونَ } فكأنه قيل: ما هي الإنابة والإسلام؟ وما جزئياتها وأعمالها؟
