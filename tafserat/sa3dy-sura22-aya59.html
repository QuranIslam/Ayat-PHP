لَيُدْخِلَنَّهُم مُّدْخَلًا يَرْضَوْنَهُ ۗ وَإِنَّ اللَّهَ لَعَلِيمٌ حَلِيمٌ|||

{ لَيُدْخِلَنَّهُمْ مُدْخَلًا يَرْضَوْنَهُ } إما ما يفتحه الله عليهم من البلدان، خصوصا فتح مكة المشرفة، فإنهم دخلوها في حالة الرضا والسرور، وإما المراد به رزق الآخرة، وأن ذلك دخول الجنة، فتكون الآية جمعت بين الرزقين، رزق الدنيا، ورزق الآخرة، واللفظ صالح لذلك كله، والمعنى صحيح، فلا مانع من إرادة الجميع { وَإِنَّ اللَّهَ لَعَلِيمٌ } بالأمور، ظاهرها، وباطنها، متقدمها، ومتأخرها، { حَلِيمٌ } يعصيه الخلائق، ويبارزونه بالعظائم، وهو لا يعاجلهم بالعقوبة مع كمال اقتداره، بل يواصل لهم رزقه، ويسدي إليهم فضله.
