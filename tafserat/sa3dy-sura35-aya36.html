وَالَّذِينَ كَفَرُوا لَهُمْ نَارُ جَهَنَّمَ لَا يُقْضَىٰ عَلَيْهِمْ فَيَمُوتُوا وَلَا يُخَفَّفُ عَنْهُم مِّنْ عَذَابِهَا ۚ كَذَٰلِكَ نَجْزِي كُلَّ كَفُورٍ|||

لما ذكر تعالى حال أهل الجنة ونعيمهم، ذكر حال أهل النار وعذابهم فقال: { وَالَّذِينَ كَفَرُوا } أي: جحدوا ما جاءتهم به رسلهم من الآيات، وأنكروا لقاء ربهم.

{ لَهُمْ نَارُ جَهَنَّمَ } يعذبون فيها أشد العذاب، وأبلغ العقاب. { لَا يُقْضَى عَلَيْهِمْ } بالموت { فَيَمُوتُوا } فيستريحوا، { وَلَا يُخَفَّفُ عَنْهُمْ مِنْ عَذَابِهَا } فشدة العذاب وعظمه، مستمر عليهم في جميع الآنات واللحظات.{ كَذَلِكَ نَجْزِي كُلَّ كَفُورٍ{
