الَّذِينَ اتَّخَذُوا دِينَهُمْ لَهْوًا وَلَعِبًا وَغَرَّتْهُمُ الْحَيَاةُ الدُّنْيَا ۚ فَالْيَوْمَ نَنسَاهُمْ كَمَا نَسُوا لِقَاءَ يَوْمِهِمْ هَٰذَا وَمَا كَانُوا بِآيَاتِنَا يَجْحَدُونَ|||

{ لَهْوًا وَلَعِبًا } أي: لهت قلوبهم وأعرضت عنه، ولعبوا واتخذوه سخريا، أو أنهم جعلوا بدل دينهم اللهو واللعب، واستعاضوا بذلك عن الدين القيم. { وَغَرَّتْهُمُ الْحَيَاةُ الدُّنْيَا } بزينتها وزخرفها وكثرة دعاتها، فاطمأنوا إليها ورضوا بها وفرحوا، وأعرضوا عن الآخرة ونسوها. { فَالْيَوْمَ نَنْسَاهُمْ } أي: نتركهم في العذاب { كَمَا نَسُوا لِقَاءَ يَوْمِهِمْ هَذَا } فكأنهم لم يخلقوا إلا للدنيا، وليس أمامهم عرض ولا جزاء. { وَمَا كَانُوا بِآيَاتِنَا يَجْحَدُونَ } والحال أن جحودهم هذا، لا عن قصور في آيات اللّه وبيناته.
