إِذْ رَأَىٰ نَارًا فَقَالَ لِأَهْلِهِ امْكُثُوا إِنِّي آنَسْتُ نَارًا لَّعَلِّي آتِيكُم مِّنْهَا بِقَبَسٍ أَوْ أَجِدُ عَلَى النَّارِ هُدًى|||

{ فَقَالَ لِأَهْلِهِ امْكُثُوا إِنِّي آنَسْتُ } أي: أبصرت { نَارًا } وكان ذلك في جانب الطور الأيمن، { لَعَلِّي آتِيكُمْ مِنْهَا بِقَبَسٍ } تصطلون به { أَوْ أَجِدُ عَلَى النَّارِ هُدًى } أي: من يهديني الطريق. وكان مطلبه، النور الحسي والهداية الحسية، فوجد ثم النور المعنوي، نور الوحي، الذي تستنير به الأرواح والقلوب، والهداية الحقيقية، هداية الصراط المستقيم، الموصلة إلى جنات النعيم، فحصل له أمر لم يكن في حسابه، ولا خطر بباله.
