هَٰذَا نُزُلُهُمْ يَوْمَ الدِّينِ|||

{ هَذَا } الطعام والشراب { نُزُلُهُمْ } أي: ضيافتهم { يَوْمَ الدِّينِ } وهي الضيافة التي قدموها لأنفسهم، وآثروها على ضيافة الله لأوليائه.

قال تعالى: { إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ كَانَتْ لَهُمْ جَنَّاتُ الْفِرْدَوْسِ نُزُلًا خَالِدِينَ فِيهَا لَا يَبْغُونَ عَنْهَا حِوَلًا }
