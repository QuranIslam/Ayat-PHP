مَّا عَلَى الرَّسُولِ إِلَّا الْبَلَاغُ ۗ وَاللَّهُ يَعْلَمُ مَا تُبْدُونَ وَمَا تَكْتُمُونَ|||

{ مَا عَلَى الرَّسُولِ إِلَّا الْبَلَاغُ } وقد بلَّغ كما أُمِر، وقام بوظيفته، وما سـوى ذلك فليـس لـه مـن الأمـر شـيء. { وَاللَّهُ يَعْلَمُ مَا تُبْدُونَ وَمَا تَكْتُمُونَ } فيجازيكم بما يعلمه تعالى منكم.
