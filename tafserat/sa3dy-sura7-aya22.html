فَدَلَّاهُمَا بِغُرُورٍ ۚ فَلَمَّا ذَاقَا الشَّجَرَةَ بَدَتْ لَهُمَا سَوْآتُهُمَا وَطَفِقَا يَخْصِفَانِ عَلَيْهِمَا مِن وَرَقِ الْجَنَّةِ ۖ وَنَادَاهُمَا رَبُّهُمَا أَلَمْ أَنْهَكُمَا عَن تِلْكُمَا الشَّجَرَةِ وَأَقُل لَّكُمَا إِنَّ الشَّيْطَانَ لَكُمَا عَدُوٌّ مُّبِينٌ|||

{ فَدَلَّاهُمَا } أي: نزَّلهما عن رتبتهما العالية، التي هي البعد عن الذنوب والمعاصي إلى التلوث بأوضارها، فأقدما على أكلها. { فَلَمَّا ذَاقَا الشَّجَرَةَ بَدَتْ لَهُمَا سَوْآتُهُمَا } أي: ظهرت عورة كل منهما بعد ما كانت مستورة، فصار للعري الباطن من التقوى في هذه الحال أثر في اللباس الظاهر، حتى انخلع فظهرت عوراتهما، ولما ظهرت عوراتهما خَجِلا وجَعَلا يخصفان على عوراتهما من أوراق شجر الجنة، ليستترا بذلك. { وَنَادَاهُمَا رَبُّهُمَا } وهما بتلك الحال موبخا ومعاتبا: { أَلَمْ أَنْهَكُمَا عَنْ تِلْكُمَا الشَّجَرَةِ وَأَقُلْ لَكُمَا إِنَّ الشَّيْطَانَ لَكُمَا عَدُوٌّ مُبِينٌ } فلم اقترفتما المنهي، وأطعتما عدوَّكُما؟
