يَا أَيُّهَا النَّاسُ إِنَّ وَعْدَ اللَّهِ حَقٌّ ۖ فَلَا تَغُرَّنَّكُمُ الْحَيَاةُ الدُّنْيَا ۖ وَلَا يَغُرَّنَّكُم بِاللَّهِ الْغَرُورُ|||

يقول تعالى: { يَا أَيُّهَا النَّاسُ إِنَّ وَعْدَ اللَّهِ } بالبعث والجزاء على الأعمال، { حَقٌّ } أي: لا شك فيه، ولا مرية، ولا تردد، قد دلت على ذلك الأدلة السمعية والبراهين العقلية، فإذا كان وعده حقا، فتهيئوا له، وبادروا أوقاتكم الشريفة بالأعمال الصالحة، ولا يقطعكم عن ذلك قاطع، { فَلَا تَغُرَّنَّكُمُ الْحَيَاةُ الدُّنْيَا } بلذاتها وشهواتها ومطالبها النفسية، فتلهيكم عما خلقتم له، { وَلَا يَغُرَّنَّكُمْ بِاللَّهِ الْغَرُورُ }
