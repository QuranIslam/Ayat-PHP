وَدَخَلَ مَعَهُ السِّجْنَ فَتَيَانِ ۖ قَالَ أَحَدُهُمَا إِنِّي أَرَانِي أَعْصِرُ خَمْرًا ۖ وَقَالَ الْآخَرُ إِنِّي أَرَانِي أَحْمِلُ فَوْقَ رَأْسِي خُبْزًا تَأْكُلُ الطَّيْرُ مِنْهُ ۖ نَبِّئْنَا بِتَأْوِيلِهِ ۖ إِنَّا نَرَاكَ مِنَ الْمُحْسِنِينَ|||

أي: { و } لما دخل يوسف السجن، كان في جملة من { دَخَلَ مَعَهُ السِّجْنَ فَتَيَانِ } أي: شابان، فرأى كل واحد منهما رؤيا، فقصها على يوسف ليعبرها، .فـ { قَالَ أَحَدُهُمَا إِنِّي أَرَانِي أَعْصِرُ خَمْرًا وَقَالَ الْآخَرُ إِنِّي أَرَانِي أَحْمِلُ فَوْقَ رَأْسِي خُبْزًا } وذلك الخبز { تَأْكُلُ الطَّيْرُ مِنْهُ نَبِّئْنَا بِتَأْوِيلِهِ } أي: بتفسيره، وما يؤول إليه أمرهما، وقولهما: { إِنَّا نَرَاكَ مِنَ الْمُحْسِنِينَ } أي: من أهل الإحسان إلى الخلق، فأحسن إلينا في تعبيرك لرؤيانا، كما أحسنت إلى غيرنا، فتوسلا ليوسف بإحسانه.
