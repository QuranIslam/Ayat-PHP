إِنَّ هَٰؤُلَاءِ يُحِبُّونَ الْعَاجِلَةَ وَيَذَرُونَ وَرَاءَهُمْ يَوْمًا ثَقِيلًا|||

{ إِنَّ هَؤُلَاءِ } أي: المكذبين لك أيها الرسول بعد ما بينت لهم الآيات، ورغبوا ورهبوا، ومع ذلك، لم يفد فيهم ذلك شيئا، بل لا يزالون يؤثرون، { الْعَاجِلَةَ } ويطمئنون إليها، { وَيَذَرُونَ } أي: يتركون العمل ويهملون { وَرَاءَهُمْ } أي: أمامهم { يَوْمًا ثَقِيلًا } وهو يوم القيامة، الذي مقداره خمسون ألف سنة مما تعدون، وقال تعالى: { يَقُولُ الْكَافِرُونَ هَذَا يَوْمٌ عَسِرٌ } فكأنهم ما خلقوا إلا للدنيا والإقامة فيها.
