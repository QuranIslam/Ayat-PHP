فَدَعَا رَبَّهُ أَنِّي مَغْلُوبٌ فَانتَصِرْ|||

فعند ذلك دعا نوح ربه [فقال:] { أَنِّي مَغْلُوبٌ } لا قدرة لي على الانتصار منهم، لأنه لم يؤمن من قومه إلا القليل النادر، ولا قدرة لهم على مقاومة قومهم، { فَانْتَصِرْ } اللهم لي منهم، وقال في الآية الأخرى: { رَبِّ لَا تَذَرْ عَلَى الْأَرْضِ مِنَ الْكَافِرِينَ دَيَّارًا } الآيات
