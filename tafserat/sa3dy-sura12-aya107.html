أَفَأَمِنُوا أَن تَأْتِيَهُمْ غَاشِيَةٌ مِّنْ عَذَابِ اللَّهِ أَوْ تَأْتِيَهُمُ السَّاعَةُ بَغْتَةً وَهُمْ لَا يَشْعُرُونَ|||

{ أَفَأَمِنُوا } أي: الفاعلون لتلك الأفعال، المعرضون عن آيات الله { أَنْ تَأْتِيَهُمْ غَاشِيَةٌ مِنْ عَذَابِ اللَّهِ } أي: عذاب يغشاهم ويعمهم ويستأصلهم، { أَوْ تَأْتِيَهُمُ السَّاعَةُ بَغْتَةً } أي: فجأة { وَهُمْ لَا يَشْعُرُونَ } أي: فإنهم قد استوجبوا لذلك، فليتوبوا إلى الله، ويتركوا ما يكون سببا في عقابهم.
