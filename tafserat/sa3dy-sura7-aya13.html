قَالَ فَاهْبِطْ مِنْهَا فَمَا يَكُونُ لَكَ أَن تَتَكَبَّرَ فِيهَا فَاخْرُجْ إِنَّكَ مِنَ الصَّاغِرِينَ|||

فقال اللّه له: { فَاهْبِطْ مِنْهَا } أي: من الجنة { فَمَا يَكُونُ لَكَ أَنْ تَتَكَبَّرَ فِيهَا } لأنها دار الطيبين الطاهرين، فلا تليق بأخبث خلق اللّه وأشرهم. { فَاخْرُجْ إِنَّكَ مِنَ الصَّاغِرِينَ } أي: المهانين الأذلين، جزاء على كبره وعجبه بالإهانة والذل
