وَيَا قَوْمِ هَٰذِهِ نَاقَةُ اللَّهِ لَكُمْ آيَةً فَذَرُوهَا تَأْكُلْ فِي أَرْضِ اللَّهِ وَلَا تَمَسُّوهَا بِسُوءٍ فَيَأْخُذَكُمْ عَذَابٌ قَرِيبٌ|||

{ وَيَا قَوْمِ هَذِهِ نَاقَةُ اللَّهِ لَكُمْ آيَةً } لها شرب من البئر يوما، ثم يشربون كلهم من ضرعها، ولهم شرب يوم معلوم.{ فَذَرُوهَا تَأْكُلْ فِي أَرْضِ اللَّهِ } أي: ليس عليكم من مؤنتها وعلفها شيء، { وَلَا تَمَسُّوهَا بِسُوءٍ } أي: بعقر { فَيَأْخُذَكُمْ عَذَابٌ قَرِيبٌ }
