فَلَنَسْأَلَنَّ الَّذِينَ أُرْسِلَ إِلَيْهِمْ وَلَنَسْأَلَنَّ الْمُرْسَلِينَ|||

وقوله: { فَلَنَسْأَلَنَّ الَّذِينَ أُرْسِلَ إِلَيْهِمْ } أي: لنسألن الأمم الذين أرسل اللّه إليهم المرسلين، عما أجابوا به رسلهم، { وَيَوْمَ يُنَادِيهِمْ فَيَقُولُ مَاذَا أَجَبْتُمُ الْمُرْسَلِينَ } الآيات. { وَلَنَسْأَلَنَّ الْمُرْسَلِينَ } عن تبليغهم لرسالات ربهم، وعما أجابتهم به أممهم.
