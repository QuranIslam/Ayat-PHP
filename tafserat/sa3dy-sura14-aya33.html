وَسَخَّرَ لَكُمُ الشَّمْسَ وَالْقَمَرَ دَائِبَيْنِ ۖ وَسَخَّرَ لَكُمُ اللَّيْلَ وَالنَّهَارَ|||

{ وَسَخَّرَ لَكُمُ الشَّمْسَ وَالْقَمَرَ دَائِبَيْنِ } لا يفتران، ولا ينيان، يسعيان لمصالحكم، من حساب أزمنتكم ومصالح أبدانكم، وحيواناتكم، وزروعكم، وثماركم، { وَسَخَّرَ لَكُمُ اللَّيْلَ } لتسكنوا فيه { وَالنَّهَارِ } مبصرا لتبتغوا من فضله.
