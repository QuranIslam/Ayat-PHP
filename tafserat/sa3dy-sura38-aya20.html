وَشَدَدْنَا مُلْكَهُ وَآتَيْنَاهُ الْحِكْمَةَ وَفَصْلَ الْخِطَابِ|||

ثم ذكر منته عليه بالملك العظيم فقال: { وَشَدَدْنَا مُلْكَهُ } أي: قويناه بما أعطيناه من الأسباب وكثرة الْعَدَد والْعُدَدِ التي بها قوَّى اللّه ملكه، ثم ذكر منته عليه بالعلم فقال: { وَآتَيْنَاهُ الْحِكْمَةَ } أي: النبوة والعلم العظيم، { وَفَصْلَ الْخِطَابِ } أي: الخصومات بين الناس.
