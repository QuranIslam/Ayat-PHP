مَن يَأْتِيهِ عَذَابٌ يُخْزِيهِ وَيَحِلُّ عَلَيْهِ عَذَابٌ مُّقِيمٌ|||

و { مَنْ يَأْتِيهِ عَذَابٌ يُخْزِيهِ } في الدنيا. { وَيَحِلُّ عَلَيْهِ } في الأخرى { عَذَابٌ مُقِيمٌ } لا يحول عنه ولا يزول،.وهذا تهديد عظيم لهم، وهم يعلمون أنهم المستحقون للعذاب المقيم، ولكن الظلم والعناد حال بينهم وبين الإيمان.
