وَمَثَلُ كَلِمَةٍ خَبِيثَةٍ كَشَجَرَةٍ خَبِيثَةٍ اجْتُثَّتْ مِن فَوْقِ الْأَرْضِ مَا لَهَا مِن قَرَارٍ|||

ثم ذكر ضدها وهي كلمة الكفر وفروعها فقال: { وَمَثَلُ كَلِمَةٍ خَبِيثَةٍ كَشَجَرَةٍ خَبِيثَةٍ } المأكل والمطعم وهي: شجرة الحنظل ونحوها، { اجْتُثَّتْ } هذه الشجرة { مِنْ فَوْقِ الْأَرْضِ مَا لَهَا مِنْ قَرَارٍ } أي: من ثبوت فلا عروق تمسكها، ولا ثمرة صالحة، تنتجها، بل إن وجد فيها ثمرة، فهي ثمرة خبيثة، كذلك كلمة الكفر والمعاصي، ليس لها ثبوت نافع في القلب، ولا تثمر إلا كل قول خبيث وعمل خبيث يستضر به صاحبه، ولا ينتفع، فلا يصعد إلى الله منه عمل صالح ولا ينفع نفسه، ولا ينتفع به غيره.
