وَكَذَٰلِكَ أَوْحَيْنَا إِلَيْكَ قُرْآنًا عَرَبِيًّا لِّتُنذِرَ أُمَّ الْقُرَىٰ وَمَنْ حَوْلَهَا وَتُنذِرَ يَوْمَ الْجَمْعِ لَا رَيْبَ فِيهِ ۚ فَرِيقٌ فِي الْجَنَّةِ وَفَرِيقٌ فِي السَّعِيرِ|||

ثم ذكر منته على رسوله وعلى الناس، حيث أنزل الله { قُرْآنًا عَرَبِيًّا } بين الألفاظ والمعاني { لِتُنْذِرَ أُمَّ الْقُرَى } وهي مكة المكرمة { وَمَنْ حَوْلَهَا } من قرى العرب، ثم يسري هذا الإنذار إلى سائر الخلق. { وَتُنْذِرَ } الناس { يَوْمَ الْجَمْعِ } الذي يجمع الله به الأولين والآخرين، وتخبرهم أنه { لَا رَيْبَ فِيهِ } وأن الخلق ينقسمون فيه فريقين { فَرِيقٌ فِي الْجَنَّةِ } وهم الذين آمنوا بالله، وصدقوا المرسلين، { وَفَرِيقٌ فِي السَّعِيرِ } وهم أصناف الكفرة المكذبين.
