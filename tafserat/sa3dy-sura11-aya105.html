يَوْمَ يَأْتِ لَا تَكَلَّمُ نَفْسٌ إِلَّا بِإِذْنِهِ ۚ فَمِنْهُمْ شَقِيٌّ وَسَعِيدٌ|||

{ يَوْمَ يَأْتِ ْ} ذلك اليوم، ويجتمع الخلق { لَا تَكَلَّمُ نَفْسٌ إِلَّا بِإِذْنِهِ ْ} حتى الأنبياء، والملائكة الكرام، لا يشفعون إلا بإذنه، { فَمِنْهُمْ ْ} أي: الخلق { شَقِيٌّ وَسَعِيدٌ ْ} فالأشقياء، هم الذين كفروا بالله، وكذبوا رسله، وعصوا أمره، والسعداء، هم: المؤمنون المتقون.
