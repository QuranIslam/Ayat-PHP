فَأَخَذَتْهُمُ الصَّيْحَةُ بِالْحَقِّ فَجَعَلْنَاهُمْ غُثَاءً ۚ فَبُعْدًا لِّلْقَوْمِ الظَّالِمِينَ|||

{ فَأَخَذَتْهُمُ الصَّيْحَةُ بِالْحَقِّ } لا بالظلم والجور، بل بالعدل وظلمهم، أخذتهم الصيحة، فأهلكتهم عن آخرهم.

{ فَجَعَلْنَاهُمْ غُثَاءً } أي: هشيما يبسا بمنزلة غثاء السيل الملقى في جنبات الوادي، وقال في الآية الأخرى { إِنَّا أَرْسَلْنَا عَلَيْهِمْ صَيْحَةً وَاحِدَةً فَكَانُوا كَهَشِيمِ الْمُحْتَظِرِ }

{ فَبُعْدًا لِلْقَوْمِ الظَّالِمِينَ } أي: أتبعوا مع عذابهم، البعد واللعنة والذم من العالمين { فَمَا بَكَتْ عَلَيْهِمُ السَّمَاءُ وَالْأَرْضُ وَمَا كَانُوا مُنْظَرِينَ }
