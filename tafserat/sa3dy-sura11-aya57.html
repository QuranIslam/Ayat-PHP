فَإِن تَوَلَّوْا فَقَدْ أَبْلَغْتُكُم مَّا أُرْسِلْتُ بِهِ إِلَيْكُمْ ۚ وَيَسْتَخْلِفُ رَبِّي قَوْمًا غَيْرَكُمْ وَلَا تَضُرُّونَهُ شَيْئًا ۚ إِنَّ رَبِّي عَلَىٰ كُلِّ شَيْءٍ حَفِيظٌ|||

{ فَإِنْ تَوَلَّوْا } عما دعوتكم إليه { فَقَدْ أَبْلَغْتُكُمْ مَا أُرْسِلْتُ بِهِ إِلَيْكُمْ } فلم يبق عليَّ تبعة من شأنكم.

{ وَيَسْتَخْلِفُ رَبِّي قَوْمًا غَيْرَكُمْ } يقومون بعبادته، ولا يشركون به شيئا، { وَلَا تَضُرُّونَهُ شَيْئًا } فإن ضرركم، إنما يعود عليكم، فالله لا تضره معصية العاصين. ولا تنفعه طاعة المطيعين { من عمل صالحا فلنفسه ومن أساء فعليها } [ { إِنَّ رَبِّي عَلَى كُلِّ شَيْءٍ حَفِيظٌ } ].
