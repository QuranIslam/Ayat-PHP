فَأَسْرِ بِأَهْلِكَ بِقِطْعٍ مِّنَ اللَّيْلِ وَاتَّبِعْ أَدْبَارَهُمْ وَلَا يَلْتَفِتْ مِنكُمْ أَحَدٌ وَامْضُوا حَيْثُ تُؤْمَرُونَ|||

{ فَأَسْرِ بِأَهْلِكَ بِقِطْعٍ مِنَ اللَّيْلِ } أي: في أثنائه حين تنام العيون ولا يدري أحد عن مسراك، { وَلَا يَلْتَفِتْ مِنْكُمْ أَحَدٌ } أي: بادروا وأسرعوا، { وَامْضُوا حَيْثُ تُؤْمَرُونَ } كأن معهم دليلا يدلهم إلى أين يتوجهون
