هُمْ وَأَزْوَاجُهُمْ فِي ظِلَالٍ عَلَى الْأَرَائِكِ مُتَّكِئُونَ|||

ومن ذلك افتضاض العذارى الجميلات، كما قال: { هُمْ وَأَزْوَاجُهُمْ } من الحور العين، اللاتي قد جمعن حسن الوجوه والأبدان وحسن الأخلاق. { فِي ظِلَالٍ عَلَى الْأَرَائِكِ } أي: على السرر المزينة باللباس المزخرف الحسن. { مُتَّكِئُونَ } عليها، اتكاء على كمال الراحة والطمأنينة واللذة.
