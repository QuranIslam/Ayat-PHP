مُّهْطِعِينَ إِلَى الدَّاعِ ۖ يَقُولُ الْكَافِرُونَ هَٰذَا يَوْمٌ عَسِرٌ|||

{ مُهْطِعِينَ إِلَى الدَّاعِ } أي: مسرعين لإجابة النداء الداعي وهذا يدل على أن الداعي يدعوهم ويأمرهم بالحضور لموقف القيامة، فيلبون دعوته، ويسرعون إلى إجابته، { يَقُولُ الْكَافِرُونَ } الذين قد حضر عذابهم: { هَذَا يَوْمٌ عَسِرٌ } كما قال تعالى { على الكافرين غير يسير } [مفهوم ذلك أنه يسير سهل على المؤمنين]
