وَإِذْ قُلْنَا ادْخُلُوا هَٰذِهِ الْقَرْيَةَ فَكُلُوا مِنْهَا حَيْثُ شِئْتُمْ رَغَدًا وَادْخُلُوا الْبَابَ سُجَّدًا وَقُولُوا حِطَّةٌ نَّغْفِرْ لَكُمْ خَطَايَاكُمْ ۚ وَسَنَزِيدُ الْمُحْسِنِينَ|||

وهذا أيضا من نعمته عليهم بعد معصيتهم إياه, فأمرهم بدخول قرية تكون لهم عزا ووطنا ومسكنا, ويحصل لهم فيها الرزق الرغد، وأن يكون دخولهم على وجه خاضعين لله فيه بالفعل, وهو دخول الباب { سجدا } أي: خاضعين ذليلين، وبالقول وهو أن يقولوا: { حِطَّةٌ } أي أن يحط عنهم خطاياهم بسؤالهم إياه مغفرته. { نَغْفِرْ لَكُمْ خَطَايَاكُمْ } بسؤالكم المغفرة، { وَسَنَزِيدُ الْمُحْسِنِينَ } بأعمالهم, أي: جزاء عاجل وآجلا.
