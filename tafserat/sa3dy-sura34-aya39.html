قُلْ إِنَّ رَبِّي يَبْسُطُ الرِّزْقَ لِمَن يَشَاءُ مِنْ عِبَادِهِ وَيَقْدِرُ لَهُ ۚ وَمَا أَنفَقْتُم مِّن شَيْءٍ فَهُوَ يُخْلِفُهُ ۖ وَهُوَ خَيْرُ الرَّازِقِينَ|||

ثم أعاد تعالى أنه { يَبْسُطُ الرِّزْقَ لِمَنْ يَشَاءُ مِنْ عِبَادِهِ وَيَقْدِرُ لَهُ } ليرتب عليه قوله: { وَمَا أَنْفَقْتُمْ مِنْ شَيْءٍ } نفقة واجبة, أو مستحبة, على قريب, أو جار, أو مسكين, أو يتيم, أو غير ذلك، { فَهُوَ } تعالى { يُخْلِفُهُ } فلا تتوهموا أن الإنفاق مما ينقص الرزق, بل وعد بالخلف للمنفق, الذي يبسط الرزق لمن يشاء ويقدر { وَهُوَ خَيْرُ الرَّازِقِينَ } فاطلبوا الرزق منه, واسعوا في الأسباب التي أمركم بها.
