لَا يُفَتَّرُ عَنْهُمْ وَهُمْ فِيهِ مُبْلِسُونَ|||

{ لَا يُفَتَّرُ عَنْهُمْ } العذاب ساعة، بإزالته، ولا بتهوين عذابه، { وَهُمْ فِيهِ مُبْلِسُونَ } أي: آيسون من كل خير، غير راجين للفرج، وذلك أنهم ينادون ربهم فيقولون: { رَبَّنَا أَخْرِجْنَا مِنْهَا فَإِنْ عُدْنَا فَإِنَّا ظَالِمُونَ قَالَ اخْسَئُوا فِيهَا وَلَا تُكَلِّمُونِ }
