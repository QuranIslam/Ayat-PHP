ثُمَّ آتَيْنَا مُوسَى الْكِتَابَ تَمَامًا عَلَى الَّذِي أَحْسَنَ وَتَفْصِيلًا لِّكُلِّ شَيْءٍ وَهُدًى وَرَحْمَةً لَّعَلَّهُم بِلِقَاءِ رَبِّهِمْ يُؤْمِنُونَ|||

{ ثُمَّ } في هذا الموضع، ليس المراد منها الترتيب الزماني، فإن زمن موسى عليه السلام، متقدم على تلاوة الرسول محمد صلى الله عليه وسلم هذا الكتاب، وإنما المراد الترتيب الإخباري. فأخبر أنه آتى { مُوسَى الْكِتَابَ } وهو التوراة { تَمَامًا } لنعمته، وكمالا لإحسانه. { عَلَى الَّذِي أَحْسَنَ } من أُمة موسى، فإن الله أنعم على المحسنين منهم بنِعَم لا تحصى. من جملتها وتمامها إنزال التوراة عليهم. فتمت عليهم نعمة الله، ووجب عليهم القيام بشكرها. { وَتَفْصِيلًا لِكُلِّ شَيْءٍ } يحتاجون إلى تفصيله، من الحلال والحرام، والأمر والنهي، والعقائد ونحوها. { وَهُدًى وَرَحْمَةً } أي: يهديهم إلى الخير، ويعرفهم بالشر، في الأصول والفروع. { وَرَحْمَةٌ } يحصل به لهم السعادة والرحمة والخير الكثير. { لَعَلَّهُمْ } بسبب إنزالنا الكتاب والبينات عليهم { بِلِقَاءِ رَبِّهِمْ يُؤْمِنُونَ } فإنه اشتمل من الأدلة القاطعة على البعث والجزاء بالأعمال، ما يوجب لهم الإيمان بلقاء ربهم والاستعداد له.
