فَوَقَاهُمُ اللَّهُ شَرَّ ذَٰلِكَ الْيَوْمِ وَلَقَّاهُمْ نَضْرَةً وَسُرُورًا|||

{ فَوَقَاهُمُ اللَّهُ شَرَّ ذَلِكَ الْيَوْمِ } فلا يحزنهم الفزع الأكبر، وتتلقاهم الملائكة [هذا يومكم الذي كنتم توعدون].

{ وَلَقَّاهُمْ } أي: أكرمهم وأعطاهم { نَضْرَةً } في وجوههم { وَسُرُورًا } في قلوبهم، فجمع لهم بين نعيم الظاهر والباطن
