بَلْ مَتَّعْتُ هَٰؤُلَاءِ وَآبَاءَهُمْ حَتَّىٰ جَاءَهُمُ الْحَقُّ وَرَسُولٌ مُّبِينٌ|||

فقال تعالى: { بَلْ مَتَّعْتُ هَؤُلاءِ وَآبَاءَهُمْ } بأنواع الشهوات، حتى صارت هي غايتهم ونهاية مقصودهم، فلم تزل يتربى حبها في قلوبهم، حتى صارت صفات راسخة، وعقائد متأصلة. { حَتَّى جَاءَهُمُ الْحَقُّ } الذي لا شك فيه ولا مرية ولا اشتباه. { وَرَسُولٌ مُبِينٌ } أي: بين الرسالة، قامت أدلة رسالته قياما باهرا، بأخلاقه ومعجزاته، وبما جاء به، وبما صدق به المرسلين، وبنفس دعوته صلى اللّه عليه وسلم.
