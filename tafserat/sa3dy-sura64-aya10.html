وَالَّذِينَ كَفَرُوا وَكَذَّبُوا بِآيَاتِنَا أُولَٰئِكَ أَصْحَابُ النَّارِ خَالِدِينَ فِيهَا ۖ وَبِئْسَ الْمَصِيرُ|||

{ وَالَّذِينَ كَفَرُوا وَكَذَّبُوا بِآيَاتِنَا } أي: كفروا [بها] من غير مستند شرعي ولا عقلي، بل جاءتهم الأدلة والبينات، فكذبوا بها، وعاندوا ما دلت عليه.

{ أُولَئِكَ أَصْحَابُ النَّارِ خَالِدِينَ فِيهَا وَبِئْسَ الْمَصِيرُ } لأنها جمعت كل بؤس وشدة، وشقاء وعذاب.
