إِنَّ اللَّهَ يُدْخِلُ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ جَنَّاتٍ تَجْرِي مِن تَحْتِهَا الْأَنْهَارُ يُحَلَّوْنَ فِيهَا مِنْ أَسَاوِرَ مِن ذَهَبٍ وَلُؤْلُؤًا ۖ وَلِبَاسُهُمْ فِيهَا حَرِيرٌ|||

{ إِنَّ اللَّهَ يُدْخِلُ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ جَنَّاتٍ تَجْرِي مِنْ تَحْتِهَا الْأَنْهَارُ } ومعلوم أن هذا الوصف لا يصدق على غير المسلمين، الذين آمنوا بجميع الكتب، وجميع الرسل، { يُحَلَّوْنَ فِيهَا مِنْ أَسَاوِرَ مِنْ ذَهَبٍ } أي: يسورون في أيديهم، رجالهم ونساؤهم أساور الذهب.{ وَلِبَاسُهُمْ فِيهَا حَرِيرٌ } فتم نعيمهم بذكر أنواع المأكولات اللذيذات المشتمل عليها، لفظ الجنات، وذكر الأنهار السارحات، أنهار الماء واللبن والعسل والخمر، وأنواع اللباس، والحلي الفاخر
