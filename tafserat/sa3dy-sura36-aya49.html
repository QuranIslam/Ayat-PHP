مَا يَنظُرُونَ إِلَّا صَيْحَةً وَاحِدَةً تَأْخُذُهُمْ وَهُمْ يَخِصِّمُونَ|||

{ مَا يَنْظُرُونَ إِلَّا صَيْحَةً وَاحِدَةً } وهي نفخة الصور { تَأْخُذُهُمْ } أي: تصيبهم { وَهُمْ يَخِصِّمُونَ } أي: وهم لا هون عنها، لم تخطر على قلوبهم في حال خصومتهم، وتشاجرهم بينهم، الذي لا يوجد في الغالب إلا وقت الغفلة.
