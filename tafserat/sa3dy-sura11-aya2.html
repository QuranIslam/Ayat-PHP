أَلَّا تَعْبُدُوا إِلَّا اللَّهَ ۚ إِنَّنِي لَكُم مِّنْهُ نَذِيرٌ وَبَشِيرٌ|||

} فإذا كان إحكامه وتفصيله من عند الله الحكيم الخبير، فلا تسأل بعد هذا، عن عظمته وجلالته واشتماله على كمال الحكمة، وسعة الرحمة . وإنما أنزل الله كتابه لـ { أَنْ لَا تَعْبُدُوا إِلَّا اللَّهَ } أي: لأجل إخلاص الدين كله لله، وأن لا يشرك به أحد من خلقه.

{ إِنَّنِي لَكُمْ } أيها الناس { مِنْهُ } أي: من الله ربكم { نَذِيرٍ } لمن تجرأ على المعاصي بعقاب الدنيا والآخرة، { وَبَشِيرٌ } للمطيعين لله بثواب الدنيا والآخرة.
