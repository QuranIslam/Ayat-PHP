فَلَيْسَ لَهُ الْيَوْمَ هَاهُنَا حَمِيمٌ|||

{ فَلَيْسَ لَهُ الْيَوْمَ هَاهُنَا } أي: يوم القيامة { حَمِيمٍ } أي: قريب أو صديق يشفع له لينجو من عذاب الله أو يفوز بثواب الله: { وَلَا تَنْفَعُ الشَّفَاعَةُ عِنْدَهُ إِلَّا لِمَنْ أَذِنَ لَهُ } { مَا لِلظَّالِمِينَ مِنْ حَمِيمٍ وَلَا شَفِيعٍ يُطَاعُ } .
