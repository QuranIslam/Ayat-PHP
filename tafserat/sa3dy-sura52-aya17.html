إِنَّ الْمُتَّقِينَ فِي جَنَّاتٍ وَنَعِيمٍ|||

لما ذكر تعالى عقوبة المكذبين، ذكر نعيم المتقين، ليجمع بين الترغيب والترهيب، فتكون القلوب بين الخوف والرجاء، فقال: { إِنَّ الْمُتَّقِينَ } لربهم، الذين اتقوا سخطه وعذابه، بفعل أسبابه من امتثال الأوامر واجتناب النواهي.

{ فِي جَنَّاتِ } أي: بساتين، قد اكتست رياضها من الأشجار الملتفة، والأنهار المتدفقة، والقصور المحدقة، والمنازل المزخرفة، { وَنَعِيمٍ } [وهذا] شامل لنعيم القلب والروح والبدن
