فَتَقَطَّعُوا أَمْرَهُم بَيْنَهُمْ زُبُرًا ۖ كُلُّ حِزْبٍ بِمَا لَدَيْهِمْ فَرِحُونَ|||

، ولكن أبى الظالمون المفترقون إلا عصيانا، ولهذا قال: { فَتَقَطَّعُوا أَمْرَهُمْ بَيْنَهُمْ زُبُرًا } أي: تقطع المنتسبون إلى اتباع الأنبياء { أَمْرُهُمْ } أي: دينهم { بَيْنَهُمْ زُبُرًا } أي: قطعا { كُلُّ حِزْبٍ بِمَا لَدَيْهِمْ } أي: بما عندهم من العلم والدين { فَرِحُونَ } يزعمون أنهم المحقون، وغيرهم على غير الحق، مع أن المحق منهم، من كان على طريق الرسل، من أكل الطيبات، والعمل الصالح، وما عداهم فإنهم مبطلون.
