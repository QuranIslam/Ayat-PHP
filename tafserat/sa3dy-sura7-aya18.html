قَالَ اخْرُجْ مِنْهَا مَذْءُومًا مَّدْحُورًا ۖ لَّمَن تَبِعَكَ مِنْهُمْ لَأَمْلَأَنَّ جَهَنَّمَ مِنكُمْ أَجْمَعِينَ|||

أي: قال اللّه لإبليس لما قال ما قال: { اخْرُجْ مِنْهَا } خروج صغار واحتقار، لا خروج إكرام بل { مَذْءُومًا } أي: مذموما { مَدْحُورًا } مبعدا عن اللّه وعن رحمته وعن كل خير. { لَأَمْلَأَنَّ جَهَنَّمَ } منك وممن تبعك منهم { أَجْمَعِينَ } وهذا قسم منه تعالى، أن النار دار العصاة، لا بد أن يملأها من إبليس وأتباعه من الجن والإنس.
