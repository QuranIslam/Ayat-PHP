لُعِنَ الَّذِينَ كَفَرُوا مِن بَنِي إِسْرَائِيلَ عَلَىٰ لِسَانِ دَاوُودَ وَعِيسَى ابْنِ مَرْيَمَ ۚ ذَٰلِكَ بِمَا عَصَوا وَّكَانُوا يَعْتَدُونَ|||

{ لُعِنَ الَّذِينَ كَفَرُوا مِنْ بَنِي إِسْرَائِيلَ } أي: طردوا وأبعدوا عن رحمة الله { عَلَى لِسَـانِ دَاوُدَ وَعِيسَـى ابْنِ مَرْيَــمَ } أي: بشهادتهما وإقرارهما، بأن الحجة قد قامت عليهم، وعاندوها. { ذَلِكَ } الكفر واللعن { بِمَا عَصَوْا وَكَانُوا يَعْتَدُونَ } أي: بعصيانهم لله، وظلمهم لعباد الله، صار سببا لكفرهم وبعدهم عن رحمة الله، فإن للذنوب والظلم عقوبات.
