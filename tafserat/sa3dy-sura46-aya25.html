تُدَمِّرُ كُلَّ شَيْءٍ بِأَمْرِ رَبِّهَا فَأَصْبَحُوا لَا يُرَىٰ إِلَّا مَسَاكِنُهُمْ ۚ كَذَٰلِكَ نَجْزِي الْقَوْمَ الْمُجْرِمِينَ|||

{ تُدَمِّرُ كُلَّ شَيْءٍ } تمر عليه من شدتها ونحسها.

فسلطها الله عليهم { سبع ليالي وثمانية أيام حسوما فترى القوم فيها صرعى كأنهم أعجاز نخل خاوية } [ { بِأَمْرِ رَبِّهَا } أي: بإذنه ومشيئته]. { فَأَصْبَحُوا لَا يُرَى إِلَّا مَسَاكِنُهُمْ } قد تلفت مواشيهم وأموالهم وأنفسهم. { كَذَلِكَ نَجْزِي الْقَوْمَ الْمُجْرِمِينَ } بسبب جرمهم وظلمهم.
