يَوْمَ يَقُومُ الرُّوحُ وَالْمَلَائِكَةُ صَفًّا ۖ لَّا يَتَكَلَّمُونَ إِلَّا مَنْ أَذِنَ لَهُ الرَّحْمَٰنُ وَقَالَ صَوَابًا|||

وفي ذلك اليوم { يَقُومُ الرُّوحُ } وهو جبريل عليه السلام، الذي هو أشرف الملائكة { وَالْمَلَائِكَةِ } [أيضا يقوم الجميع] { صَفًّا } خاضعين لله { لَا يَتَكَلَّمُونَ } إلا بما أذن لهم الله به .
