وَلَوْ شَاءَ اللَّهُ لَجَعَلَهُمْ أُمَّةً وَاحِدَةً وَلَٰكِن يُدْخِلُ مَن يَشَاءُ فِي رَحْمَتِهِ ۚ وَالظَّالِمُونَ مَا لَهُم مِّن وَلِيٍّ وَلَا نَصِيرٍ|||

{ و } مع هذا { لَوْ شَاءَ اللَّهُ } لجعل الناس، أي: جعل الناس { أُمَّةً وَاحِدَةً } على الهدى، لأنه القادر الذي لا يمتنع عليه شيء، ولكنه أراد أن يدخل في رحمته من شاء من خواص خلقه.

وأما الظالمون الذين لا يصلحون لصالح، فإنهم محرومون من الرحمة، فـ { مَا لَهُمْ } من دون الله { مِنْ وَلِيٍّ } يتولاهم، فيحصل لهم المحبوب { وَلَا نَصِيرٍ } يدفع عنهم المكروه.
