وَلَوْ تَرَىٰ إِذِ الْمُجْرِمُونَ نَاكِسُو رُءُوسِهِمْ عِندَ رَبِّهِمْ رَبَّنَا أَبْصَرْنَا وَسَمِعْنَا فَارْجِعْنَا نَعْمَلْ صَالِحًا إِنَّا مُوقِنُونَ|||

لما ذكر تعالى رجوعهم إليه يوم القيامة، ذكر حالهم في مقامهم بين [يديه] فقال: { وَلَوْ تَرَى إِذِ الْمُجْرِمُونَ } الذين أصروا على الذنوب العظيمة، { نَاكِسُو رُءُوسِهِمْ عِنْدَ رَبِّهِمْ } خاشعين خاضعين أذلاء، مقرين بجرمهم، سائلين الرجعة قائلين: { رَبَّنَا أَبْصَرْنَا وَسَمِعْنَا } أي: بأن لنا الأمر، ورأيناه عيانًا، فصار عين يقين.

{ فَارْجِعْنَا نَعْمَلْ صَالِحًا إِنَّا مُوقِنُونَ } أي: صار عندنا الآن، يقين بما [كنا] نكذب به، أي: لرأيت أمرا فظيعًا، وحالاً مزعجة، وأقوامًا خاسرين، وسؤلًا غير مجاب، لأنه قد مضى وقت الإمهال.
