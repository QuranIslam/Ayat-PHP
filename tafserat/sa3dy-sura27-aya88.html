وَتَرَى الْجِبَالَ تَحْسَبُهَا جَامِدَةً وَهِيَ تَمُرُّ مَرَّ السَّحَابِ ۚ صُنْعَ اللَّهِ الَّذِي أَتْقَنَ كُلَّ شَيْءٍ ۚ إِنَّهُ خَبِيرٌ بِمَا تَفْعَلُونَ|||

ومن هوله أنك { ترى الْجِبَالَ تَحْسَبُهَا جَامِدَةً } لا تفقد [شيئا] منها وتظنها باقية على الحال المعهودة وهي قد بلغت منها الشدائد والأهوال كل مبلغ وقد تفتت ثم تضمحل وتكون هباء منبثا. ولهذا قال: { وَهِيَ تَمُرُّ مَرَّ السَّحَابِ } من خفتها وشدة ذلك الخوف وذلك { صُنْعَ اللَّهِ الَّذِي أَتْقَنَ كُلَّ شَيْءٍ إِنَّهُ خَبِيرٌ بِمَا تَفْعَلُونَ } فيجازيكم بأعمالكم.
