الْمُلْكُ يَوْمَئِذٍ لِّلَّهِ يَحْكُمُ بَيْنَهُمْ ۚ فَالَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ فِي جَنَّاتِ النَّعِيمِ|||

{ الْمُلْكُ يَوْمَئِذٍ } أي: يوم القيامة { لِلَّهِ } تعالى، لا لغيره، { يَحْكُمُ بَيْنَهُمْ } بحكمه العدل، وقضائه الفصل، { فَالَّذِينَ آمَنُوا } بالله ورسله، وما جاءوا به { وَعَمِلُوا الصَّالِحَاتِ } ليصدقوا بذلك إيمانهم { فِي جَنَّاتِ النَّعِيمِ } نعيم القلب والروح والبدن، مما لا يصفه الواصفون، ولا تدركه العقول.
